# Remote Program

## Reservation Execution Status

Sent when the host computer queries the reservation execution status on the remote mode interface.

**Command Word:** `0x5500`

### Request Parameters

| Parameter | Type | Required | Description |
|--------|------|------|------|
| robot | int | Yes | Select robot, range [1, 4] |

### Request Example

```json
{
    "robot": 1
}
```

Controller replies with reservation execution status query result.

**Command Word:** `0x5501`

### Response Parameters

| Parameter | Type | Description |
|--------|------|------|
| robot | int | Select robot, range [1, 4] |
| current | object | Current task status |
| current.station | int | Station number |
| current.name | string | Program name |
| current.runtime | int | Run time |
| current.times | int | Run count |
| current.count | int | Completed run count |
| current.status | int | Status: 0-No reservation, 1-Reserving, 2-Running, 3-Reserved, 4-Program paused |
| queue | array | Reservation queue |
| queue[].station | int | Station number |
| queue[].name | string | Program name |
| queue[].times | int | Run count |
| queue[].count | int | Completed run count |
| queue[].status | int | Status |

### Response Example

```json
{
    "robot": 1,
    "current": {
        "station": 1,
        "name": "Q1",
        "runtime": 1,
        "times": 3,
        "count": 1,
        "status": 2
    },
    "queue": [
        {
            "station": 1,
            "name": "Q4",
            "times": 2,
            "count": 0,
            "status": 1
        },
        {
            "station": 1,
            "name": "Q1",
            "times": 6,
            "count": 0,
            "status": 1
        },
        {
            "station": 1,
            "name": "Q3",
            "times": 1,
            "count": 0,
            "status": 1
        },
        {
            "station": 1,
            "name": "",
            "times": 1,
            "count": 0,
            "status": 0
        },
        {
            "station": 1,
            "name": "",
            "times": 2,
            "count": 0,
            "status": 0
        }
    ]
}
```

## Remote Status

Host computer sends remote control setting.

**Command Word:** `0x5510`

### Request Parameters

| Parameter | Type | Required | Description |
|--------|------|------|------|
| remotectl | bool | Yes | Remote control status |

### Request Example

```json
{
    "remotectl": false
}
```

Host computer sends remote control query.

**Command Word:** `0x5511`

date: none

Controller replies to query.

**Command Word:** `0x5512`

### Response Parameters

| Parameter | Type | Description |
|--------|------|------|
| remotectl | bool | Remote control status |

### Response Example

```json
{
    "remotectl": false
}
```

## Remote Program

Host computer sends set remote program.

**Command Word:** `0x5520`

### Request Parameters

| Parameter | Type | Required | Description |
|--------|------|------|------|
| robot | int | Yes | Select robot, range [1, 4] |
| jobFile | array | Yes | Program list |
| jobFile[].name | string | Yes | Program name |
| jobFile[].times | int | Yes | Run count |
| interpolationMethod | int | No | Interpolation method |
| absolutePosResolution | double | No | Absolute position resolution |
| stopTime | int | No | Pause time |
| runDelayTime | int | No | Run delay time |
| remoteDefaultSpeed | int | No | Remote mode speed 1-100% |

### Request Example

```json
{
    "robot": 1,
    "jobFile": [
        {"name": "Q1", "times": 3},
        {"name": "Q2", "times": 3},
        {"name": "", "times": 1},
        {"name": "Q4", "times": 2},
        {"name": "", "times": 1}
    ],
    "interpolationMethod": 4,
    "absolutePosResolution": 0.06,
    "stopTime": 5,
    "runDelayTime": 600,
    "remoteDefaultSpeed": 1
}
```

## Remote Mode Parameter Settings

Host computer sends remote mode parameter settings.

**Command Word:** `0x5530`

### Request Parameters

| Parameter | Type | Required | Description |
|--------|------|------|------|
| robot | int | Yes | Select robot, range [1, 4] |
| waitingTime | double | No | IO repeat trigger time |
| reserveIsStart | int | No | Auto start running after reservation IO: 0-Off, 1-On |
| interpolationMethod | int | No | Interpolation method |
| reserveTrgTime | double | No | Remote mode IO reservation and start time (milliseconds) |
| absolutePosResolution | int | No | Absolute position resolution |
| stopTime | int | No | Pause time |
| runDelayTime | int | No | Run delay time |
| remoteDefaultSpeed | int | No | Remote mode speed 1-100% |

### Request Example

```json
{
    "robot": 1,
    "waitingTime": 1,
    "reserveIsStart": 0,
    "interpolationMethod": 4,
    "reserveTrgTime": 200,
    "absolutePosResolution": 0.06,
    "stopTime": 5,
    "runDelayTime": 600,
    "remoteDefaultSpeed": 1
}
```

Host computer sends remote mode parameter query.

**Command Word:** `0x5531`

### Request Parameters

| Parameter | Type | Required | Description |
|--------|------|------|------|
| robot | int | Yes | Select robot, range [1, 4] |

### Request Example

```json
{
    "robot": 1
}
```

Controller replies with query result.

**Command Word:** `0x5532`

### Response Parameters

| Parameter | Type | Description |
|--------|------|------|
| remoteDefaultSpeed | int | Remote mode speed 1-100% |
| remoteProgramNum | int | Remote program count |
| reserveIsStart | bool | Auto start running after reservation IO: 0-Off, 1-On |
| waitingTime | int | IO repeat trigger time |

### Response Example

```json
{

    "remoteProgramNum": 10,
    "reserveIsStart": false,
    "waitingTime": 500
}
```

## Remote IO Function Interface Settings

Host computer sends set IO function interface.

**Command Word:** `0x5540`

### Request Parameters

| Parameter | Type | Required | Description |
|--------|------|------|------|
| robot | int | Yes | Select robot, range [1, 4] |
| remoteProgramNum | int | Yes | Remote IO function count |
| inPort | object | No | IO trigger port binding |
| inPort.clearoutagekeep | int | No | Clear power-off saved data, bound IO port: 0-None, 1-Port 1 |
| inPort.faultReset | int | No | Clear alarm, bound IO port |
| inPort.pause | int | No | Pause, bound IO port |
| inPort.start | int | No | Start, bound IO port |
| inPort.stop | int | No | Stop, bound IO port |
| inValue | object | No | IO parameters |
| inValue.clearoutagekeep | int | No | Clear power-off saved data, bound IO port trigger required state: 0-Low active, 1-High active |
| inValue.faultReset | int | No | Clear alarm, bound IO port trigger required state |
| inValue.pause | int | No | Pause, bound IO port trigger required state |
| inValue.start | int | No | Start, bound IO port trigger required state |
| inValue.stop | int | No | Stop, bound IO port trigger required state |
| program | array | No | Program list |
| program[].port | int | No | Port |
| program[].value | int | No | Value |

## Remote Mode Connection Status

Sent when querying Modbus and external IO connection status.

**Command Word:** `0x5560`

date: none

Returns when receiving connection status query.

**Command Word:** `0x5561`

### Response Parameters

| Parameter | Type | Description |
|--------|------|------|
| ModbusConnect | bool | Modbus connection status: false-Not connected, true-Connected |
| ExternIOConnect | bool | External IO connection status: false-Not connected, true-Connected |
| type | int | Type: 0-NAN, 1-RTU, 2-TCP |

### Response Example

```json
{
    "ModbusConnect": false,
    "ExternIOConnect": true,
    "type": 2
}
```

Controller sends whether job file was modified via IO (when external button triggers job file modification, controller sends to host computer).

**Command Word:** `0x5562`

### Response Parameters

| Parameter | Type | Description |
|--------|------|------|
| isModify | bool | Whether job file was modified via IO: true-Yes, false-No |

### Response Example

```json
{
    "isModify": false
}
```
