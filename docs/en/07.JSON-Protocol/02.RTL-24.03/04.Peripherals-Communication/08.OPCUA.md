# OPCUA

## Upper Computer Sets OPCUA Parameters

**Command:** `0x7350`

### Parameter List

| Field | Type | Description |
|------|------|------|
| client | object | Client parameters |
| client.enable | bool | Connection enable |
| client.ip | string | Server IP address |
| client.port | int | Port number, range [0, 65535] |
| client.resource | string | Server name |
| server | object | Server parameters |
| server.enable | bool | Connection enable |
| server.ip | string | IP address when the controller acts as a server |
| server.port | int | Port number, range [0, 65535] |

### Request Example

```json
{
  "client":
  {
    "enable":false,
    "ip":"192.168.1.240",
    "port":49400,
    "resource":"ProsysServer"
  },
  "server":
  {
    "enable":false,
    "ip":"192.168.0.229",
    "port":4840
  }
}
```
