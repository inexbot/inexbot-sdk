# Auto-Start Program Execution

## Teach Pendant Settings

**Register:** `0x5081` `SELF_START_RUN_SET`

### Parameters

| Parameter | Type   | Required | Description       | Example |
|-----------|--------|----------|-------------------|---------|
| robot     | int    | Yes      | Robot number      | 1       |
| enable    | bool   | Yes      | Enable            | false   |
| count     | int    | Yes      | Run count         | 1       |
| speed     | int    | Yes      | Run speed         | 15      |
| jobname   | string | Yes      | Job file name     | "Q1"    |

### JSON Example

```json
{
    "robot": 1,
    "enable": false,
    "count": 1,
    "speed": 15,
    "jobname": "Q1"
}
```
