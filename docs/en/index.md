# iNexBot Technology Knowledge Base

iNexBot Technology was founded in 2015 and is a technology pioneer in China's high-end motion control field. With the core philosophy of "software-defined control, algorithm-driven intelligence," we provide a full-stack technology system from underlying hardware and middleware platforms to upper-layer applications.

## Knowledge Base Directory Structure

This repository is organized by technical domain and application scenario, making it easy for developers and engineers to quickly search:

- **Controller** — Controller secondary development APIs, examples, and custom commands
- **Teach Pendant** — Teach pendant secondary development APIs and quick start
- **Host Computer** — C++/C#/Python progressive tutorials and interface examples
- **Master Library** — Master communication library documentation
- **JSON Protocol** — RTL-22.07 JSON communication protocol reference
- **ROS** — ROS-1 / ROS-2 integration guides
