# DH Parameters

## Robot Basic Parameters

> Description: Includes setting and querying of robot basic parameters such as DH parameters, joint parameters, and Cartesian parameters

---

## 1. Host Sets Current Robot DH Parameters

**Command:** `0x20C0`
