# Modbus

## 1. Host Sets Modbus Master Parameters

**Command:** `0x7300`

### Parameter List

| Parameter | Type | Required | Description |
|--------|------|------|------|
| masterStation | object | Yes | Master station parameters, including RTU and TCP parameters |
| startAddress | bool | No | Address offset; true means address unchanged, false means address auto-decrement by 1 |

#### masterStation Parameters

| Parameter | Type | Required | Description |
|--------|------|------|------|
| type | string | Yes | Master station type; values "TCP", "RTU" |
| processNumber | int | Yes | Process number to set; range [1,9] |
| RTU | object | No | RTU parameters |
| TCP | object | No | TCP parameters |

#### RTU Parameters

| Parameter | Type | Required | Description |
|--------|------|------|------|
| baudrate | int | Yes | Baud rate |
| checkBit | string | Yes | Check bit; values "N", "E", "O" |
| dataBit | int | Yes | Data bit; range [5,8] |
| port | int | Yes | Port number |
| slaveId | int | Yes | Slave ID; range [0,65535] |
| stopBit | int | Yes | Stop bit; range [0,1] |

#### TCP Parameters

| Parameter | Type | Required | Description |
|--------|------|------|------|
| IP | string | Yes | IP address |
| endian_type | int | Yes | Float endianness; 0 means AB CD, 1 means CD AB, 2 means BA DC, 3 means DC BA |
| port | int | Yes | Port number; range [0,65535] |

### Request Example

```json
{
  "masterStation": {
    "RTU": {
      "baudrate": 115200,
      "checkBit": "E",
      "dataBit": 5,
      "port": 2,
      "slaveId": 1,
      "stopBit": 1
    },
    "TCP": {
      "IP": "192.168.1.14",
      "endian_type": 1,
      "port": 503
    },
    "processNumber": 1,
    "type": "TCP"
  },
  "startAddress": false
}
```

## 3. Controller Replies Host Modbus Parameters

**Command:** `0x7302`

| Parameter | Type | Required | Description |
|--------|------|------|------|
| type | string | Yes | Master station type; values "TCP", "RTU" |
| RTU | object | No | RTU parameters |
| TCP | object | No | TCP parameters |
| modbus_state | bool | Yes | Modbus connection status; true means connected, false means not connected |
| response_time_out | int | Yes | Read/write response time; unit ms |
| startAddress | bool | Yes | Address offset; true means address unchanged, false means address auto-decrement by 1 |

#### RTU Parameters

| Parameter | Type | Required | Description |
|--------|------|------|------|
| baudrate | int | Yes | Baud rate |
| checkBit | string | Yes | Check bit; values "N", "E", "O" |

| port | int | Yes | Port number |
| slaveId | int | Yes | Slave ID; range [0,65535] |
| stopBit | int | Yes | Stop bit; range [0,1] |

#### TCP Parameters

| Parameter | Type | Required | Description |
|--------|------|------|------|
| IP | string | Yes | IP address |
| endian_type | int | Yes | Float endianness; 0 means AB CD, 1 means CD AB, 2 means BA DC, 3 means DC BA |
| port | int | Yes | Port number; range [0,65535] |

### Response Example

```json
{
  "RTU": {
    "baudrate": 115200,
    "checkBit": "E",
    "dataBit": 5,
    "port": 2,
    "slaveId": 1,
    "stopBit": 1
  },
```

## 6. Host Sets Modbus Slave Parameters

**Command:** `0x7305`

### Parameter List

| Parameter | Type | Required | Description |
|--------|------|------|------|
| type | string | Yes | Communication method; values "TCP", "RTU" |
| RTU | object | No | RTU parameters |
| TCP | object | No | TCP parameters |
| master-slave | int | Yes | Modbus communication master/slave mode; 0 means master, 1 means slave |
| scancycle | int | Yes | Scan cycle; range [8,1000]ms |
| stoprun | int | Yes | Operation on communication disconnect; 0 means no stop, 1 means stop |

#### RTU Parameters

| Parameter | Type | Required | Description |
|--------|------|------|------|
| baudrate | int | Yes | Baud rate |
| port | int | Yes | Port number |
| slaveId | int | Yes | Slave ID; range [0,65535] |

#### TCP Parameters

| Parameter | Type | Required | Description |
|--------|------|------|------|
| IP | string | Yes | IP address (default value) |
| endian_type | int | Yes | Float endianness; 0 means AB CD, 1 means CD AB, 2 means BA DC, 3 means DC BA |
| port | int | Yes | Port number; range [0,65535] |

### Request Example

```json
{
  "RTU": {
    "baudrate": 115200,
    "port": 2,
    "slaveId": 1
  },
  "TCP": {
    "IP": "192.168.1.11",
    "endian_type": 0,
    "port": 502
  },
  "master-slave": 1,
  "scancycle": 100,
  "stoprun": 0,
  "type": "TCP"
}
```
