# 티치 펜던트

## 소개

티치 펜던트 2차 개발은 티치 펜던트 소프트웨어의 정적 라이브러리가 제공하는 인터페이스를 기반으로 한 2차 개발입니다.

2차 개발 정적 라이브러리는 Windows, Linux, ARM-Linux(T30 티치 펜던트) 세 가지 시스템 소프트웨어 개발을 지원합니다:

Windows 버전 기반으로 호스트 컴퓨터 프로그램 개발 가능;

Linux 버전의 2차 개발 정적 라이브러리로 개발된 프로그램은 디버깅에 사용하거나 Linux 시스템 기반의 호스트 컴퓨터 소프트웨어를 생성할 수 있습니다;

ARM-Linux(T30) 버전의 2차 개발 정적 라이브러리로 생성된 프로그램은 본사 T30 티치 펜던트에서만 실행 가능합니다.

완전한 티치 펜던트 기본 기능을 제공하며, 사용자가 직접 필요한 프로세스 인터페이스를 커스터마이즈하고, 사용자 정의 명령어, 사용자 정의 인터페이스 삽입, 범용 데이터 송수신 인터페이스(통신 데이터 형식은 JSON 형식으로 통일)를 제공합니다.

## 학습 경로

티치 펜던트의 2차 개발을 학습하기 전에 다음 지식을 갖추고 있다고 가정합니다:

- C++ 언어;
- Qt 사용;
- Linux 시스템 (Ubuntu 배포판) 사용;
- 산업용 로봇 관련 기본 지식;
- QtCreator 편집기.
다음 지식이 없는 경우 다음 튜토리얼을 권장합니다:

- C++ 튜토리얼 - Qt 튜토리얼
- Ubuntu 완전 튜토리얼, Ubuntu 마스터가 되세요!
- Linux Ubuntu 기본 조작 명령어
- iNexBot 강좌
- iNexBot 제어 시스템 조작 매뉴얼
