# 2차 개발 API

## 2차 개발静态库소개

### 기능介绍

제공完整的티치 펜던트기능，지원추가사용자自定义인터페이스，지원特定필드的通信与제어시스템通信。

### 静态库디렉터리구조

![티치 펜던트 SDK 静态库디렉터리구조图，展示 nextpLib 下的 Include 头파일和 Library 平台库파일](assets/api-01.png)

nextpLib 为总파일夹；

Include 파일夹에서头파일；

Library 파일夹에서静态库파일包括 linux 平台和 ARM 平台的库파일（사용ARM 平台交叉编译的프로그램只适用于纳博特公司的 T30 티치 펜던트사용）；

### 静态库구조설명

#### 1.nextp.h 头파일인터페이스설명：

```cpp
//创建 Nextp 클래스객체
```

#### 2.json/json.h 头파일제공 JSON 데이터형식的组装和解析

组装 json 데이터예시:

```cpp
Json::Value root;
Json::FastWriter wt;
root["robot"] =1;
root["booldata"] =true;
root["data"] = 1.1;
root["name"] ="nihao";
```

解析 Json 데이터예시

```cpp
QByteArray jsonData //컨트롤러전송的데이터
Json::Value root;
Json::Reader reader;
QString jsonData = param.data();
if(reader.parse(jsonData.toStdString(), root))
{
    int robot = root["robot"].asInt();
    bool booldata= root["booldata"].asBool();
    Int data= root["data"].asDouble();
    Std::string name = root["name "].asString();
}
```
