## 사전 설명

컨트롤러에는 세 가지 작동 모드가 있습니다: 티치 모드, 실행 모드, 원격 모드. 티치 모드에서 모션을 수행하려면 전원 켜기 작업이 필요합니다. 실행 모드에서는 모션 모드를 열고 서보가 준비되었는지 확인하기만 하면 됩니다.
