# API 참조

C++ SDK API 전체 문서 (SWIG에 의해 자동 생성됨).

## 온라인 문서

https://doc.hmilib.inexbot.coision.cn/nrc__interface_8h.html

## 주요 헤더 파일

| 헤더 파일 | 설명 |
|--------|------|
| `cpp_interface/nrc_api.h` | 기본 연결, 모션 제어 인터페이스 |
| `cpp_interface/nrc_interface.h` | 모션 모드, 서보 제어 인터페이스 |
| `cpp_interface/nrc_queue_operate.h` | 큐 작업 인터페이스 |
| `parameter/command.h` | 명령어 및 매개변수 정의 |

## 포트 설명

| 포트 | 용도 |
|------|------|
| 6000 | 제어/조회: 전원 켜기/끄기, 모드 전환, 상태 조회 |
| 6001 | 티치 프로그래밍: 티치 모드에서 작업 편집, 프로그램 제어 |
| 7000 | 서보 추적 데이터: servoJ, servo_move, 포인트 위치 제어 |
