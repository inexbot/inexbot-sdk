# 사용자 정의 명령어 demo

이 demo의 버전은 rtl-22.07.64 버전이며, C1201 컨트롤러에 적용됩니다 (다른 버전의 SDK가 필요한 경우 관련 다운로드 내에서 해당 라이브러리를 다운로드하여 교체하시기 바랍니다).

## 다운로드 안내

컨트롤러와 티치 펜던트의 2차 개발 demo는 모두 **관련 다운로드 → 컨트롤러 SDK**에 있습니다.

## 디렉터리 구조

| 폴더 | 설명 |
|--------|------|
| 컨트롤러 구현 | 컨트롤러 측 2차 개발 demo, 컨트롤러 내에서 사용자 정의 명령어 콜백 함수를 등록하는 방법을 시연합니다 |
| 티치 펜던트 구현 | 티치 펜던트 측 2차 개발 demo, 사용자 정의 명령어 전용 UI 인터페이스를 작성하는 방법을 시연합니다 |
