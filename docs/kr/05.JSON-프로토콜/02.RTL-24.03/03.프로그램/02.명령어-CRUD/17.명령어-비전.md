# 명령어-비전명령클래스

## 비전시작

**설명**：실행비전시작명령어뒤컨트롤러및카메라연결

**명령어이름**：VISION_RUN

| 매개변수 | 유형 | 필수 | 설명 |
|------|------|------|------|
| ID | int | 예 | 프로세스번호，가져오기값을범위[1,99] |
| logout | bool | 예 | 주석 처리 여부：true=에서작업파일에 복사합니다값을아님실행해당명령어；false=미주석 처리됨，실행 가능 |
| type | int | 예 | VISION_RUN에서열거형수열type에 복사합니다이(가)79 |
| userParamInt | int | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

**JSON예시**：
```json
{
  "ID": 14,
  "logout": false,
  "type": 79,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 飞拍시작

**설명**：로봇도구모션经过拍照포인트일 때，快速触전송카메라拍照，에서拍照过程에 복사합니다物体또는카메라상태모션상태

**명령어이름**：ONFLY_INIT

| 매개변수 | 유형 | 필수 | 설명 |
|------|------|------|------|
| Delay | object | 예 | 记기록좌표지연，전역또는로컬double유형수동 입력 값또는int/double유형변수또는绑설정변수，가져오기값을범위[0,1000] |
| axisNum | object | 아니오 | 축，string유형수동 입력 값또는int유형전역또는로컬변수또는绑설정변수 |
| _posIdType | int | 예 | phonePos노드예포인트변수还예绑설정포인트변수：0=포인트변수；1=绑설정포인트변수 |
| IOPort | object | 아니오 | IO포트，int유형또는绑설정변수유형，가져오기값을범위[1,1000] |
| IOValue | object | 아니오 | 포트값을，bool유형수동 입력 값또는변수또는绑설정변수 |
| cameraTriggerDelayTime | object | 예 | 카메라触전송지연，double유형수동 입력 값또는int유형，가져오기값을범위[0,1000] |
| craftID | object | 예 | 프로세스번호，int유형，가져오기값을범위[1,99] |
| logout | bool | 예 | 주석 처리 여부：true=에서작업파일에 복사합니다값을아님실행해당명령어；false=미주석 처리됨，실행 가능 |
| triggerAngleSpacing | object | 아니오 | 触전송角도사이隔，double유형수동 입력 값또는int/double유형전역또는로컬변수또는绑설정변수，가져오기값을범위[-360,360] |
| triggerCount | object | 아니오 | 触전송회수，int유형수동 입력 값또는전역또는로컬绑설정변수또는변수，가져오기값을범위[0,10]；아님사용일 때`{"data":0.0,"secondvalue":0,"value":1,"varname":"[-]"}` |
| triggerStartPos | object | 아니오 | 触전송起시작위치，double유형수동 입력 값또는전역또는로컬int/double유형변수또는绑설정변수 |
| phonePos | string | 예 | 拍照포인트，string유형，전역또는로컬P포인트또는E포인트변수또는绑설정변수 |
| radius | object | 예 | 拍照포인트범위半径，가져오기값을범위[0.001,1000] |
| type | int | 예 | ONFLY_INIT에서열거형수열type에 복사합니다이(가)239 |
| userParamInt | int | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
