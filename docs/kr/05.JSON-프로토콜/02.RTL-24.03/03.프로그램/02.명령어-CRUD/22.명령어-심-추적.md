| logout | bool | - | 주석 처리 여부：true=在작업파일에서 실행되지실행该명령어；false=미주석 처리됨，실행 가능 |
| type | int | - | 열거형값，ARCVTRACKOFF 在열거형수열에서 161 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| variable | object | - | 변수데이터 |

### JSON 예시

```json
{
  "logout": false,
  "type": 161,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {"data": 0.0, "secondvalue": 0, "value": 4, "varname": "GI006"}
}
```
