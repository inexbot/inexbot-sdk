# 명령어-입력출력클래스

---

## DIN IO입력

**설명**：입력예相对于컨트롤러而言，由外部到컨트롤러。IO입력的명령어。

| 매개변수이름 | 유형 | 설명 |
|--------|------|------|
| logout | bool | 주석 처리 여부；true：작업 파일에서 실행되지 않음；false：주석 처리되지 않음, 실행 가능 |
| type | int | DIN在열거형수열type에서17 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| ParaGroupNum | object | 입력그룹 번호；int유형；값 범위범위和1共多少组有关，IO板값*16/几路입력 |
| ParaGroupValue | object | 포트값存入；변수유형，전역或로컬的bool或int型변수 |
| groupType | string | din유형；1路입력：IN#，4路입력：IGH#，8路입력：IG# |

**ParaGroupNum 구조**：

```json
{"data":2.0,"secondvalue":0,"value":0,"varname":""}
```

**ParaGroupValue 구조**：

```json
{"data":0.0,"secondvalue":0,"value":1,"varname":"I001"}
```

**예시**：

```json
{
    "logout": false,
    "type": 17,
    "userParamInt": 0,
    "userParamString": "",
    "ParaGroupNum": {"data": 2.0, "secondvalue": 0, "value": 0, "varname": ""},
    "ParaGroupValue": {"data": 0.0, "secondvalue": 0, "value": 1, "varname": "I001"},
    "groupType": "IGH#"
}
```

## DOUT IO출력

**설명**：将IO板上对应的IO포트置高或置低。IO출력的명령어。

| 매개변수이름 | 유형 | 설명 |
|--------|------|------|
| logout | bool | 주석 처리 여부；true：작업 파일에서 실행되지 않음；false：주석 처리되지 않음, 실행 가능 |
| type | int | DOUT在열거형수열type에서16 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| ParaGroupNum | object | 입력그룹 번호；int유형；값 범위범위和1共多少组有关，IO板값*16/几路출력 |
| ParaGroupValue | object | 포트값출력；변수유형，전역或로컬的bool或int型변수 |
| groupType | string | dout유형；1路출력：OT#，4路출력：OGH#，8路출력：OG# |
| errorHanding | int | 1：计时종료정지，0：출력값保持 |
| ParaGroupTime | object | 시간；double유형；범위[0,9999] |

**ParaGroupNum 구조**：

```json
{"data":2.0,"secondvalue":0,"value":0,"varname":""}
```

**ParaGroupValue 구조**：

```json
{"data":0.0,"secondvalue":0,"value":1,"varname":"I001"}
```

**ParaGroupTime 구조**：

```json
{"data":3.0,"secondvalue":0,"value":0,"varname":""}
```

**예시**：

```json
{
    "logout": false,
    "type": 16,
    "userParamInt": 0,
    "userParamString": "",
    "ParaGroupNum": {"data": 2.0, "secondvalue": 0, "value": 0, "varname": ""},
    "ParaGroupValue": {"data": 0.0, "secondvalue": 0, "value": 1, "varname": "I001"},
    "errorHanding": 1,
    "groupType": "OT#",
    "ParaGroupTime": {"data": 3.0, "secondvalue": 0, "value": 0, "varname": ""}
}
```

## AIN 시뮬레이션입력

**설명**：입력예相对于컨트롤러而言，由外部到컨트롤러。시뮬레이션입력的명령어。


## PULSEOUT 脉冲출력

**설명**：按照设定的脉冲주파수与번째수，在R1 PWMIO板上的DB9端子的引脚4（PWM+）上进줄출력。

| 매개변수이름 | 유형 | 설명 |
|--------|------|------|
| logout | bool | 주석 처리 여부；true：작업 파일에서 실행되지 않음；false：주석 처리되지 않음, 실행 가능 |
| type | int | PULSEOUT在열거형수열type에서22 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| rate | int | 주파수；int유형；범위[1,1000000] |
| sum | int | 번째수；int유형；범위大于零 |

**예시**：

```json
{
    "logout": false,
    "type": 22,
    "userParamInt": 0,
    "userParamString": "",
    "rate": 10,
    "sum": 100
}
```

| ParaGroupValue | object | 포트값출력；변수유형，전역或로컬的bool或int型변수 |
| groupType | string | dout유형；1路출력：OT#，4路출력：OGH#，8路출력：OG# |
