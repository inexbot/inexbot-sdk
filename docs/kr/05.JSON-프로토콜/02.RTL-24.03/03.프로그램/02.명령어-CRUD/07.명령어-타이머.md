# 명령어-타이머 유형

## 지연

**설명**: 설정된 지연 시간에 도달하면 프로그램이 계속 실행됩니다

| 매개변수 | 유형 | 필수 | 설명 | 예시 |
|--------|------|------|------|------|
| logout | bool | 예 | 주석 처리 여부; true: 작업 파일에서 실행되지 않음; false: 주석 처리되지 않음, 실행 가능 | false |
| paraTime | object | 예 | 지연 지속 시간 | {"data":500.0,"secondvalue":0,"value":0,"varname":""} |
| paraTime.data | double | 아니오 | 지연 값; 값 범위 (0,9999]; 단위 s | 500.0 |
| paraTime.value | int | 아니오 | 변수 값 (예비) | 0 |
| paraTime.secondvalue | int | 아니오 | 변수 값 (예비) | 0 |
| paraTime.varname | string | 아니오 | 변수명 (변수 사용 시 입력) | "" |
| type | int | 예 | TIMER는 열거형에서 7 | 7 |
| userParamInt | int | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 | 0 |
| userParamString | string | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 | "" |

**명령어**: TIMER

**JSON 예시**:

```json
{
  "logout": false,
  "paraTime": {
    "data": 500.0,
    "secondvalue": 0,
    "value": 0,
    "varname": ""
  },
  "type": 7,
  "userParamInt": 0,
  "userParamString": ""
}
```
