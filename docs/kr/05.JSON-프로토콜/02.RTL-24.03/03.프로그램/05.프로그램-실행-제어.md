# 프로그램실행제어

## 프로그램실행총 수清零

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 선택로봇，값 범위범위 [1, 4] |

```json
{
  "robot": 1
}
```

## 실행프로그램

对프로그램进줄조작。

### 전송시작실행프로그램

**명령어：** `0x5601`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要조작的로봇 번호，값 범위범위 [1, 4] |
| jobname | string | 프로그램이름 |
| suffixname | string | 프로그램이름后缀이름，값 범위：.JBR 主프로그램，.JBP 백그라운드로컬프로그램，.JBPG 백그라운드전역프로그램 |
| stepRunSum | int | 1共多少步（선택） |
| callerName | string | 访问者이름字（선택） |
| stepRun | bool | 여부싱글 스텝실행（선택） |
| index | int | 인덱스（선택） |
| noFileRun | bool | 없음파일실행（선택） |
| notJudgeSafePos | bool | 아님判断안전위치（선택） |
| line | int | 프로그램시작실행的줄번호 |
| globalStart | bool | 여부을/를多机모드 |
| continueRun | int | 여부을/를계속실행：0 아님계속실행，1 계속실행 |
| startOver | int | 여부을/를重头실행：0 계속실행，1 重头실행 |

```json
{
  "robot": 1,
  "jobname": "TTT",
  "suffixname": ".JBR",
  "stepRunSum": 2,
  "callerName": "tttt",
  "stepRun": false,
  "index": 0,
  "noFileRun": false,
  "notJudgeSafePos": false,
  "line": 1,
  "globalStart": false,
  "continueRun": 0,
  "startOver": 0
}
```

> 注：선택필드아님使用없음影响

## 프로그램실행상태설정

호스트 컴퓨터필요改变현재컨트롤러프로그램실행모드时전송。

**명령어：** `0x5603`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 선택로봇，값 범위범위 [1, 4] |
| type | int | 프로그램실행상태：0 싱글 스텝실행，1 일시정지，2 정지但아님전원 끄기，5 정지但아님전원 끄기（정지优先使用） |

```json
{
  "robot": 1,
  "type": 5
}
```

> 注意：对于싱글 스텝，没有전체로봇；对于일시정지和정지，只有多机모드下有전체로봇，此时将 "robot" 노드설정成 0

## 추가 쓰기모션모드

추가 쓰기모션모드예在아님使用프로그램的条件下실행프로그램。

使用메서드：
1. 열기추가 쓰기모션모드
2. 추가 쓰기모션모드삽입명령어向量
3. 추가 쓰기모션모드설정

### 열기추가 쓰기모션모드

**명령어：** `0x5640`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要조작的로봇 번호，값 범위범위 [1, 4] |
| open | bool | 여부开启추가 쓰기모션모드，true 열기，false 닫기 |

```json
{
  "robot": 1,
  "open": true
}
```

### 추가 쓰기모션삽입명령어向量

**명령어：** `0x5643`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要조작的로봇 번호，값 범위범위 [1, 4] |
| isContinue | bool | true 只수신并存储 pos，false 수신 pos 并使得로봇실행所有 pos |
| data | array | 명령어 json 열테이블，형식을/를 cmd->toJson() |

```json
{
  "data": [
    { "cmd": "toJson()" },
    { "cmd": "toJson()" },
    { "cmd": "toJson()" }
  ],
  "isContinue": false,
  "robot": 1
}
```

> 명령어详情请参阅명령어页面

## 自시작

自시작프로그램，설정后开机自시작。

### 설정自시작프로그램

**명령어：** `0x5647`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要조회的로봇 번호，값 범위범위 [1, 4] |
| enable | bool | 여부활성화，true 활성화，false 아님使用 |
| count | int | 실행회수 |
| speed | int | 실행속도 |
| jobname | string | 프로그램이름字 |

```json
{
  "robot": 1,
  "enable": false,
  "count": 1,
  "speed": 15,
  "jobname": "Q1"
}
```

**컨트롤러응답**
