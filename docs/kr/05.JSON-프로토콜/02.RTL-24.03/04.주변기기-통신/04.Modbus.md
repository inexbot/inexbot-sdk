# Modbus

## 1. 호스트 컴퓨터설정modbus마스터매개변수

**명령어：** `0x7300`

### 매개변수 목록

| 매개변수이름 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| masterStation | object | 예 | 마스터매개변수，包含RTU매개변수和TCP매개변수 |
| startAddress | bool | 아니오 | 주소偏移；true의미주소아님变，false의미주소自动-1 |

#### masterStation 매개변수

| 매개변수이름 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| type | string | 예 | 마스터유형；값 범위"TCP","RTU" |
| processNumber | int | 예 | 필요설정的프로세스번호；범위[1,9] |
| RTU | object | 아니오 | RTU매개변수 |
| TCP | object | 아니오 | TCP매개변수 |

#### RTU 매개변수

| 매개변수이름 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| baudrate | int | 예 | Baud Rate |
| checkBit | string | 예 | 검증비트；값 범위"N","E","O" |
| dataBit | int | 예 | 데이터비트；범위[5,8] |
| port | int | 예 | 포트 번호 |
| slaveId | int | 예 | 슬레이브ID；범위[0,65535] |
| stopBit | int | 예 | 정지비트；범위[0,1] |

#### TCP 매개변수

| 매개변수이름 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| IP | string | 예 | IP주소 |
| endian_type | int | 예 | Float大小端；0의미AB CD，1의미CD AB，2의미BA DC，3의미DC BA |
| port | int | 예 | 포트 번호；범위[0,65535] |

### 요청 예시

```json
{
  "masterStation": {
    "RTU": {
      "baudrate": 115200,
      "checkBit": "E",
      "dataBit": 5,
      "port": 2,
      "slaveId": 1,
      "stopBit": 1
    },
    "TCP": {
      "IP": "192.168.1.14",
      "endian_type": 1,
      "port": 503
    },
    "processNumber": 1,
    "type": "TCP"
  },
  "startAddress": false
}
```
