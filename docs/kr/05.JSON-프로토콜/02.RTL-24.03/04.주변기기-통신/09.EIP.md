## 0x7360 호스트 컴퓨터 설정设备매개변수配置

### 매개변수 목록

| 매개변수 | 유형 | 필수 | 설명 | 값 범위범위 |
|--------|------|------|------|----------|
| type | int | 예 | 几路출력(入) | 1 / 4 / 8 / 12 / 16 |
| grouptype | int | 예 | 그룹 번호的방식 | 0: 手填，1: 전역int유형변수，2: 전역int유형바인딩변수 |
| group | int | 예 | 그룹 번호中的숫자 | - |
| addtype | int | 예 | 값存入的방식 | 0: 전역int유형변수，1: 전역int유형바인딩변수，2: 전역double유형변수，3: 전역double유형바인딩변수 |
| add | int | 예 | 값存入的숫자 | - |
| num | int | 예 | 번호 | 1-10: 출력，11-20: 입력 |
| value | - | 예 | 값 | - |
| able | bool | 예 | 활성화스위치 | true: 활성화，false: 아님활성화 |

### 요청 예시

```json
{
  "type": 4,
  "grouptype": 1,
  "group": 3,
  "addtype": 2,
  "add": 100,
  "num": 5,
  "value": 1,
  "able": true
}
```

### 응답

컨트롤러응답并반환设备매개변수

## 0x7363 호스트 컴퓨터 설정연결매개변수

### 매개변수 목록

| 매개변수 | 유형 | 필수 | 설명 | 값 범위범위 |
|--------|------|------|------|----------|
| enable | bool | 예 | 연결스위치 | true: 열기연결스위치，false: 닫기연결스위치 |
| writenum | int | 예 | 쓰기길이 | [0, 256] |
| readnum | int | 예 | 읽기길이 | [0, 256] |
| scancycle | - | 예 | 扫描주기 | - |
| outtimecycle | - | 예 | 타임아웃주기 | - |
| networkname | string | 예 | 选取的网口的이름 | - |
| writetype | int | 예 | 本机写的유형 | 0: 전역bool유형변수，1: 전역bool유형바인딩변수 |
| writeadd | - | 예 | 本机写的위치 | - |
| readtype | int | 예 | 本机读的유형 | 0: 전역bool유형변수，1: 전역bool유형바인딩변수 |
| readadd | - | 예 | 本机读的위치 | - |

### 요청 예시

```json
{
  "enable": true,
  "writenum": 0,
  "readnum": 0,
  "scancycle": 0,
  "outtimecycle": 0,
  "networkname": "",
  "writetype": 0,
  "writeadd": 0,
  "readtype": 0,
  "readadd": 0
}
```
