# IO

## 상태 알림

> 설명：알림 포트 범위 최대값은 모두 현재 바인딩된 IO 수량；출력 방식 범위는 모두 [0,2], 0은 0, 1은 1, 2는 깜빡임

### 호스트 컴퓨터 설정 상태 알림 매개변수

**명령어：** `0x7100`

**매개변수 설명：**

|--------|------|------|

| outPut[].IOenable | int | 로봇 전원 인가 상태 알림 포트 |

## IO 리셋

> 설명："enable"과 "value" 노드 길이는 모두 128이며, 실제 유효 데이터는 현재 IO의 총 포트 수이고, 초과 부분은 기본값 0

### 호스트 컴퓨터 설정 IO 리셋 매개변수

**명령어：** `0x7110`

**매개변수 설명：**

## IO 구성

> 설명：시리얼 아날로그 IO 매개변수（EtherCAT IO 보드에 아날로그 IO가 있으면 해당 시리얼 아날로그 IO는 무효）

### 호스트 컴퓨터 설정 IO 구성 매개변수

**명령어：** `0x7130`

**매개변수 설명：**

| 매개변수 이름 | 유형 | 설명 |
|--------|------|------|
| serialAnalog | object | 시리얼 아날로그 매개변수 |
| serialAnalog.baudRate | int | 보드레이트 |
| serialAnalog.port | int | 포트 번호, 범위 (0,65535] |
| serialAnalog.type | string | 아날로그 유형<br/>"DAC_ANAIO"：DAC 아날로그 IO 보드<br/>"SUPER_ANAIO"：격제어 아날로그 IO 보드 |
| simuNum | int | 가상 IO 수량, 범위 [0,4] |

**요청 예시：**

```json
{
  "serialAnalog":
  {
    "baudRate":115200,
    "port":2,
    "type":"SUPER_ANAIO"
  },
  "simuNum":2
}
```

### 컨트롤러 회신 호스트 컴퓨터 IO 구성 매개변수

**명령어：** `0x7132`

**매개변수 설명：**

| 매개변수 이름 | 유형 | 설명 |
|--------|------|------|
| num | int | IO 수량（실제 IO와 가상 IO의 총 수량 포함） |
| portNum | array | 각 IO 포트 수 2차원 배열；int 유형<br/>1차원：IO 번호, 2차원 순서대로：디지털 입력 포트 수, 디지털 출력 포트 수, 아날로그 입력 포트 수, 아날로그 출력 포트 수 |
| serialAnalog | object | 시리얼 아날로그 매개변수 |
| serialAnalog.baudRate | int | 보드레이트 |
| serialAnalog.port | int | 포트 번호, 범위 (0,65535] |
