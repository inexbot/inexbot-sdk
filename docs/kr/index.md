# 나보트 테크놀로지 지식 베이스

나보트 테크놀로지(Nabot Technology)는 2015년에 설립되었으며, 중국 고엔드 모션 제어 분야의 기술 선구자입니다. 우리는 "소프트웨어 정의 제어, 알고리즘 기반 지능"을 핵심 철학으로 삼아, 저수준 하드웨어에서 미들웨어 플랫폼, 상위 애플리케이션에 이르는 풀스택 기술 체계를 제공합니다.

## 지식 베이스 디렉터리 구조

본 저장소는 기술 영역과 응용 시나리오별로 구성되어 있어, 개발자와 엔지니어가 빠르게 검색할 수 있도록 합니다.

- **컨트롤러** — 컨트롤러 2차 개발 API, 예제 및 커스텀 명령
- **티칭 펜던트** — 티칭 펜던트 2차 개발 API 및 빠른 시작
- **상위 PC** — C++/C#/Python 입문형 튜토리얼 및 인터페이스 예제
- **마스터 스테이션 라이브러리** — 마스터 스테이션 통신 라이브러리 설명
- **JSON 프로토콜** — RTL-22.07 JSON 통신 프로토콜 레퍼런스
- **ROS** — ROS-1 / ROS-2 통합 가이드
