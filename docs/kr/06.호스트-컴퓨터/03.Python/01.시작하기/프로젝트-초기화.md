# 프로젝트 초기화

자신의 Python 버전에 맞는 관련 다운로드 항목에서 해당 Python 버전의 라이브러리를 다운로드해야 합니다. 버전이 일치하지 않으면 nrc_host.so 파일을 찾을 수 없다는 오류가 발생합니다.

다운로드한 _nrc_host.so와 nrc_interface.py를 프로젝트에 넣습니다.
