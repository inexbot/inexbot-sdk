# 고급 애플리케이션

## servoJ 관절 제어

6000 포트(제어/조회)와 7000 포트(서보 추적)에 연결하고, 10ms마다 목표 관절 각도를 전송하여 부드러운 추적 운동을 구현합니다.

## 서보 포인트 위치 제어

각 통신 주기마다 포인트를 서보에 직접 전송합니다. 단일 포인트 전송(sum=1)과 일괄 전송(프레임당 최대 600개 포인트, count가 sum까지 누적된 후 운동 시작)을 지원합니다.
