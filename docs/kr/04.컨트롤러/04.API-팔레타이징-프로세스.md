# 2차 개발API（팔레타이징프로세스）

本章节介绍팔레타이징프로세스매개변수설정与제어인터페이스。

## 函数索引

| 函数 | 설명 |
|------|------|
| NRC_Pallet_SetUsePalletType / NRC_Pallet_GetUsePalletType | 简易/完整팔레타이징切换 |
| NRC_Pallet_SetGripperParm / NRC_Pallet_GetGripperParm | 抓手매개변수 |
| NRC_Pallet_SetSimplePosParm / NRC_Pallet_GetSimplePosParm | 简易팔레타이징위치매개변수 |
| NRC_Pallet_SetSimpleNumParm / NRC_Pallet_GetSimpleNumParm | 简易팔레타이징数量매개변수 |
| NRC_Pallet_ClearCurrentStatus | 清除팔레타이징상태 |
| NRC_Pallet_SetPalletedWpNum / NRC_Pallet_GetPalletedWpNum | 已码工件数量 |

### NRC_Pallet_SetUsePalletType

| 函数原型 | int NRC_Pallet_SetUsePalletType(int id, int type); |
| --- | --- |
| 기능설명 | 설정使用简易팔레타이징还是完整팔레타이징。 |
| 매개변수 설명 | 输入매개변수：<br/>id  팔레타이징프로세스的프로세스号，매개변수범위：1 <= id <= 99 &lt; br/>type  使用的팔레타이징유형  0-简易팔레타이징  1-完整팔레타이징 |

| 备注설명 | 无 |
