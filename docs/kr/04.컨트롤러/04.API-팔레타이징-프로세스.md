# 2차 개발API（팔레타이징프로세스）

본 절에서는팔레타이징프로세스매개변수설정및제어인터페이스。

## 함수인덱스

| 함수 | 설명 |
|------|------|
| NRC_Pallet_SetUsePalletType / NRC_Pallet_GetUsePalletType | 간이/전체팔레타이징전환 |
| NRC_Pallet_SetGripperParm / NRC_Pallet_GetGripperParm | 그리퍼매개변수 |
| NRC_Pallet_SetSimplePosParm / NRC_Pallet_GetSimplePosParm | 간이팔레타이징위치매개변수 |
| NRC_Pallet_SetSimpleNumParm / NRC_Pallet_GetSimpleNumParm | 간이팔레타이징수량매개변수 |
| NRC_Pallet_ClearCurrentStatus | 해제팔레타이징상태 |
| NRC_Pallet_SetPalletedWpNum / NRC_Pallet_GetPalletedWpNum | 완료적재공작물수량 |

### NRC_Pallet_SetUsePalletType

| 함수原유형 | int NRC_Pallet_SetUsePalletType(int id, int type); |
| --- | --- |
| 기능설명 | 설정사용간이팔레타이징还예전체팔레타이징。 |
| 매개변수 설명 | 입력매개변수：<br/>id  팔레타이징프로세스프로세스번호，매개변수범위：1 <= id <= 99 &lt; br/>type  사용 중인팔레타이징유형  0-간이팔레타이징  1-전체팔레타이징 |
| 반환값을 | 0：함수호출성공반환<br/>-101：없음效입력매개변수반환<br/>-102：대상객체존재하지 않음，일반표시되면에서시스템초기화未완료일 때，호출기타함수일 때가능가능반환해당값을반환<br/>-103：대상객체현재상태불가조작상태반환 |
| 비고설명 | 없음 |
