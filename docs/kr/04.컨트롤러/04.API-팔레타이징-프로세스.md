# 2차 개발API（팔레타이징프로세스）

本章节介绍팔레타이징프로세스매개변수설정与제어인터페이스。

## 함수인덱스

| 함수 | 설명 |
|------|------|
| NRC_Pallet_SetUsePalletType / NRC_Pallet_GetUsePalletType | 简易/完整팔레타이징전환 |
| NRC_Pallet_SetGripperParm / NRC_Pallet_GetGripperParm | 抓手매개변수 |
| NRC_Pallet_SetSimplePosParm / NRC_Pallet_GetSimplePosParm | 简易팔레타이징위치매개변수 |
| NRC_Pallet_SetSimpleNumParm / NRC_Pallet_GetSimpleNumParm | 简易팔레타이징数量매개변수 |
| NRC_Pallet_ClearCurrentStatus | 해제팔레타이징상태 |
| NRC_Pallet_SetPalletedWpNum / NRC_Pallet_GetPalletedWpNum | 已码공작물数量 |

### NRC_Pallet_SetUsePalletType

| 함수原型 | int NRC_Pallet_SetUsePalletType(int id, int type); |
| --- | --- |
| 기능설명 | 설정使用简易팔레타이징还예完整팔레타이징。 |
| 매개변수 설명 | 입력매개변수：<br/>id  팔레타이징프로세스的프로세스号，매개변수범위：1 <= id <= 99 &lt; br/>type  使用的팔레타이징유형  0-简易팔레타이징  1-完整팔레타이징 |
| 반환值 | 0：함수호출성공반환<br/>-101：없음效的입력매개변수반환<br/>-102：目标객체존재하지 않음，一般出现在시스템초기화未완료时，호출其他함수时可能반환该值반환<br/>-103：目标객체현재处于不可조작상태반환 |
| 备注설명 | 없음 |
