# 인터페이스 소개

다음 인터페이스는 모두 티치 펜던트 SDK의 헤더 파일에서 찾을 수 있습니다

# 사용자 정의 명령어의 신호 인터페이스, 슬롯 함수와 함께 사용해야 합니다

```cpp
/**
 * @brief 사용자 정의 명령어 인터페이스 삽입 호출 신호
 * @note 신호는 사용자 정의 인터페이스 초기화에 사용되며, 작업 파일 명령어 삽입 시 표시되는 인터페이스입니다
 * @param cmdNum 번호, 삽입 시 사용자 정의 명령어 클래스 내 위에서 아래로 몇 번째에 해당하는지
 */
void signal_userdefine_cmd_init(int cmdNum);
/**
 * @brief 사용자 정의 명령어 인터페이스 수정 호출 신호
 * @note 신호는 사용자 정의 명령어 수정에 사용되며, 작업 파일 명령어 수정 시 표시되는 인터페이스입니다
 * @param cmdNum 번호, 삽입 시 사용자 정의 명령어 클래스 내 위에서 아래로 몇 번째에 해당하는지
 * @param cmdParam 삽입 시 전달되는 문자열 매개변수, 현재 문자열 매개변수 표시에 사용
 * @param posName 삽입 시 전달되는 위치 이름, 선택 사항
*/
void signal_userdefine_cmd_alter(int cmdNum, QString cmdParam, QString posName);
```

# 사용자 정의 명령어 인터페이스

```cpp
/**
 * @brief 사용자 정의 명령어 삽입
 * @param cmdNum 번호, 삽입 시 사용자 정의 명령어 클래스 내 위에서 아래로 몇 번째에 해당하는지
 * @param cmdParam 전달되는 사용자 정의 문자열 매개변수
 */
void userdefine_cmd_insert(int cmdNum, QString cmdParam);
/**
 * @brief 삽입 인터페이스에 표시되는 사용자 정의 명령어 이름 수정
 * @param EnglishName 영어 이름
 * @param ChineseName 중국어 이름
 * @note 삽입 수량이 사용자 정의 명령어의 개수를 결정합니다
 * @note connect를 사용하여 신호 signal_userdefine_cmd_init(int)에 인터페이스를 바인딩한 후 본 매개변수를 호출하시기 바랍니다, 그렇지 않으면失效 합니다
 */
void userdefined_cmd_name_change(QStringList &EnglishName,QStringList &ChineseName);
/**
 * @brief 사용자 정의 명령어 인터페이스 닫기
 * @note 사용자 정의 위치 삽입 및 수정을 취소할 때는 시스템이 플래그 위치를 올바르게 처리하도록 본 인터페이스를 호출하시기 바랍니다
 */
void close_userdefine_cmd_widget(QWidget *);
```
## userdefined_cmd_name_change : 사용 예시

```cpp
QStringList ENname ;
ENname <<"SWITCHGUN"<<"SWITCHGUNPOS";

QStringList CNname;
CNname << tr("换枪动作")<<tr("移动到换枪位置");
Nextp::getInstance()->userdefined_cmd_name_change(ENname, CNname);
```
