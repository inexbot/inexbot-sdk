# 2차 개발API（작업파일，명령어조작）

本章节介绍작업파일管理、명령어插入与실행队列제어的全部인터페이스。

## 函数分类索引

### 실행队列插入명령어

对应작업파일插入명령어，通过 `NRC_RunqueueInsert*` 系列函数将명령어插入无파일실행队列，매개변수与작업파일版本基本一致（无需 `line` 매개변수）。
