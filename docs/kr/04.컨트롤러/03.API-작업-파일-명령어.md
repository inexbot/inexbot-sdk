# 2차 개발API（작업파일，명령어조작）

本章节介绍작업파일管理、명령어삽입与실행队열제어的전체인터페이스。

## 함수分클래스인덱스

### 작업파일管理

| 함수 | 설명 |
|------|------|
| NRC_CreateJobfile / NRC_OpenJobfile / NRC_DeleteJobfile | 新建/열기/삭제작업파일 |
| NRC_JudgeJobIsExist | 判断작업파일여부존재 |
| NRC_GetJobfileLineSum | 가져오기작업파일总줄数 |
| NRC_JobfileEraseCmd | 삭제작업파일中指定줄的명령어 |

### 작업파일실행제어

| 함수 | 설명 |
|------|------|
| NRC_StartRunJobfile | 시작/계속실행작업파일 |
| NRC_PauseRunJobfile | 일시정지실행작업파일 |
| NRC_StopRunJobfile | 정지실행작업파일（从头시작） |
| NRC_StopRunJobfileNotPoweroff | 정지실행（不下활성화） |
| NRC_StepRunJobfile | 单步실행一줄명령어 |
| NRC_GetCurrentOrderRuns | 가져오기실행方向（顺序/倒序） |
| NRC_GetDistanceToOldTrack | 가져오기距原궤적정지点距离 |
| NRC_GetCycleIndex / NRC_GetCycleCount / NRC_GetCycleTimeSec | 循环회数/时间조회 |
| NRC_SetCycleIndex | 설정循环회数 |

### 局部后台프로그램

| 함수 | 설명 |

| NRC_DeletePthreadJobfile / NRC_JudgePthreadJobIsExist | 삭제/判断后台프로그램 |

### 실행队열삽입명령어

对应작업파일삽입명령어，通过 `NRC_RunqueueInsert*` 系열함수将명령어삽입없음파일실행队열，매개변수与작업파일版本基本一致（없음需 `line` 매개변수）。

### 상태조회

| 함수 | 설명 |
|------|------|
| NRC_GetProgramRunStatus | 가져오기프로그램실행상태（0-정지 1-일시정지 2-실행） |
| NRC_GetRobotRunStatus | 가져오기로봇모션상태 |
| NRC_GetTeachBoxConnectStatus | 가져오기示教盒연결상태 |
| NRC_GetOpenJobFileName | 가져오기현재열기的작업파일名 |
| NRC_GetRunqueueCurrentRunLine | 가져오기队열현재실행줄号 |
| NRC_GetJobfileCurrentRunLine | 가져오기작업파일현재실행줄号 |
| NRC_GetCurrentInstrRestPosNum | 가져오기현재명령어剩余点数 |

### 콜백 함수

| 함수 | 설명 |
|------|------|
| NRC_SetCompleteOneInstrCallBack | 설정명령어완료콜백（없음参/带参两种重载） |
