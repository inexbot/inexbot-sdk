# 언어

## 지원 언어 조회

**명령어:** `0x1052`

**요청 파라미터:** 없음

**컨트롤러 응답**

**명령어:** `0x1053`

| 필드 | 타입 | 설명 |
|------|------|------|
| language | string[] | 컨트롤러가 지원하는 언어 목록 |
| language_used | string | 컨트롤러가 현재 사용 중인 언어 |

**언어 값:**

| 언어 코드 | 설명 |
|----------|------|
| Chinese | 중국어 |
| Korean | 한국어 |
| English | 영어 |

**응답 예시:**

```json
{
  "language": ["Chinese", "Korean", "English"],
  "language_used": "Chinese"
}
```

## 상위 시스템 현재 언어 조회

**명령어:** `0x1054`

**요청 파라미터:**

| 필드 | 타입 | 설명 |
|------|------|------|
| teachbox_language | string | 상위 시스템이 현재 사용 중인 언어 |

**요청 예시:**

```json
{
  "teachbox_language": "Chinese"
}
```

## 지원되는 언어 유형

| 언어 코드 | 설명 |
|----------|------|
| Chinese | 중국어 |
| Korean | 한국어 |
| English | 영어 |
