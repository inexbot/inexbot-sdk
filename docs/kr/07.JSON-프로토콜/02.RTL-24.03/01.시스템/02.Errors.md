# 报错

## 清错

해제서보오류时使用。

### 해제서보오류

**명령어:** `0x1100`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要操作的로봇 번호，값 범위범위 [1, 4] |

```json
{
  "robot": 1
}
```

**컨트롤러 응답**

**명령어:** `0x1101`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 로봇 번호，값 범위범위 [1, 4] |
| clearErrorFlag | bool | 예아니오해제성공，true 해제서보오류성공，false 해제실패 |

```json
{
  "robot": 1,
  "clearErrorFlag": true
}
```

## 정보알림

컨트롤러向호스트 컴퓨터 전송오류알림/경고알림/메시지알림/팝업알림。

### 오류알림

**명령어:** `0x1110`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 로봇 번호，값 범위범위 [0, 4]，等于 0 时의미미선택로봇유형或者오류内容与로봇없음关 |
| kind | int | 팝업알림种클래스：0 메시지，1 경고，2 报错 |
| data | string | 메시지条内容，아님同的오류对应아님同的内容 |
| code | int | 메시지编码 |
| param | int[] | 메시지内容매개변수，根据具体编码的아님同会有아님同번째수，预留 5 번째 |

```json
{
  "code": 28680,
  "data": "명령어[0x5743]존재하지 않음",
  "kind": 2,
  "param": [22339, 3],
  "robot": 0
}
```

### 경고알림

**명령어:** `0x1111`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 로봇 번호，값 범위범위 [0, 4]，等于 0 时의미미선택로봇유형或者경고内容与로봇없음关 |
| kind | int | 팝업알림种클래스：0 메시지，1 경고，2 报错 |
| data | string | 메시지条内容，아님同的경고对应아님同的内容 |
| code | int | 메시지编码 |
| param | int[] | 메시지内容매개변수，根据具体编码的아님同会有아님同번째수，预留 5 번째 |

```json
{
  "code": 16385,
  "data": "로봇1매개변수配置파일존재하지 않음，已创建，请配置매개변수",
  "kind": 1,
  "param": [1],
  "robot": 1
}
```

### 메시지알림

**명령어:** `0x1112`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 로봇 번호，값 범위범위 [0, 4]，等于 0 时의미미선택로봇유형或者메시지内容与로봇없음关 |
| kind | int | 팝업알림种클래스：0 메시지，1 경고，2 报错 |
| data | string | 메시지条内容，아님同的메시지对应아님同的内容 |
| code | int | 메시지编码 |

```json
{
  "code": 21764,
  "data": "自动백업성공",
  "kind": 0,
  "robot": 0
}
```

### 팝업알림

**명령어:** `0x1113`

| 필드 | 유형 | 설명 |
|------|------|------|
| count | int | 팝업按钮수량，최대을/를 3 |
| text1 | string | 팝업居中显示的主文本 |
| text2 | string | 按钮 1 的文本 |
| text3 | string | 按钮 2 的文本 |
| text4 | string | 按钮 3 的文本 |
| kind | int | 팝업알림种클래스：0 메시지，1 경고，2 报错 |
| node | int | 로봇序번호，값 범위범위 [1, 4] |
| Tips | bool | 팝업呼出，true 의미呼出 |

```json
{
  "count": 1,
  "text1": "예시",
  "text2": "예시",
  "text3": "예시",
  "text4": "예시",
  "kind": 1,
  "node": 1,
  "Tips": true
}
```

### 팝업交互反馈

根据用户对팝업的操作进줄相应설정。

**명령어:** `0x1114`

| 필드 | 유형 | 설명 |
|------|------|------|
| node | int | 로봇序번호，값 범위범위 [1, 4] |
| option | int | 对应팝업选项 1-3 的内容，값 범위범위 [1, 3] |

```json
{
  "node": 1,
  "option": 1
}
```
