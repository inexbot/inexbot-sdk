# 조작매개변수

## 1. 실행매개변수설정

**명령字：** `0x1300`

### 매개변수列表

| 매개변수名 | 유형 | 默认值 | 설명 |
|--------|------|--------|------|
| `runModeAutoManBegFlag` | bool | false | 切到실행모드自动上电；true: 自动上电；false：不自动上电 |
| `remoteUseReserveModeFlag` | bool | false | 是否为预约모드；true: 预约모드；false：非预约모드 |
| `remoteAllowContinueRunFlag` | bool | true | 远程IO断点执行；true: 断电执行；false：断电不执行 |
| `remoteAllowCurrentRunFlag` | bool | false | 远程IO当前行执行；true: 当前行执行；false：非当前行执行 |
| `remoteSwitchModeEnable` | bool | false | 无티치 펜던트切远程모드；true: 允许；false：不允许 |
| `allowReserveRunningPro` | bool | false | 远程IO프로그램실행中再次预约；true: 允许；false：不允许 |
| `SetSafeScreenTeachInvalidFlag` | bool | false | 示教모드安全光幕不生效；true: 不生效；false：生效 |
| `keepStepRunFlag` | int | 0 | 单步/回零/复位点/실행조작方式；0：单击실행；1：按下실행 |
| `defaultRunningSpeed` | int | 1 | 실행모드开机默认속도 |
| `language` | string | "chinese" | 설정默认语言；"chinese"：中文；"english"：英文；"korean"：韩文 |
| `accUnitMode` | int | 0 | 关节/笛卡尔매개변수中最大加속도单位；0: 倍数值；1: 数值 |
| `craftSelect` | string | "general" | 프로세스选择；"general"：通用프로세스；"special"：专用프로세스；"punch"：冲压프로세스；"pallet"：팔레타이징 프로세스；"weld"：용접 프로세스；"cut"：切割프로세스 |
| `base_coord_of_user_coord_` | int | 0 | 사용자 좌표系基准좌표；0: 직교 좌표系；1: 大地좌표系 |
| `remote_io_reset_clear_all_reserve_program_status_flag` | bool | true | 远程IO复位清空预约队列；true: 清空；false：不清空 |
| `autoSetUserAndTool` | bool | false | 点位形态/도구手/사용자值설정方式；true: 自动；false: 手动 |
| `checkServoStatusWordMaxTime` | int | 0 | 伺服응답时间；取值범위 [0, 100]，单位 ms |
| `OutageKeepFunctionFlag` | bool | true | 断电保持标志位；true: 保持；false: 不保持 |
| `quickInsert` | bool | false | 快捷명령어打开标志；true: 打开；false: 关闭 |

### 요청 예시

```json
{
  "runModeAutoManBegFlag": false,
  "remoteUseReserveModeFlag": false,
  "remoteAllowContinueRunFlag": true,
  "remoteAllowCurrentRunFlag": false,
  "remoteSwitchModeEnable": false,
  "allowReserveRunningPro": false,
  "SetSafeScreenTeachInvalidFlag": false,
  "keepStepRunFlag": 0,
  "defaultRunningSpeed": 1,
  "language": "chinese",
  "accUnitMode": 0,
  "craftSelect": "general",
  "base_coord_of_user_coord_": 0,
  "remote_io_reset_clear_all_reserve_program_status_flag": true,
  "autoSetUserAndTool": false,
  "checkServoStatusWordMaxTime": 0,
  "OutageKeepFunctionFlag": true,
  "quickInsert": false
}
```
