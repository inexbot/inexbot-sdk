# 조작매개변수

## 1. 실행매개변수설정

**명령字：** `0x1300`

### 매개변수열表

| 매개변수名 | 유형 | 기본값 | 설명 |
|--------|------|--------|------|
| `runModeAutoManBegFlag` | bool | false | 切到실행모드自动전원 켜기；true: 自动전원 켜기；false：不自动전원 켜기 |
| `remoteUseReserveModeFlag` | bool | false | 여부为예약모드；true: 예약모드；false：非예약모드 |
| `remoteAllowContinueRunFlag` | bool | true | 원격IO断点실행；true: 断电실행；false：断电不실행 |
| `remoteAllowCurrentRunFlag` | bool | false | 원격IO현재줄실행；true: 현재줄실행；false：非현재줄실행 |
| `remoteSwitchModeEnable` | bool | false | 없음티치 펜던트切원격모드；true: 允许；false：不允许 |
| `allowReserveRunningPro` | bool | false | 원격IO프로그램실행中再회예약；true: 允许；false：不允许 |
| `SetSafeScreenTeachInvalidFlag` | bool | false | 티치모드安全光幕不生效；true: 不生效；false：生效 |
| `keepStepRunFlag` | int | 0 | 单步/원점 복귀/홈 위치/실행조작방식；0：单击실행；1：按下실행 |
| `defaultRunningSpeed` | int | 1 | 실행모드开机默认속도 |
| `language` | string | "chinese" | 설정默认언어；"chinese"：中文；"english"：英文；"korean"：韩文 |
| `accUnitMode` | int | 0 | 관절/笛卡尔매개변수中최대加속도단위；0: 倍값；1: 값 |
| `craftSelect` | string | "general" | 프로세스선택；"general"：범용프로세스；"special"：专用프로세스；"punch"：스탬핑프로세스；"pallet"：팔레타이징 프로세스；"weld"：용접 프로세스；"cut"：절단프로세스 |
| `base_coord_of_user_coord_` | int | 0 | 사용자 좌표系基准좌표；0: 직교 좌표系；1: 大地좌표系 |
| `remote_io_reset_clear_all_reserve_program_status_flag` | bool | true | 원격IO리셋비우기예약队열；true: 비우기；false：不비우기 |
| `autoSetUserAndTool` | bool | false | 포인트형태/도구手/사용자值설정방식；true: 自动；false: 手动 |
| `checkServoStatusWordMaxTime` | int | 0 | 서보응답시간；값 범위범위 [0, 100]，단위 ms |
| `OutageKeepFunctionFlag` | bool | true | 断电保持标志비트；true: 保持；false: 不保持 |
| `quickInsert` | bool | false | 바로가기명령어열기标志；true: 열기；false: 닫기 |

### 요청 예시

```json
{
  "runModeAutoManBegFlag": false,
  "remoteUseReserveModeFlag": false,
  "remoteAllowContinueRunFlag": true,
  "remoteAllowCurrentRunFlag": false,
  "remoteSwitchModeEnable": false,
  "allowReserveRunningPro": false,
  "SetSafeScreenTeachInvalidFlag": false,
  "keepStepRunFlag": 0,
  "defaultRunningSpeed": 1,
  "language": "chinese",
  "accUnitMode": 0,
  "craftSelect": "general",
  "base_coord_of_user_coord_": 0,
  "remote_io_reset_clear_all_reserve_program_status_flag": true,
  "autoSetUserAndTool": false,
  "checkServoStatusWordMaxTime": 0,
  "OutageKeepFunctionFlag": true,
  "quickInsert": false
}
```
