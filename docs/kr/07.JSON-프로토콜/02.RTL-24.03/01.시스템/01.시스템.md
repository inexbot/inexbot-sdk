# 시스템

## 상태

### 조회初始化상태

시스템开机需要进行初始化，否则시스템将无法使用。

**명령어：** `0x1000`

- 无需전송데이터

**컨트롤러回复**

**명령어：** `0x1001`

| 필드 | 유형 | 설명 |
|------|------|------|
| finishinit | bool | 初始化是否完成，true 完成，false 未完成 |

```json
{
  "finishinit": true
}
```

### 조회是否为 T5 시스템

**명령어：** `0x1002`

- 无需전송데이터

**컨트롤러回复**

**명령어：** `0x1003`

| 필드 | 유형 | 설명 |
|------|------|------|
| T5_system | bool | 是否为 T5 시스템，true 是，false 不是 |

```json
{
  "T5_system": false
}
```

### 조회硬件시스템发布版本

**명령어：** `0x1004`

- 无需전송데이터

**컨트롤러回复**

**명령어：** `0x1005`

| 필드 | 유형 | 설명 |
|------|------|------|
| result | bool | 조회是否成功，true 成功，false 失败 |
| version | string | 版本号（若조회成功则반환） |

```json
{
  "result": true,
  "version": "xx"
}
```

## 版本升级

시스템开机时需要조회当前版本号，若出现不匹配则需要进行版本升级。

### 조회当前시스템版本号

**명령어：** `0x1010`

| 필드 | 유형 | 설명 |
|------|------|------|
| version | string | 版本号内容 |

```json
{
  "version": "v1.0-rc1-67-gf34dae7"
}
```

**컨트롤러回复**

**명령어：** `0x1011`

| 필드 | 유형 | 설명 |
|------|------|------|
| configFileVersionMismatch | bool | 구성파일是否匹配，false 不匹配，true 匹配 |
| jobFileVersion | string | 작업파일版本 |
| rtlVersion | string | 컨트롤러版本 |
| sysClock | string | 시스템时钟，格式如 "2023.11.10 13:29:25" |
| version | string | 版本号内容，若조회时전송的 version 为空字符串则반환同样内容 |

```json
{
  "configFileVersionMismatch": false,
  "jobFileVersion": "5.2.0",
  "rtlVersion": "22.08.0",
  "sysClock": "2023.11.10 13:29:25",
  "version": "v1.0-rc1-67-gf34dae7"
}
```

## 日志

### 조회崩溃日志파일

**명령어：** `0x1020`

- 无需전송데이터

**컨트롤러回复**

**명령어：** `0x1021`

| 필드 | 유형 | 설명 |
|------|------|------|
| absolutepath | string | 路径 |
| dumpLogfilenum | int | 파일数量 |
| dumpLogfilelist | string[] | 日志파일列表 |

```json
{
  "absolutepath": "",
  "dumpLogfilenum": 3,
  "dumpLogfilelist": ["", "", "", ""]
}
```

### 요청获取日志파일列表

**명령어：** `0x1022`

| 필드 | 유형 | 설명 |
|------|------|------|
| num | int | 表示获取最近多少个파일，可取 5、30、100 |

```json
{
  "num": 5
}
```

**컨트롤러回复**

**명령어：** `0x1023`

| 필드 | 유형 | 설명 |
|------|------|------|
| absolutepath | string | 日志所在目录 |
| logfilenum | int | 日志数目（与 num 不一定相等） |
| logfilelist | string[] | 日志列表 |

```json
{
  "absolutepath": "log/",
  "logfilenum": 3,
  "logfilelist": ["logInfo.0", "logInfo.1", "logInfo.2"]
}
```

| clearExtendedFile | bool | 清除所有扩展파일，true 清除，false 未清除 |

```json
{
  "clearFactoryIO": true,
  "clearFactoryRobot": true,
  "clearFactoryToolhand": true,
  "clearConfigFile": true,
  "clearExtendedFile": true
}
```

## IP 설정

### 컨트롤러 IP 설정

**명령어：** `0x1040`

| 필드 | 유형 | 설명 |
|------|------|------|
| name | string | 网口名称 |
| address | string | 설정컨트롤러 IP |
| gateway | string | 설정컨트롤러网关 |
| dns | string | 설정컨트롤러 DNS |

```json
{
  "name": "p1p1",
  "address": "192.168.1.13",
  "gateway": "192.168.1.1",
  "dns": "114.114.114.114"
}
```

### 조회연결 IP

**명령어：** `0x1041`

- 无需전송데이터

**컨트롤러回复**

**명령어：** `0x1042`

| 필드 | 유형 | 설명 |
|------|------|------|
| num | int | 网口数目 |
| network | array | 网络信息列表 |
| network[].name | string | 网口名称 |
| network[].address | string | 컨트롤러 IP |
| network[].gateway | string | 컨트롤러网关 |
| network[].dns | string | 컨트롤러 DNS |

```json
{
  "num": 2,
  "network": [
    {
      "name": "p1p1",
      "address": "192.168.1.13",
      "gateway": "192.168.1.1",
      "dns": "114.114.114.114"
    },
    {
      "name": "p2p1",
      "address": "192.168.1.14",
      "gateway": "192.168.1.1",
      "dns": "114.114.114.114"
    }
  ]
}
```

## 时间설정

### 전송时间조회

**명령어：** `0x1050`

| 필드 | 유형 | 설명 |
|------|------|------|
| type | int | 时间유형：0 现实时间，1 获取시스템开始时间至现在秒数 |
| format | int | 格式：0 获取时间戳，1 获取北京时间 |

```json
{
  "type": 0,
  "format": 0
}
```

**컨트롤러回复**

**명령어：** `0x1051`

| 필드 | 유형 | 설명 |
|------|------|------|
| tType | int | 时间유형：0 现实时间，1 시스템开始时间至现在秒数 |
| format | int | 格式：0 时间戳，1 北京时间 |
| date | object | 日期和时间信息 |
| date.year | int | 年份 |
| date.mon | int | 月份 |
| date.day | int | 日期 |
| date.hour | int | 小时 |
| date.min | int | 分钟 |
| date.sec | int | 秒数 |
| date.msec | int | 毫秒数 |
| timestamp | object | 时间戳（format 为 0 时显示） |
| timestamp.s | int | 时间戳秒数 |
| timestamp.ns | int | 时间戳纳秒数 |

```json
{
  "tType": 0,
  "format": 0,
  "date": {
    "year": 2023,
    "mon": 1,
    "day": 1,
    "hour": 1,
    "min": 1,
    "sec": 1,
    "msec": 1
  },
  "timestamp": {
    "s": 1,
    "ns": 1
  }
}
```

## 시스템保养

- 无需전송데이터

**컨트롤러回复**

**명령어：** `0x1061`

| 필드 | 유형 | 설명 |
|------|------|------|
| controller_ID | string | 컨트롤러 ID |
| controller_total_time | int | 컨트롤러通电总时间，单位为小时 |
| controller_spent_time | int | 컨트롤러距离上次保养时长，单位为小时 |
| controller_cycle_time | int | 컨트롤러保养间隔周期，单位为小时 |
| robot_total_time | int | 로봇通电总时间，单位为小时 |
| robot_spent_time | int | 로봇距离上次保养时长，单位为小时 |
| robot_cycle_time | int | 로봇保养间隔周期，单位为小时 |

```json
{
  "controller_ID": "4DC9F0249098C82E",
  "controller_total_time": 0,
  "controller_spent_time": 0,
  "controller_cycle_time": 0,
  "robot_total_time": 0,
  "robot_spent_time": 0,
  "robot_cycle_time": 0
```


## 시스템备份

### 조회是否存在当前版本티치 펜던트프로그램

**명령어：** `0x1090`

| 필드 | 유형 | 설명 |
|------|------|------|
| config_md5 | string | config.db 파일 MD5 值 |
| pinyin_md5 | string | Pinyin.db 파일 MD5 值，null 表示没有这个파일或者파일大小是 0 |
| pro_md5 | string | 티치 펜던트프로그램파일的 MD5 值 |
| pro_name | string | 티치 펜던트프로그램파일的名称，可选 Qt-tp 或 T30_ARM_Linux_Teach-pendant |

```json
{
  "config_md5": "2803f92ecdecfd3c178b0998b9391a4d",
  "pinyin_md5": "null",
  "pro_md5": "ce407851346d030dad5207948f55ed6c",
  "pro_name": "Qt-tp"
}
```

**컨트롤러回复**

**명령어：** `0x1091`

| 필드 | 유형 | 설명 |
|------|------|------|
| path | object | 파일路径信息 |
| path.config | string | config.db 파일上传路径 |
| path.dir | string | 所有파일的路径 |
| path.pinyin | string | Pinyin.db 파일上传路径，空字符串表示不需要上传 |
| path.pro | string | Qt-tp 파일上传路径 |

```json
{
  "path": {
    "config": "qttpBackup/rtl-24.03.07-6.0.9-1.0.6",
    "dir": "qttpBackup/rtl-24.03.07-6.0.9-1.0.6",
    "pinyin": "",
    "pro": "qttpBackup/rtl-24.03.07-6.0.9-1.0.6"
```

### 조회컨트롤러기능限制

**명령어：** `0x10B1`

- 无需전송데이터

**컨트롤러回复**

**명령어：** `0x10B2`

| 필드 | 유형 | 설명 |
|------|------|------|
| maxRobotCount | int | 로봇最大数目 |
| crafts | string[] | 已解锁的프로세스列表 |
| robotTypes | string[] | 已解锁的로봇유형 |

프로세스유형值：
- `C_Pallet`：팔레타이징프로세스
- `C_Welding`：용접프로세스
- `C_Lasercut`：레이저 절단프로세스
- `C_Polish`：그라인딩프로세스
- `C_Spray`：스프레이프로세스
- `C_Search`：심 추적프로세스
- `C_Vision`：비전프로세스

로봇유형值：
- `R_NULL`：未选择로봇유형
- `R_GENERAL_6S`：六轴串联多관절
- `R_GENERAL_6S_1`：六轴协作
- `R_SIXAXIS_SPRAY_BBR`：六轴스프레이로봇
- `R_GENERAL_6S_2`：六轴异型二
- `R_GENERAL_5S`：五轴로봇
- `R_SCARA`：四轴 SCARA 로봇
- `R_SCARA_1`：四轴 SCARA 异型一로봇
- `R_FOURAXIS_PALLET`：四轴连杆팔레타이징로봇
- `R_FOURAXIS_PALLET_1`：四轴팔레타이징丝杆로봇

### 조회컨트롤러 ID

**명령어：** `0x10B3`

- 无需전송데이터

**컨트롤러回复**

**명령어：** `0x10B4`

| 필드 | 유형 | 설명 |
|------|------|------|
| controllerID | string | 컨트롤러 ID |

```json
{
  "controllerID": "4DC9F0249098C82E"
}
```

> 설명：获取到연결的컨트롤러 ID 之后，在同步작업파일时需判断此 ID 是否已知。已知则判断路径 "dir/controllerID" 是否存在，若存在则不做조작；若不存在则创建工作파일路径，格式："dir/controllerID/robotJob/R1"。多个로봇则表示 R1-4。

### 获取컨트롤러剩余使用天数

**명령어：** `0x10B5`

- 无需전송데이터

**컨트롤러回复**

**명령어：** `0x10B6`

| 필드 | 유형 | 설명 |
|------|------|------|
| restdays | int | 剩余使用天数，-1 无限期使用，0 使用期限已到，其余表示剩余多少天 |
| year | int | 过期年份 |
| mon | int | 过期月份 |
| day | int | 过期日 |

```json
{
  "restdays": 30,
  "year": 1,
  "mon": 1,
  "day": 1
}
```

### 컨트롤러序列号更新

**명령어：** `0x10B8`

| 필드 | 유형 | 설명 |
|------|------|------|
| license | string | 컨트롤러序列号 |

```json
{
  "license": "ACOCWKBZHSKBEJHI"
}
```

**컨트롤러回复**

**명령어：** `0x10B9`

| 필드 | 유형 | 설명 |
|------|------|------|
| result | string | 更新结果，"success" 成功，"ivalid" 非法密钥，"notInit" 未初始化 |
| ivalid | string | 无效的密钥 |
| error | string | 存在破解嫌疑，拒绝解锁 |

```json
{
  "result": "success",
  "ivalid": "无效的密钥!",
  "error": "存在破解嫌疑，拒绝解锁!"
}
```

> 密钥更新成功后，컨트롤러即将重启。
