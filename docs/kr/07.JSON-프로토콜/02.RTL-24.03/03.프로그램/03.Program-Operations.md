# 프로그램操作

## 프로그램열기

正式실행프로그램之前필요열기프로그램。

**명령어:** `0x5100`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要操作的로봇 번호，값 범위범위 [1, 4] |
| jobName | string | 新建작업 파일이름 |
| suffixname | string | 파일확장자：.JBR 前台프로그램，.JBP 백그라운드로컬프로그램，.JBPG 백그라운드전역프로그램 |

```json
{
  "robot": 1,
  "jobName": "xxxx",
  "suffixname": ".JBR"
}
```

## 프로그램복사

복사后프로그램的后缀예1样的。

**명령어:** `0x5101`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要操作的로봇 번호，값 범위범위 [1, 4] |
| oldjobName | string | 旧파일명 |
| newjobName | string | 복사后파일명 |
| suffixname | string | 파일확장자：.JBR 前台프로그램，.JBP 백그라운드로컬프로그램，.JBPG 백그라운드전역프로그램 |

```json
{
  "robot": 1,
  "oldjobName": "QQQQ",
  "newjobName": "WWWW",
  "suffixname": ".JBR"
}
```

## 프로그램해제

会해제현재로봇的所有프로그램，但아님会해제리셋프로그램。

**명령어:** `0x5102`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要조회的로봇 번호，값 범위범위 [1, 4] |
| lock_enable | bool | 예아니오비우기작업 파일활성화，true 비우기활성화，false 비우기아님활성화 |

```json
{
  "robot": 1,
  "lock_enable": true
}
```

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要操作的로봇 번호，값 범위범위 [1, 4] |

  "robot": 1
}
```

**컨트롤러 응답**

**명령어:** `0x5104`

| 필드 | 유형 | 설명 |
|------|------|------|

```

## 主프로그램

主프로그램可以认을/를예特殊的前台프로그램，只有前台프로그램可以설정成主프로그램。

### 설정작업 파일을/를主프로그램

**명령어:** `0x5110`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要조회的로봇 번호，값 범위범위 [1, 4] |
| jobName | string | 작업 파일이름，空문자열의미取消主프로그램 |

```json
{
  "robot": 1,
  "jobName": "xxx"
}
```

### 조회主프로그램

**명령어:** `0x5111`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要조회的로봇 번호，값 범위범위 [1, 4] |

```json
{
  "robot": 1
}
```

**컨트롤러 응답**

**명령어:** `0x5112`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要조회的로봇 번호，값 범위범위 [1, 4] |
| jobName | string | 작업 파일이름，空의미没有 |

```json
{
  "robot": 1,
  "jobName": "xxx"
}
```

## 프로그램저장

프로그램进줄수정、삽입、삭제后필요使用该协议进줄저장。

**명령어:** `0x5120`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要조회的로봇 번호，값 범위범위 [1, 4] |
| jobname | string | 작업 파일이름 |
| suffix | string | 파일확장자：.JBR 前台프로그램，.JBP 백그라운드로컬프로그램，.JBPG 백그라운드전역프로그램 |
| time | string | 타임스탬프，형식如 "2024.08.08 14:16:39" |

```json
{
  "robot": 1,
  "jobname": "xxxx",
  "suffix": ".JBR",
  "time": "2024.08.08 14:16:39"
}
```

## 동기화프로그램열기

> 功能可能有문제，只有在 6001 열기时동기화到 6000 的操作

컨트롤러열기프로그램后通知호스트 컴퓨터，告知호스트 컴퓨터该프로그램已在컨트롤러열기。

**명령어:** `0x3130`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 현재反馈的로봇 번호，값 범위범위 [1, 4] |
| jobName | string | 작업 파일이름 |

```json
{
  "robot": 1,
  "jobName": ""
}
```

## 전송프로그램명령어

을/를없음작업 파일的명령어삽입。실행방식与작업실행방식1样，但예필요将 "noFileRun" 노드的값写을/를 true。

**명령어:** `0x5122`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要操作的로봇 번호，값 범위범위 [1, 4] |
| cmd | object | 삽입的명령어详情，형식을/를 cmd->toJson()，见명령어页面 |

```json
{
  "robot": 1,
  "cmd": {}
}
```

## 프로그램동기화

호스트 컴퓨터 전송给컨트롤러，通知티치 펜던트동기화작업 파일。

**명령어:** `0x1222`

- 없음需전송데이터

**컨트롤러응답**

**명령어:** `0x1223`

| 필드 | 유형 | 설명 |
|------|------|------|
| finished | bool | true 성공，false 실패 |

```json
{
  "finished": true
}
```

### 컨트롤러通知티치 펜던트동기화작업 파일

**명령어:** `0x1224`

- 없음需전송데이터

**티치 펜던트응답**

**명령어:** `0x1225`

| 필드 | 유형 | 설명 |
|------|------|------|
| finished | bool | true 성공，false 실패 |

```json
{
  "finished": true
}
```
