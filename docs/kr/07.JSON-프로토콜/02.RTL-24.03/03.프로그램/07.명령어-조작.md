# 명령어조작

## 1 외부 축

### 1.1 외부 축위치조회

**설명：** 로봇+变位机同步插入명령어时的위치조회

**명령어：** `0x5300`

**호스트 컴퓨터 전송：**

```json
{
  "robot": 1,
  "coord": 0
}
```

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| robot | int | 是 | 选择로봇，取值범위 [1，4] |
| coord | int | 是 | 컨트롤러当前좌표：0-관절좌표 1-直角좌표 2-도구좌표 3-사용자좌표 |

---

### 1.2 외부 축위치수신

**설명：** 로봇+变位机同步插入명령어时的위치수신

**명령어：** `0x5301`

**컨트롤러 응답：**

```json
{
  "robot": 1,
  "coord": 0,
  "configuration": 1,
  "position": [0, 0.1, 2, 3.3, 44, 555.55, 66.6],
  "positionDeg": [0, 0.1, 2, 3.3, 44, 555.55, 66.6],
  "externalPosition": [0, 0.1, 2, 3.3, 44]
}
```

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| robot | int | 是 | 选择로봇，取值범위 [1，4] |
| coord | int | 是 | 컨트롤러当前좌표：0-관절좌표 1-直角좌표 2-도구좌표 3-사용자좌표 |
| configuration | int | 是 | 姿态或左右手 |
| position | double[] | 是 | 当前위치（弧度制），7位考虑双机情况 |
| positionDeg | double[] | 是 | 当前위치（角度制） |
| externalPosition | double[] | 是 | 외부 축当前위치，5位 |

---

## 4 语法检查

### 4.1 获取语法检查结果

**명령어：** `0x9121`

**호스트 컴퓨터 전송：**

```json
{
  "robot_id": 1,
  "job_name": "QQQ",
  "suffix": ".JBR"
}
```

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| robot_id | int | 是 | 로봇号 |
| job_name | string | 是 | 작업名 |
| suffix | string | 是 | 프로그램后缀：`.JBR` 主프로그램、`.JBP` 后台局部프로그램、`.JBPG` 后台全局프로그램 |

---

### 4.2 语法检查结果반환

**명령어：** `0x9121`

**컨트롤러 응답：**

```json
{
  "robot_id": 1,
  "job_name": "QQQ",
  "suffix": ".JBR",
  "result": "true",
  "cur_error_command_num": 5
}
```

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| robot_id | int | 是 | 로봇号 |
| job_name | string | 是 | 작업名 |
| suffix | string | 是 | 프로그램后缀 |
| result | bool | 是 | 检查结果：true-成功，false-失败 |
| cur_error_command_num | int | 否 | 失败时行号 |

## 5 快捷명령어

```json
    [0, 1],
    [0, 2],
    [0, 3],
    [0, 5],
    [-1, -1],
    [-1, -1],
    [-1, -1]
  ]
}
```

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| quickInsert | array | 是 | 快捷명령어数组，共8个元素，每个元素第一位代表명령어유형，第二位代表명령어 |
