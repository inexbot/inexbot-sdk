# 프로그램增删改查

## 1 프로그램삭제

**명령字：** `0x5000`

**설명：** 호스트 컴퓨터下发삭제프로그램

| 매개변수 | 유형 | 필수 | 설명 |
|------|------|------|------|
| robot | int | 예 | 로봇번호，값 범위범위[1,4] |
| jobfilename | string | 예 | 작업파일이름 |
| suffixname | string | 예 | 작업파일확장자 |

**확장자유형：**
- `.JBR` - 主프로그램
- `.JBP` - 后台로컬프로그램
- `.JBPG` - 后台전역프로그램

```json
{
  "robot": 1,
  "jobfilename": "QQQQ",
  "suffixname": ".JBR"
}
```

## 2 프로그램이름 변경

**명령字：** `0x5001`

**설명：** 将原有프로그램的名字更改，호스트 컴퓨터下发이름 변경프로그램

| 매개변수 | 유형 | 필수 | 설명 |
|------|------|------|------|
| robot | int | 예 | 로봇번호，값 범위범위[1,4] |
| lastname | string | 예 | 수정前프로그램名 |
| newname | string | 예 | 수정后프로그램名 |
| suffixname | string | 예 | 프로그램확장자 |

**확장자유형：**
- `.JBR` - 主프로그램
- `.JBP` - 后台로컬프로그램
- `.JBPG` - 后台전역프로그램

```json
{
  "robot": 1,
  "lastname": "QQQQ",
  "newname": "WWWW",
  "suffixname": ".JBR"
}
```

## 3 프로그램创建

**명령字：** `0x5002`

**설명：** 
- 프로그램的新建필요在티치모드下
- 프로그램的실행필요再실행모드下，并且서보就绪，전원 켜기没有报错
- 호스트 컴퓨터下发新建프로그램

| 매개변수 | 유형 | 필수 | 설명  |  |
|------|------|------|------|------|
| robot | int | 예 | 로봇번호，값 범위범위[1,4]  |  |
| jobName | string | 예 | 新建프로그램名，命名首字母不得为숫자  |  |
| suffix | string | 예 | 프로그램확장자  |  |
| time | string | 예 | 삽입명령어的时间，형식如 `2025.03.11 20:28:23`  |  |

**확장자유형：**
- `.JBR` - 主프로그램
- `.JBP` - 后台로컬프로그램
- `.JBPG` - 后台전역프로그램

```json
{
  "robot": 1,
  "jobName": "TTT",
  "suffixname": ".JBR",
  "time": "2025.03.11 20:28:23"
}
```

## 4 프로그램파일열表조회

**설명：** 호스트 컴퓨터下发조회프로그램파일열表，若성공将会반환 `0x5011`、`0x5012`、`0x5013` 三번째명령字

- 사용时，在수신到 `0x5011` 后，开启计时，设定5s，作用为判断다운로드파일过程中여부대기타임아웃
- 반환 `0x5012` 时，每번째로봇会单独전송，并且每회전송파일수량为五번째
- 파일전송완료后，반환 `0x5013` 作为전송완료的标志

### 4.1 조회프로그램파일열表（0x5010）

**명령字：** `0x5010`

**설명：** 호스트 컴퓨터下发조회프로그램파일열表

| 매개변수 | 유형 | 필수 | 설명 |
|------|------|------|------|
| - | - | - | 없음노드 |

### 4.2 반환프로그램파일총 수（0x5011）

**명령字：** `0x5011`

**설명：** 컨트롤러 응답프로그램파일총 수，가져오기所有로봇的프로그램파일열表

| 매개변수 | 유형 | 필수 | 설명 |
|------|------|------|------|
| absolutepath | string배열 | 예 | 프로그램파일的파일夹열表，如 `{"/job/R1/","/job/R2/","/job/R3/","/job/R4/"}`，当只有一번째로봇时，其余위치填空문자열 |
| jobfilenum | int배열 | 예 | 프로그램파일夹中포함的파일번째数，分别对应로봇1,2,3,4，수량포함리셋프로그램；当只有一번째로봇时，其余위치为0 |

**备注：** 리셋프로그램 - 当点击리셋时실행리셋프로그램或者이동到홈 위치，具体조작看사용자설정

```json
{
  "absolutepath": ["/job/R1/", "/job/R2/", "/job/R3/", "/job/R4/"],
  "jobfilenum": [4, 1, 1, 1]
}
```

### 4.3 반환프로그램파일열表（0x5012）

**명령字：** `0x5012`

**설명：** 컨트롤러主动向호스트 컴퓨터 전송명령어，向호스트 컴퓨터응답时每번째로봇分개발送

| 매개변수 | 유형 | 필수 | 설명 |
|------|------|------|------|
| robot | int | 예 | 로봇번호，값 범위범위[1,4] |
| listnum | int | 예 | 전송的프로그램파일수량，单회最多전송5번째，범위[0,5] |
| jobfilelist | object배열 | 예 | 프로그램파일배열，요소번째数为 `listnum` 的值 |

**jobfilelist 요소구조：**

| 매개변수 | 유형 | 설명 |
|------|------|------|
| name | string | 프로그램파일名，如 `xxx.JBR` |
| MD5 | string | MD5值，用于判断파일여부相同 |

**备注：** `RobotResetProgram.ResetPro` 为리셋프로그램

```json
{
  "robot": 1,
  "listnum": 2,
  "jobfilelist": [
    {
      "name": "RobotResetProgram.ResetPro",
      "MD5": "e4b21ca518cd2d2eecd4b3f463c02ab1"
    },
    {
      "name": "yyy.JBR",
      "MD5": "5b172a7a37a07aa6ccec1c60cfd5db7d"
    }
  ]
}
```

### 4.4 프로그램파일열表조회종료（0x5013）

**명령字：** `0x5013`

**설명：** 标志着프로그램파일열表조회종료，컨트롤러 응답프로그램파일열表조회종료的标识

| 매개변수 | 유형 | 필수 | 설명 |
|------|------|------|------|
| - | - | - | 없음노드 |

### 5.2 반환Lua파일열表（0x5021）

**명령字：** `0x5021`

**설명：** 컨트롤러 응답lua파일열表

| 매개변수 | 유형 | 필수 | 설명 |
|------|------|------|------|
| absolutepath | string | 예 | lua脚本所在디렉터리，如 `lua/` |
| sum | int | 예 | 열表총 수 |
| scriptlist | string배열 | 예 | lua파일열表 |

```json
{
  "absolutepath": "lua/",
  "sum": 2,
  "scriptlist": ["ww.lua", "ee.lua"]
}
```
