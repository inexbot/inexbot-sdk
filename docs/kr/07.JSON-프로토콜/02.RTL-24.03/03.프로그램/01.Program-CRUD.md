# 프로그램增删改查

## 1 프로그램删除

**명령字：** `0x5000`

**설명：** 호스트 컴퓨터下发删除프로그램

| 매개변수 | 유형 | 必填 | 설명 |
|------|------|------|------|
| robot | int | 是 | 로봇编号，取值범위[1,4] |
| jobfilename | string | 是 | 작업파일名称 |
| suffixname | string | 是 | 작업파일后缀名 |

**后缀名유형：**
- `.JBR` - 主프로그램
- `.JBP` - 后台局部프로그램
- `.JBPG` - 后台全局프로그램

```json
{
  "robot": 1,
  "jobfilename": "QQQQ",
  "suffixname": ".JBR"
}
```

## 2 프로그램重命名

**명령字：** `0x5001`

**설명：** 将原有프로그램的名字更改，호스트 컴퓨터下发重命名프로그램

| 매개변수 | 유형 | 必填 | 설명 |
|------|------|------|------|
| robot | int | 是 | 로봇编号，取值범위[1,4] |
| lastname | string | 是 | 修改前프로그램名 |
| newname | string | 是 | 修改后프로그램名 |
| suffixname | string | 是 | 프로그램后缀名 |

**后缀名유형：**
- `.JBR` - 主프로그램
- `.JBP` - 后台局部프로그램
- `.JBPG` - 后台全局프로그램

```json
{
  "robot": 1,
  "lastname": "QQQQ",
  "newname": "WWWW",
  "suffixname": ".JBR"
}
```

## 3 프로그램创建

**명령字：** `0x5002`

**설명：** 
- 프로그램的新建필요在示教모드下
- 프로그램的실행필요再실행모드下，并且伺服就绪，上电没有报错
- 호스트 컴퓨터下发新建프로그램

| 매개변수 | 유형 | 必填 | 설명  |  |
|------|------|------|------|------|
| robot | int | 是 | 로봇编号，取值범위[1,4]  |  |
| jobName | string | 是 | 新建프로그램名，命名首字母不得为数字  |  |
| suffix | string | 是 | 프로그램后缀名  |  |
| time | string | 是 | 插入명령어的时间，格式如 `2025.03.11 20:28:23`  |  |

**后缀名유형：**
- `.JBR` - 主프로그램
- `.JBP` - 后台局部프로그램
- `.JBPG` - 后台全局프로그램

```json
{
  "robot": 1,
  "jobName": "TTT",
  "suffixname": ".JBR",
  "time": "2025.03.11 20:28:23"
}
```

**설명：** 컨트롤러전송给티치 펜던트外部按键触发的新建작업

|------|------|------|------|
| jobName | string | 是 | 필요新建的작업名称 |

```json
{
  "jobName": "파일1"
}
```

## 4 프로그램파일列表조회

**설명：** 호스트 컴퓨터下发조회프로그램파일列表，若成功将会반환 `0x5011`、`0x5012`、`0x5013` 三个명령字

- 사용时，在수신到 `0x5011` 后，开启计时，设定5秒，作用为判断다운로드파일过程中是否等待超时
- 반환 `0x5012` 时，每个로봇会单独전송，并且每次전송파일数量为五个
- 파일전송完成后，반환 `0x5013` 作为전송完成的标志

### 4.1 조회프로그램파일列表（0x5010）

**명령字：** `0x5010`

**설명：** 호스트 컴퓨터下发조회프로그램파일列表

| 매개변수 | 유형 | 必填 | 설명 |
|------|------|------|------|
| - | - | - | 无节点 |

### 4.2 반환프로그램파일总数（0x5011）

**명령字：** `0x5011`

**설명：** 컨트롤러 응답프로그램파일总数，获取所有로봇的프로그램파일列表

| 매개변수 | 유형 | 必填 | 설명 |
|------|------|------|------|
| absolutepath | string数组 | 是 | 프로그램파일的파일夹列表，如 `{"/job/R1/","/job/R2/","/job/R3/","/job/R4/"}`，当只有一个로봇时，其余위치填空字符串 |
| jobfilenum | int数组 | 是 | 프로그램파일夹中포함的파일个数，分别对应로봇1,2,3,4，数量포함复位프로그램；当只有一个로봇时，其余위치为0 |

**备注：** 复位프로그램 - 当点击复位时执行复位프로그램或者移动到复位点，具体조작看사용자설정

```json
{
  "absolutepath": ["/job/R1/", "/job/R2/", "/job/R3/", "/job/R4/"],
  "jobfilenum": [4, 1, 1, 1]
}
```

### 4.3 반환프로그램파일列表（0x5012）

**명령字：** `0x5012`

**설명：** 컨트롤러主动向호스트 컴퓨터 전송명령어，向호스트 컴퓨터回复时每个로봇分개발送

| 매개변수 | 유형 | 必填 | 설명 |
|------|------|------|------|
| robot | int | 是 | 로봇编号，取值범위[1,4] |
| listnum | int | 是 | 전송的프로그램파일数量，单次最多전송5个，범위[0,5] |
| jobfilelist | object数组 | 是 | 프로그램파일数组，元素个数为 `listnum` 的值 |

**jobfilelist 元素结构：**

| 매개변수 | 유형 | 설명 |
|------|------|------|
| name | string | 프로그램파일名，如 `xxx.JBR` |
| MD5 | string | MD5值，用于判断파일是否相同 |

**备注：** `RobotResetProgram.ResetPro` 为复位프로그램

```json
{
  "robot": 1,
  "listnum": 2,
  "jobfilelist": [
    {
      "name": "RobotResetProgram.ResetPro",
      "MD5": "e4b21ca518cd2d2eecd4b3f463c02ab1"
    },
    {
      "name": "yyy.JBR",
      "MD5": "5b172a7a37a07aa6ccec1c60cfd5db7d"
    }
  ]
}
```

### 4.4 프로그램파일列表조회结束（0x5013）

**명령字：** `0x5013`

**설명：** 标志着프로그램파일列表조회结束，컨트롤러 응답프로그램파일列表조회结束的标识

| 매개변수 | 유형 | 必填 | 설명 |
|------|------|------|------|
| - | - | - | 无节点 |

## 5 Lua파일列表조회

### 5.1 조회Lua파일列表（0x5020）

**명령字：** `0x5020`

**설명：** 호스트 컴퓨터 전송조회lua파일列表

| 매개변수 | 유형 | 必填 | 설명 |
|------|------|------|------|
| - | - | - | 无节点 |

### 5.2 반환Lua파일列表（0x5021）

**명령字：** `0x5021`

**설명：** 컨트롤러 응답lua파일列表

| 매개변수 | 유형 | 必填 | 설명 |
|------|------|------|------|
| absolutepath | string | 是 | lua脚本所在目录，如 `lua/` |
| sum | int | 是 | 列表总数 |
| scriptlist | string数组 | 是 | lua파일列表 |

```json
{
  "absolutepath": "lua/",
  "sum": 2,
  "scriptlist": ["ww.lua", "ee.lua"]
}
```
