# 프로그램状态

## 프로그램行号

当控制器开始运行一条指令时，向호스트 컴퓨터 전송：

**명령어:** `0x5400`

| 字段 | 类型 | 说明 |
|------|------|------|
| robot | int | 选择机器人，取值范围 [1，4] |
| jobname | string | 作业文件名称 |
| num | int | 已运行指令数量 |

```json
{
  "robot": 1,
  "jobname": "Q1",
  "num": 3
}
```

## CRAFTLINE 指令跳转

当控制器执行 CRAFTLINE 指令时，向示教器发送跳转：

**명령어:** `0x5401`

| 字段 | 类型 | 说明 |
|------|------|------|
| robot | int | 选择机器人，取值范围 [1，4] |
| num | int | 已运行指令数量 |

```json
{
  "robot": 1,
  "num": 2
}
```

## 조회当前行号

当示教器与控制器建立网络连接时，호스트 컴퓨터 조회当前运行到哪一行：

**명령어:** `0x5402`

| 字段 | 类型 | 说明 |
|------|------|------|
| robot | int | 选择机器人，取值范围 [1，4] |

```json
{
  "robot": 1
}
```

**컨트롤러 응답**

**명령어:** `0x5403`

| 字段 | 类型 | 说明 |
|------|------|------|
| robot | int | 选择机器人，取值范围 [1，4] |
| jobname | string | 作业文件名称 |
| num | int | 行号 |

```json
{
  "robot": 1,
  "jobname": "Q1",
  "num": 3
}
```

## 프로그램运行状态

程序状态包括：暂停、运行、停止。

### 조회程序运行状态

**명령어:** `0x5410`

| 字段 | 类型 | 说明 |
|------|------|------|
| robot | int | 选择机器人，取值范围 [1，4] |

```json
{
  "robot": 1
}
```

**컨트롤러 응답**

**명령어:** `0x5411`

| 字段 | 类型 | 说明 |
|------|------|------|
| robot | int | 选择机器人，取值范围 [1，4] |
| status | int | 程序运行状态：0 停止，1 暂停，2 运行 |

```json
{
  "robot": 1,
  "status": 0
}
```

### 조회机器人运行状态

**명령어:** `0x5412`

| 字段 | 类型 | 说明 |
|------|------|------|
| robot | int | 选择机器人，取值范围 [1，4] |
| jobfilename | string | 作业文件名 |

```json
{
  "robot": 1,
  "jobfilename": ""
}
```

**컨트롤러 응답**

**명령어:** `0x5413`

| 字段 | 类型 | 说明 |
|------|------|------|
| robot | int | 选择机器人，取值范围 [1，4] |
| status | int | 运行状态：0 停止，1 暂停，2 运行 |
| continueRun | int | 예아니오从当前行运行：0 从头运行，1 从当前行运行 |
| currentRun | bool | 当前예아니오运行 |
| mainProgramRun | int | 主程序예아니오运行 |

```json
{
  "robot": 1,
  "status": 0,
  "continueRun": 0,
  "currentRun": false,
  "mainProgramRun": 0
}
```

### 清除运行状态

上位机在导入程序成功后发送此协议请求重置程序运行状态：

**명령어:** `0x5417`

- 无需发送数据

**컨트롤러 응답**

**명령어:** `0x5418`

- 无需回复数据

### 조회运动队列剩余指令数量

**명령어:** `0x5419`

| 字段 | 类型 | 说明 |
|------|------|------|
| robot | int | 当前机器人，取值范围 [1, 4] |

```json
{
  "robot": 1
}
```

**컨트롤러 응답**

**명령어:** `0x541A`

| 字段 | 类型 | 说明 |
|------|------|------|
| sum | int | 运动队列的行号数量 |

```json
{
  "sum": 4
}
```
