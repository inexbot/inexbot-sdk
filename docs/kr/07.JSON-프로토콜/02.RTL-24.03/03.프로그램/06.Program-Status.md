# 프로그램상태

## 프로그램줄号

当컨트롤러시작실행一条명령어时，向호스트 컴퓨터 전송：

**명령어:** `0x5400`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 선택로봇，값 범위범위 [1，4] |
| jobname | string | 작업 파일이름 |
| num | int | 已실행명령어수량 |

```json
{
  "robot": 1,
  "jobname": "Q1",
  "num": 3
}
```

## CRAFTLINE 명령어跳转

当컨트롤러실행 CRAFTLINE 명령어时，向티치 펜던트전송跳转：

**명령어:** `0x5401`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 선택로봇，값 범위범위 [1，4] |
| num | int | 已실행명령어수량 |

```json
{
  "robot": 1,
  "num": 2
}
```

## 조회현재줄号

当티치 펜던트与컨트롤러建立网络연결时，호스트 컴퓨터 조회현재실행到哪一줄：

**명령어:** `0x5402`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 선택로봇，값 범위범위 [1，4] |

```json
{
  "robot": 1
}
```

**컨트롤러 응답**

**명령어:** `0x5403`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 선택로봇，값 범위범위 [1，4] |
| jobname | string | 작업 파일이름 |
| num | int | 줄号 |

```json
{
  "robot": 1,
  "jobname": "Q1",
  "num": 3
}
```

## 프로그램실행상태

프로그램상태包括：일시정지、실행、정지。

### 조회프로그램실행상태

**명령어:** `0x5410`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 선택로봇，값 범위범위 [1，4] |

```json
{
  "robot": 1
}
```

**컨트롤러 응답**

**명령어:** `0x5411`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 선택로봇，값 범위범위 [1，4] |
| status | int | 프로그램실행상태：0 정지，1 일시정지，2 실행 |

```json
{
  "robot": 1,
  "status": 0
}
```

### 조회로봇실행상태

**명령어:** `0x5412`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 선택로봇，값 범위범위 [1，4] |
| jobfilename | string | 작업 파일名 |

```json
{
  "robot": 1,
  "jobfilename": ""
}
```

**컨트롤러 응답**

**명령어:** `0x5413`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 선택로봇，값 범위범위 [1，4] |
| status | int | 실행상태：0 정지，1 일시정지，2 실행 |
| continueRun | int | 예아니오从현재줄실행：0 从头실행，1 从현재줄실행 |
| currentRun | bool | 현재예아니오실행 |
| mainProgramRun | int | 主프로그램예아니오실행 |

```json
{
  "robot": 1,
  "status": 0,
  "continueRun": 0,
  "currentRun": false,
  "mainProgramRun": 0
}
```

### 해제실행상태

호스트 컴퓨터在가져오기프로그램성공后전송此协议请求재설정프로그램실행상태：

**명령어:** `0x5417`

- 없음需전송数据

**컨트롤러 응답**

**명령어:** `0x5418`

- 없음需응답数据
