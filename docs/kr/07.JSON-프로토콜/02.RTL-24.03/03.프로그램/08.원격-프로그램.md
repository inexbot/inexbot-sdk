# 远程프로그램

## 预约执行상태

원격 모드界面，호스트 컴퓨터조회预约执行상태时，전송。

**명령어：** `0x5500`

### 요청 매개변수

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| robot | int | 是 | 选择로봇，取值범위 [1，4] |

### 요청 예시

```json
{
    "robot": 1
}
```

컨트롤러回复预约执行상태조회结果。

**명령어：** `0x5501`

### 응답 매개변수

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| robot | int | 选择로봇，取值범위 [1，4] |
| current | object | 当前任务상태 |
| current.station | int | 工位编号 |
| current.name | string | 프로그램名称 |
| current.runtime | int | 실행时间 |
| current.times | int | 실행次数 |
| current.count | int | 已실행次数 |
| current.status | int | 상태：0-无预约，1-预约中，2-실행中，3-已预约，4-프로그램暂停 |
| queue | array | 预约队列 |
| queue[].station | int | 工位编号 |
| queue[].name | string | 프로그램名称 |
| queue[].times | int | 실행次数 |
| queue[].count | int | 已실행次数 |
| queue[].status | int | 상태 |

### 응답 예시

```json
{
    "robot": 1,
    "current": {
        "station": 1,
        "name": "Q1",
        "runtime": 1,
        "times": 3,
        "count": 1,
        "status": 2
    },
    "queue": [
        {
            "station": 1,
            "name": "Q4",
            "times": 2,
            "count": 0,
            "status": 1
        },
        {
            "station": 1,
            "name": "Q1",
            "times": 6,
            "count": 0,
            "status": 1
        },
        {
            "station": 1,
            "name": "Q3",
            "times": 1,
            "count": 0,
            "status": 1
        },
        {
            "station": 1,
            "name": "",
            "times": 1,
            "count": 0,
            "status": 0
        },
        {
            "station": 1,
            "name": "",
            "times": 2,
            "count": 0,
            "status": 0
        }
    ]
}
```

## 远程프로그램

호스트 컴퓨터 전송설정远程프로그램。

**명령어：** `0x5520`

### 요청 매개변수

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| robot | int | 是 | 选择로봇，取值범위 [1，4] |
| jobFile | array | 是 | 프로그램列表 |
| jobFile[].name | string | 是 | 프로그램名称 |
| jobFile[].times | int | 是 | 실행次数 |
| interpolationMethod | int | 否 | 插补方法 |
| absolutePosResolution | double | 否 | 绝对위치分辨率 |
| stopTime | int | 否 | 暂停时间 |
| runDelayTime | int | 否 | 실행延时时间 |
| remoteDefaultSpeed | int | 否 | 원격 모드속도 1-100% |

### 요청 예시

```json
{
    "robot": 1,
    "jobFile": [
        {"name": "Q1", "times": 3},
        {"name": "Q2", "times": 3},
        {"name": "", "times": 1},
        {"name": "Q4", "times": 2},
        {"name": "", "times": 1}
    ],
    "interpolationMethod": 4,
    "absolutePosResolution": 0.06,
    "stopTime": 5,
    "runDelayTime": 600,
    "remoteDefaultSpeed": 1
}
```

호스트 컴퓨터 전송获取远程프로그램。

**명령어：** `0x5521`

### 요청 매개변수

|--------|------|------|------|
| robot | int | 是 | 选择로봇，取值범위 [1，4] |

### 요청 예시

```json

}
```

컨트롤러收到호스트 컴퓨터조회프로그램名字时回复。

**명령어：** `0x5522`

| jobFile | array | 프로그램列表 |
| jobFile[].name | string | 프로그램名称 |
| jobFile[].times | int | 실행次数 |
| remoteProgramNum | int | 远程프로그램个数 |

### 응답 예시

| remoteDefaultSpeed | int | 否 | 원격 모드속도 1-100% |

{

    "reserveTrgTime": 200,
    "absolutePosResolution": 0.06,

## 远程IO기능界面설정

| remoteProgramNum | int | 是 | 远程IO数기능 |

| inPort.faultReset | int | 否 | 清除报警，绑定IO포트 |

| inPort.stop | int | 否 | 停止，绑定IO포트 |
| inValue | object | 否 | IO매개변수 |
| inValue.clearoutagekeep | int | 否 | 清除断电保存데이터，绑定IO포트触发所需상태：0-低电平有效，1-高电平有效 |
| inValue.faultReset | int | 否 | 清除报警，绑定IO포트触发所需상태 |
| inValue.pause | int | 否 | 暂停，绑定IO포트触发所需상태 |

## 원격 모드연결상태

조회 Modbus、外部 IO 연결상태时，전송。

**명령어：** `0x5560`

date：无

收到조회연결상태时，반환。

**명령어：** `0x5561`

### 응답 매개변수

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| ModbusConnect | bool | Modbus 연결상태：false-未연결，true-已연결 |
| ExternIOConnect | bool | 外部 IO 연결상태：false-未연결，true-已연결 |
| type | int | 유형：0-NAN，1-RTU，2-TCP |

### 응답 예시

```json
{
    "ModbusConnect": false,
    "ExternIOConnect": true,
    "type": 2
}
```

컨트롤러전송，是否通过 IO 修改了작업파일（外部按键触发修改작업파일时，컨트롤러전송给호스트 컴퓨터）。

**명령어：** `0x5562`

### 응답 매개변수

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| isModify | bool | 是否通过 IO 修改了작업파일：true-是，false-否 |

### 응답 예시

```json
{
    "isModify": false
}
```
