# 指令-FINSTCP类

## 1. 打开FINSTCP连接

**指令名**: FINSTCP_OPEN

**说明**: 用于在运行模式下打开FINSTCP通讯连接

### 매개변수 목록

| 参数名 | 类型 | 取值范围 | 说明 |
|--------|------|----------|------|
| ID | int | [1,9] | FINSTCP主站工艺号 |
| logout | bool | - | 주석 처리 여부；true：작업 파일에서 실행되지 않음；false：주석 처리되지 않음, 실행 가능 |
| type | int | 215 | 打开FINSTCP连接在枚举数列type中为210 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| variable | object | - | 变量객체，包含data、secondvalue、value、varname等属性 |

### JSON示例

```json
{
  "ID": 5,
  "logout": false,
  "type": 215,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {
    "data": 0.0,
    "secondvalue": 0,
    "value": 0,
    "varname": ""
  }
}
```

## 3. FINSTCP连接状态

**指令名**: FINSTCP_CONNECTION_STATUS

**说明**: 在运行模式下获取FINSTCP连接状态

### 매개변수 목록

| 参数名 | 类型 | 取值范围 | 说明 |
|--------|------|----------|------|
| ID | int | [1,9] | FINSTCP主站工艺号 |
| logout | bool | - | 주석 처리 여부；true：작업 파일에서 실행되지 않음；false：주석 처리되지 않음, 실행 가능 |
| type | int | 217 | FINSTCP_CONNECTION_STATUS在枚举数列type中为217 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| variable | object | - | 变量객체 |

### variable객체

| 매개변수 | 유형 | 설명 |
|--------|------|------|
| data | float | 数据值 |
| secondvalue | int | 第二个值 |
| value | int | 状态值；6表示连接 |
| varname | string | 변수명，如"GB[I001]" |

### 返回值说明

FINSTCP的连接状态；bool类型变量或绑定变量，全局或局部

- `true`：连接
- `false`：未连接

### JSON示例

```json
{
  "ID": 5,
  "logout": false,
  "type": 217,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {
    "data": 0.0,
    "secondvalue": 1,
    "value": 6,
    "varname": "GB[I001]"
  }
}
```
