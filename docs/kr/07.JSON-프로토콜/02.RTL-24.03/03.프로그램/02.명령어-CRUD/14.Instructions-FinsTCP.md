# 명령어-FinsTCP 유형

## 1. FinsTCP 연결 열기

**명령어명**: FINSTCP_OPEN

**설명**: 실행 모드에서 FinsTCP 통신 연결을 여는 데 사용됩니다

### 매개변수 목록

| 매개변수 | 유형 | 값 범위 | 설명 |
|--------|------|----------|------|
| ID | int | [1,9] | FinsTCP 마스터 스테이션 프로세스 번호 |
| logout | bool | - | 주석 처리 여부; true: 작업 파일에서 실행되지 않음; false: 주석 처리되지 않음, 실행 가능 |
| type | int | 215 | FINSTCP_OPEN은 열거형에서 215 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| variable | object | - | 변수 객체, data, secondvalue, value, varname 등 속성 포함 |

### JSON 예시

```json
{
  "ID": 5,
  "logout": false,
  "type": 215,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {
    "data": 0.0,
    "secondvalue": 0,
    "value": 0,
    "varname": ""
  }
}
```

|--------|------|----------|------|

| type | int | 216 | FINSTCP_CLOSE는 열거형에서 216 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |

### JSON 예시

```json
{

  "type": 216,
  "userParamInt": 0,
```

## 3. FinsTCP 연결 상태

**명령어명**: FINSTCP_CONNECTION_STATUS

**설명**: 실행 모드에서 FinsTCP 연결 상태를 가져옵니다

### 매개변수 목록

| 매개변수 | 유형 | 값 범위 | 설명 |
|--------|------|----------|------|
| ID | int | [1,9] | FinsTCP 마스터 스테이션 프로세스 번호 |
| logout | bool | - | 주석 처리 여부; true: 작업 파일에서 실행되지 않음; false: 주석 처리되지 않음, 실행 가능 |
| type | int | 217 | FINSTCP_CONNECTION_STATUS는 열거형에서 217 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| variable | object | - | 변수 객체 |

### variable 객체

| 매개변수 | 유형 | 설명 |
|--------|------|------|
| data | float | 데이터 값 |
| secondvalue | int | 두 번째 값 |
| value | int | 상태 값; 6은 연결됨을 의미 |
| varname | string | 변수명, 예: "GB[I001]" |

### 반환 값 설명

FinsTCP 연결 상태; bool 유형 변수 또는 바인딩 변수, 전역 또는 로컬

- `true`: 연결됨
- `false`: 미연결

### JSON 예시

```json
{
  "ID": 5,
  "logout": false,
  "type": 217,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {
    "data": 0.0,
    "secondvalue": 1,
    "value": 6,
    "varname": "GB[I001]"
  }
}
```
