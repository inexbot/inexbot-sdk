# 명령어-팔레타이징제어类

## 1. 팔레타이징开始 (PALON)

**설명**：팔레타이징开始的判断，可通过修改변수제어码第几层第几个工件。

### 매개변수 설명

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| ID | int | 팔레타이징프로세스号，取值범위[1,99] |
| STACKTYPE | int | 팔레타이징유형，取值범위[0,1]<br/>0：팔레타이징<br/>1：卸垛 |
| _pCurrentLayersNum | 변수对象 | 当前层팔레타이징数；int유형변수或绑定변수，全局或局部<br/>当该변수的值表示string유형[-]时：表示不使用 |
| _pLayersNum | 변수对象 | 当前팔레타이징层；int유형변수或绑定변수，全局或局部<br/>当该변수的值表示string유형[-]时：表示不使用 |
| _pWorkpieceSum | 변수对象 | 当前已码总数；int유형변수或绑定변수，全局或局部<br/>当该변수的值表示string유형[-]时：表示不使用 |
| _stackSequence | int | 单层팔레타이징/卸垛顺序，取值범위[0,1]<br/>0：正序<br/>1：倒序 |
| logout | bool | 주석 처리 여부<br/>true：在작업파일中将不执行该명령어<br/>false：未주석 처리됨，可执行 |
| type | int | PALON在枚举数列type中为70 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 |

### 예시

```json
{
  "ID": 1,
  "STACKTYPE": 0,
  "_pCurrentLayersNum": {"data": 0.0, "secondvalue": 1, "value": 4, "varname": "GI[GI001]"},
  "_pLayersNum": {"data": 0.0, "secondvalue": 1, "value": 1, "varname": "I[I001]"},
  "_pWorkpieceSum": {"data": 0.0, "secondvalue": 0, "value": 1, "varname": "[-]"},
  "_stackSequence": 1,
  "logout": false,
  "type": 70,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 3. 팔레타이징入口点 (PLAENTER)

**설명**：选取抓手，팔레타이징入口点。

### 매개변수 설명

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| ID | int | 팔레타이징프로세스号，取值범위[1,99] |
| ParaACC | 변수对象 | 加속도；int유형或double유형변수或绑定변수，全局或局部；取值범위[1,100] |
| ParaDEC | 변수对象 | 减속도；int유형或double유형변수或绑定변수，全局或局部；取值범위[1,100] |
| ParaLINE | 변수对象 | 门型位移距离；手填值或int유형或double유형변수或绑定변수，全局或局部；取值범위[-5000,5000]<br/>插补方式选择门型时使用 |
| ParaPL | 변수对象 | 平滑；int유형변수或绑定변수，全局或局部；取值범위[0,5] |
| ParaTIME | 변수对象 | 提前执行时间；int유형或double유형변수或绑定변수，全局或局部 |
| ParaSTRAIGHT | 변수对象 | 门型直线距离；手填值或int유형或double유형변수或绑定변수，全局或局部；取值범위[0,5000] |
| ParaV | 변수对象 | 모션속도；int유형或double유형변수或绑定변수，全局或局部<br/>当모션方式为관절모션时：m_vUnit的值为2，속도单位为%；取值범위[1,100]<br/>当모션方式为圆弧或直线时：m_vUnit的值为1，속도单位为mm/s；取值범위[1,1000] |
| Posture | int | 姿态，取值범위[0,2]<br/>0：关闭<br/>1：手动写入<br/>自动计算姿态 |
| logout | bool | 주석 처리 여부<br/>true：在작업파일中将不执行该명령어<br/>false：未주석 처리됨，可执行 |
| moveType | string | 모션插补方式<br/>MovJ：관절<br/>MovL：直线<br/>MovC：圆弧<br/>MovArch：门型 |
| para_posture_ | 변수对象 | 姿态手动写入值；int유형或double유형변수或绑定변수，全局或局部 |
| type | int | PLAENTER在枚举数列type中为73 |
| offsetAxis | int | 位移轴，取值범위[0,2]<br/>0：X轴<br/>1：Y轴<br/>2：Z轴 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| xyOpt | int | 优化XY轴모션路径，取值범위[0,1]<br/>0：关闭<br/>1：打开 |
| zOpt | int | 优化Z轴모션路径，取值범위[0,1]<br/>0：关闭<br/>1：打开<br/>优化Z轴모션路径，需在팔레타이징前插入一个固定点 |

## 4. 팔레타이징辅助点 (PALSHIFT)

**설명**：팔레타이징辅助点。

**高度规则설명**：
- 当入口点高度比固定点低时，入口点在高度上会和固定点、辅助点处于同一值线上（侧视同一直线，俯视不是同一直线，XY轴不变）
- 当入口点高度位于固定点与辅助点之间时，入口点高度不变
- 当入口点高度位于固定点、辅助点之上时，入口点高度将优化到与固定点水平的高度
- 当入口点、辅助点高度都高于固定点时，入口点高度将优化到与辅助点水平高度

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| ID | int | 팔레타이징프로세스号，取值범위[1,99] |
| ParaACC | 변수对象 | 加속도；int유형或double유형변수或绑定변수，全局或局部；取值범위[1,100] |
| ParaDEC | 변수对象 | 减속도；int유형或double유형변수或绑定변수，全局或局部；取值범위[1,100] |
| ParaLINE | 변수对象 | 门型位移距离；手填值或int유형或double유형변수或绑定변수，全局或局部；取值범위[-5000,5000]<br/>插补方式选择门型时使用 |
| ParaPL | 변수对象 | 平滑；int유형변수或绑定변수，全局或局部；取值범위[0,5] |
| ParaSTRAIGHT | 변수对象 | 门型直线距离；手填值或int유형或double유형변수或绑定변수，全局或局部；取值범위[0,5000] |
| ParaTIME | 변수对象 | 提前执行时间；int유형或double유형변수或绑定변수，全局或局部 |
| ParaV | 변수对象 | 모션속도；int유형或double유형변수或绑定변수，全局或局部<br/>当모션方式为관절모션时：m_vUnit的值为2，속도单位为%；取值범위[1,100]<br/>当모션方式为圆弧或直线时：m_vUnit的值为1，속도单位为mm/s；取值범위[1,1000] |
| logout | bool | 주석 처리 여부<br/>true：在작업파일中将不执行该명령어<br/>false：未주석 처리됨，可执行 |
| moveType | string | 모션插补方式<br/>MovJ：관절<br/>MovL：直线<br/>MovC：圆弧<br/>MovArch：门型 |
| type | int | PALSHIFT在枚举数列type中为74 |
| offsetAxis | int | 位移轴，取值범위[0,2]<br/>0：X轴<br/>1：Y轴<br/>2：Z轴 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 |
