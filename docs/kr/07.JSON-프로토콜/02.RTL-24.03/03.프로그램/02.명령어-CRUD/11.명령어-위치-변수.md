## 사용자좌표수정 (USERFRAME_SET)

**설명**：수정또는者읽기사용자좌표계某一축값

**명령어名**：USERFRAME_SET | **ID**：888

### 매개변수 목록

| 매개변수名 | 유형 | 가져오기값을범위 | 설명 |
|--------|------|----------|------|
| coord | string | UX、UY、UZ、UA、UB、UC、UCUSTOM(UX、UY、UZ) | 사용자좌표매개변수。UX、UY、UZ、UA、UB、UC：읽기사용자좌표계某一축값；UCUSTOM(UX、UY、UZ)：사용자 정의，括번호에 복사합니다이(가)사용자 정의선택 시에 복사합니다사용자좌표 |
| logout | bool | true/false | 주석 처리 여부。true：에서작업파일에 복사합니다값을不실행해당명령어；false：未주석 처리됨，실행 가능 |
| m_varName | string | int또는double유형변수，전역또는로컬 | 변수。쓰기일 때가능手채우기int유형；가져오기값을범위[-2000,2000]。쓰기일 때：直연결手채우기값수정좌표축값，변수赋값을形式수정좌표축값。읽기일 때：사용하여변수形式값을선택좌표축값읽기출력来 |
| mode | int | [0,1] | 수정방식。0：읽기사용자좌표某一축값，1：수정사용자좌표某一축값 |
| type | int | 97 | 사용자좌표수정에서열거형数열type에 복사합니다이(가)97 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| varType | int | [0,4] | 변수값을유형。0：수동 입력 값，1：INT，2：DOUBLE，3：GINT，4：GDOUBLE |

### JSON 예시

```json
{
  "ID": 888,
  "coord": "UCUSTOM(UX、UY、UZ)",
  "logout": false,
  "m_varName": "I001",
  "mode": 1,
  "type": 97,
  "userParamInt": 0,
  "userParamString": "",
  "varType": 1
}
```

## 읽기포인트 (READPOS)

**설명**：값을위치변수某번째축값읽기입력부동소수점형변수

**명령어名**：READPOS | **type**：92

### 매개변수 목록

| 매개변수名 | 유형 | 가져오기값을범위 | 설명 |
|--------|------|----------|------|
| axis | int | [1,13] | 위치변수축。가져오기값을1~7:1~7축，가져오기값을8~13：외부 축O1~O6；현재구성有什么축，선택 시什么축 |
| coord | string | RF、BF、TF、UF | 위치변수좌표계。RF：관절좌표，BF：직교좌표，TF：도구좌표，UF：사용자좌표 |
| logout | bool | true/false | 주석 처리 여부。true：에서작업파일에 복사합니다값을不실행해당명령어；false：未주석 처리됨，실행 가능 |
| posidname | string | P,GP,E,GE변수名 | 위치변수名。때문자열이(가)空일 때의미현재읽기예로봇현재위치좌표。P,GP,E,GE:값을포인트저장입력변수，읽기일 때필요선택想필요읽기대상변수즉가능 |
| posidtype | int | [0,1] | 위치변수名标인식。0：위치변수名固설정，例：P001，G001；1：위치변수名不固설정，例：P$I001，P$GI001，G$I001，G$GI001 |
| type | int | 92 | 읽기포인트에서열거형数열type에 복사합니다이(가)92 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| varname | string | int유형변수名 | 대상변수名 |

### JSON 예시

```json
{
  "axis": 1,
  "coord": "RF",
  "logout": false,
  "posidname": "GE$GI001",
  "posidtype": 1,
  "type": 92,
  "userParamInt": 0,
  "userParamString": "",
  "varname": "D001"
}
```

## 포인트추가 (POSADD)

**설명**：에서不同좌표계에게위치변수某一번째변수축增추가값

**주의**：
- 수정대상포인트일 때주의超限问题와/과대상위치불가达，좌표없음法변환问题
- 관절超限：예를 들어로봇一축正리밋예180°,대상포인트一축좌표예150°，에서설정매개변수예에게변수축一축赋값을50°,실행명령어일 때하면표시되면관절超限问题
- 좌표없음法변환问题：예를 들어대상포인트예관절포인트,에서설정매개변수일 때선택변수좌표계예직교좌표계，에게대상포인트위치변수축赋값을，실행포인트추가명령어，포인트에서运算过程에 복사합니다가능가능하면표시되면좌표없음法변환问题

**명령어名**：POSADD | **type**：93

### 매개변수 목록

| 매개변수名 | 유형 | 가져오기값을범위 | 설명 |
|--------|------|----------|------|
| axis | int | [1,13] | 위치변수축。가져오기값을1~7:1~7축，가져오기값을8~13：외부 축O1~O6；현재구성有什么축，선택 시什么축 |
| caltype | int | [0,2] | 运算유형。0：ADDTYPE포인트추가；1：SUBTYPE포인트감소；2：SETTYPE포인트수정 |
| coord | string | RF、BF、TF、UF | 위치변수좌표계。RF：관절좌표，BF：직교좌표，TF：도구좌표，UF：사용자좌표 |
| datatype | int | [0,4] | 변수값을유형。0：수동 입력 값，1：INT，2：DOUBLE，3：GINT，4：GDOUBLE |
| desData | string | 수동 입력 값또는int/double유형변수，전역또는로컬 | 위치변수。가져오기값을범위[-50000，50000] |
| logout | bool | true/false | 주석 처리 여부。true：에서작업파일에 복사합니다값을不실행해당명령어；false：未주석 처리됨，실행 가능 |
| posidtype | int | [0,1] | 위치변수名标인식。0：위치변수名固설정，例：P001，G001；1：위치변수名不固설정，例：P$I001，P$GI001，G$I001，G$GI001 |
| positionId | string | P또는GP또는E또는GE | 필요수정포인트위치변수 |
| type | int | 93 | 포인트추가에서열거형数열type에 복사합니다이(가)93 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |

### JSON 예시

```json
{
  "axis": 5,
  "caltype": 0,
  "coord": "RF",
  "datatype": 1,
  "desData": "GI001",
  "logout": false,
  "posidtype": 0,
  "positionId": "GP0001",
  "type": 93,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 복사포인트 (COPYPOS)

**설명**：값을一번째위치변수포인트데이터복사另一번째위치변수
