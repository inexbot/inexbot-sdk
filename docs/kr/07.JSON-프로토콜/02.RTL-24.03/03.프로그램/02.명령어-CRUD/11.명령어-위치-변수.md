## 사용자좌표수정 (USERFRAME_SET)

**설명**：수정或者읽기사용자좌표계某一축的값

**명령어名**：USERFRAME_SET | **ID**：888

### 매개변수 목록

| 매개변수名 | 유형 | 取值범위 | 설명 |
|--------|------|----------|------|
| coord | string | UX、UY、UZ、UA、UB、UC、UCUSTOM(UX、UY、UZ) | 사용자좌표매개변수。UX、UY、UZ、UA、UB、UC：읽기사용자좌표계某一축的값；UCUSTOM(UX、UY、UZ)：自定义，括号中为自定义选中的사용자좌표 |
| logout | bool | true/false | 주석 처리 여부。true：在작업파일中将不실행该명령어；false：未주석 처리됨，실행 가능 |
| m_varName | string | int或double유형변수，全局或局部 | 변수。쓰기时可手填int유형；取值범위[-2000,2000]。쓰기时：直接手填값수정좌표축的값，변수赋值的形式수정좌표축的값。읽기时：用변수的形式将选择的좌표축的값읽기出来 |
| mode | int | [0,1] | 수정方式。0：읽기사용자좌표某一축的值，1：수정사용자좌표某一축的值 |
| type | int | 97 | 사용자좌표수정在열거형数열type中为97 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| varType | int | [0,4] | 변수值유형。0：수동 입력 값，1：INT，2：DOUBLE，3：GINT，4：GDOUBLE |

### JSON 예시

```json
{
  "ID": 888,
  "coord": "UCUSTOM(UX、UY、UZ)",
  "logout": false,
  "m_varName": "I001",
  "mode": 1,
  "type": 97,
  "userParamInt": 0,
  "userParamString": "",
  "varType": 1
}
```

| mode | int | [0,1] | 수정方式。0：읽기도구좌표某一축的值，1：수정도구좌표某一축的值 |

| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| varType | int | [0,4] | 변수值유형。0：수동 입력 값，1：INT，2：DOUBLE，3：GINT，4：GDOUBLE |

### JSON 예시

  "ID": 55,

## 읽기포인트 (READPOS)

**설명**：将위치변수的某번째축的值读入到부동소수점형변수

**명령어名**：READPOS | **type**：92

### 매개변수 목록

| 매개변수名 | 유형 | 取值범위 | 설명 |
|--------|------|----------|------|
| axis | int | [1,13] | 위치변수축。取值1~7:1~7축，取值8~13：외부 축O1~O6；현재구성有什么축，可以选什么축 |
| coord | string | RF、BF、TF、UF | 위치변수좌표계。RF：관절좌표，BF：直角좌표，TF：도구좌표，UF：사용자좌표 |
| logout | bool | true/false | 주석 처리 여부。true：在작업파일中将不실행该명령어；false：未주석 처리됨，실행 가능 |
| posidname | string | P,GP,E,GE변수名 | 위치변수名。当문자열为空时表示현재읽기的예로봇현재위치的좌표。P,GP,E,GE:将포인트存入변수，읽기时只需要选择想要읽기的目标변수即可 |
| posidtype | int | [0,1] | 위치변수名标识。0：위치변수名固定，例：P001，G001；1：위치변수名不固定，例：P$I001，P$GI001，G$I001，G$GI001 |
| type | int | 92 | 읽기포인트在열거형数열type中为92 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| varname | string | int유형변수名 | 目标변수名 |

### JSON 예시

```json
{
  "axis": 1,
  "coord": "RF",
  "logout": false,
  "posidname": "GE$GI001",
  "posidtype": 1,
  "type": 92,
  "userParamInt": 0,
  "userParamString": "",
  "varname": "D001"
}
```

## 포인트加 (POSADD)

**설명**：在不同좌표계下给위치변수的某一번째변수축增加값

**注意**：
- 수정目标포인트时注意超限问题和目标위치不可达，좌표없음法转换问题
- 관절超限：例如로봇一축的正리밋예180°,目标포인트一축的좌표예150°，在설정매개변수예给변수축一축赋值50°,실행명령어时就会出现관절超限问题
- 좌표없음法转换问题：例如目标포인트예관절포인트,在설정매개변수时选择的변수좌표계예直角좌표계，给目标포인트的위치변수축赋值，실행포인트加명령어，포인트在运算过程中可能会出现좌표없음法转换问题

**명령어名**：POSADD | **type**：93

### 매개변수 목록

| 매개변수名 | 유형 | 取值범위 | 설명 |
|--------|------|----------|------|
| axis | int | [1,13] | 위치변수축。取值1~7:1~7축，取值8~13：외부 축O1~O6；현재구성有什么축，可以选什么축 |
| caltype | int | [0,2] | 运算유형。0：ADDTYPE포인트加；1：SUBTYPE포인트减；2：SETTYPE포인트改 |
| coord | string | RF、BF、TF、UF | 위치변수좌표계。RF：관절좌표，BF：直角좌표，TF：도구좌표，UF：사용자좌표 |
| datatype | int | [0,4] | 변수值유형。0：수동 입력 값，1：INT，2：DOUBLE，3：GINT，4：GDOUBLE |
| desData | string | 수동 입력 값或int/double유형변수，全局或局部 | 위치변수。取值범위[-50000，50000] |
| logout | bool | true/false | 주석 처리 여부。true：在작업파일中将不실행该명령어；false：未주석 처리됨，실행 가능 |
| posidtype | int | [0,1] | 위치변수名标识。0：위치변수名固定，例：P001，G001；1：위치변수名不固定，例：P$I001，P$GI001，G$I001，G$GI001 |
| positionId | string | P或GP或E或GE | 需要수정포인트的위치변수 |
| type | int | 93 | 포인트加在열거형数열type中为93 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |

### JSON 예시

```json
{
  "axis": 5,
  "caltype": 0,
  "coord": "RF",
  "datatype": 1,
  "desData": "GI001",
  "logout": false,
  "posidtype": 0,
  "positionId": "GP0001",
  "type": 93,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 복사포인트 (COPYPOS)

**설명**：将一번째위치변수的포인트데이터복사到另一번째위치변수

**注意**：最大支持연결5번째외부 축，在복사외부 축포인트时只支持O1、O2、O3、O4、O5축的복사，如果在설정매개변수时勾选了O6、O7축，在실행복사포인트时O6、O7축不生效

**명령어名**：COPYPOS | **type**：118

### 매개변수 목록

| 매개변수名 | 유형 | 取值범위 | 설명 |
|--------|------|----------|------|
| cpaxles | int[] | 배열，每번째元素代表相应축号 | 选择需要복사的좌표축데이터 |
| cpaxletype | int | [0,1] | 表示여부所有좌표축데이터都복사。0：表示所有좌표축데이터都복사，此时没有"cpaxles"节点；1：看"cpaxles"节点选择了什么 |
| cpdposname | string | P或E，局部或全局 | 目标위치변수名 |
| cpdpostype | int | [0,1] | 위치변수名标识。0：위치변수名固定，例：P001，G001；1：위치변수名不固定，例：P$I001，P$GI001，G$I001，G$GI001 |
| cpposname | string | P,GP,E,GE변수名 | 源위치변수名。当문자열为空时表示현재읽기的예로봇현재위치的좌표 |
| cppostype | int | [0,1] | 위치변수名标识。0：위치변수名固定，例：P001，G001；1：위치변수名不固定，例：P$I001，P$GI001，G$I001，G$GI001 |
| logout | bool | true/false | 주석 처리 여부。true：在작업파일中将不실행该명령어；false：未주석 처리됨，실행 가능 |
| posidtype | int | [0,1] | 위치변수名标识。0：위치변수名固定，例：P001，G001；1：위치변수名不固定，例：P$I001，P$GI001，G$I001，G$GI001 |
| type | int | 118 | 복사포인트在열거형数열type中为118 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |

### JSON 예시

```json
{
  "cpaxles": [2, 4, 6],
  "cpaxletype": 1,
  "cpdposname": "GE$GI001",
  "cpdpostype": 1,
  "cpposname": "P$GI001",
  "cppostype": 1,
  "logout": false,
  "type": 118,
  "userParamInt": 0,
  "userParamString": ""
}
```

| varName | object | INT或GINT변수 | 形态值首번째변수。将计算出来的形态值存入到选择的目标변수，通过选择的首변수顺延저장计算出来的形态 |
| varNumber | object | INT或GINT변수 | 形态值번째数。将计算出来的形态值번째数存入到选择的目标변수 |
