## 사용자좌표修改 (USERFRAME_SET)

**설명**：修改或者读取사용자좌표계某一轴的数值

**명령어名**：USERFRAME_SET | **ID**：888

### 매개변수 목록

| 매개변수名 | 유형 | 取值범위 | 설명 |
|--------|------|----------|------|
| coord | string | UX、UY、UZ、UA、UB、UC、UCUSTOM(UX、UY、UZ) | 사용자좌표매개변수。UX、UY、UZ、UA、UB、UC：读取사용자좌표계某一轴的数值；UCUSTOM(UX、UY、UZ)：自定义，括号中为自定义选中的사용자좌표 |
| logout | bool | true/false | 주석 처리 여부。true：在작업파일中将不执行该명령어；false：未주석 처리됨，可执行 |
| m_varName | string | int或double유형변수，全局或局部 | 변수。写入时可手填int유형；取值범위[-2000,2000]。写入时：直接手填数值修改좌표轴的数值，변수赋值的形式修改좌표轴的数值。读取时：用변수的形式将选择的좌표轴的数值读取出来 |
| mode | int | [0,1] | 修改方式。0：读取사용자좌표某一轴的值，1：修改사용자좌표某一轴的值 |
| type | int | 97 | 사용자좌표修改在枚举数列type中为97 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| varType | int | [0,4] | 변수值유형。0：手填值，1：INT，2：DOUBLE，3：GINT，4：GDOUBLE |

### JSON 예시

```json
{
  "ID": 888,
  "coord": "UCUSTOM(UX、UY、UZ)",
  "logout": false,
  "m_varName": "I001",
  "mode": 1,
  "type": 97,
  "userParamInt": 0,
  "userParamString": "",
  "varType": 1
}
```

## 读取点位 (READPOS)

**설명**：将위치변수的某个轴的值读入到浮点型변수

**명령어名**：READPOS | **type**：92

### 매개변수 목록

| 매개변수名 | 유형 | 取值범위 | 설명 |
|--------|------|----------|------|
| axis | int | [1,13] | 위치변수轴。取值1~7:1~7轴，取值8~13：외부 축O1~O6；当前구성有什么轴，可以选什么轴 |
| coord | string | RF、BF、TF、UF | 위치변수좌표계。RF：관절좌표，BF：直角좌표，TF：도구좌표，UF：사용자좌표 |
| logout | bool | true/false | 주석 처리 여부。true：在작업파일中将不执行该명령어；false：未주석 처리됨，可执行 |
| posidname | string | P,GP,E,GE변수名 | 위치변수名。当字符串为空时表示当前读取的是로봇当前위치的좌표。P,GP,E,GE:将点位存入변수，读取时只需要选择想要读取的目标변수即可 |
| posidtype | int | [0,1] | 위치변수名标识。0：위치변수名固定，例：P001，G001；1：위치변수名不固定，例：P$I001，P$GI001，G$I001，G$GI001 |
| type | int | 92 | 读取点位在枚举数列type中为92 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| varname | string | int유형변수名 | 目标변수名 |

### JSON 예시

```json
{
  "axis": 1,
  "coord": "RF",
  "logout": false,
  "posidname": "GE$GI001",
  "posidtype": 1,
  "type": 92,
  "userParamInt": 0,
  "userParamString": "",
  "varname": "D001"
}
```

## 点位加 (POSADD)

**설명**：在不同좌표계下给위치변수的某一个변수轴增加数值

**注意**：
- 修改目标点位时注意超限问题和目标위치不可达，좌표无法转换问题
- 관절超限：例如로봇一轴的正限位是180°,目标点位一轴的좌표是150°，在설정매개변수是给변수轴一轴赋值50°,실행명령어时就会出现관절超限问题
- 좌표无法转换问题：例如目标点位是관절点位,在설정매개변수时选择的변수좌표계是直角좌표계，给目标点位的위치변수轴赋值，执行点位加명령어，点位在运算过程中可能会出现좌표无法转换问题

**명령어名**：POSADD | **type**：93

### 매개변수 목록

| 매개변수名 | 유형 | 取值범위 | 설명 |
|--------|------|----------|------|
| axis | int | [1,13] | 위치변수轴。取值1~7:1~7轴，取值8~13：외부 축O1~O6；当前구성有什么轴，可以选什么轴 |
| caltype | int | [0,2] | 运算유형。0：ADDTYPE点位加；1：SUBTYPE点位减；2：SETTYPE点位改 |
| coord | string | RF、BF、TF、UF | 위치변수좌표계。RF：관절좌표，BF：直角좌표，TF：도구좌표，UF：사용자좌표 |
| datatype | int | [0,4] | 변수值유형。0：手填值，1：INT，2：DOUBLE，3：GINT，4：GDOUBLE |
| desData | string | 手填值或int/double유형변수，全局或局部 | 위치변수。取值범위[-50000，50000] |
| logout | bool | true/false | 주석 처리 여부。true：在작업파일中将不执行该명령어；false：未주석 처리됨，可执行 |
| posidtype | int | [0,1] | 위치변수名标识。0：위치변수名固定，例：P001，G001；1：위치변수名不固定，例：P$I001，P$GI001，G$I001，G$GI001 |
| positionId | string | P或GP或E或GE | 需要修改点位的위치변수 |
| type | int | 93 | 点位加在枚举数列type中为93 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |

### JSON 예시

```json
{
  "axis": 5,
  "caltype": 0,
  "coord": "RF",
  "datatype": 1,
  "desData": "GI001",
  "logout": false,
  "posidtype": 0,
  "positionId": "GP0001",
  "type": 93,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 复制点位 (COPYPOS)

**설명**：将一个위치변수的点位데이터复制到另一个위치변수

**注意**：最大支持연결5个외부 축，在复制외부 축点位时只支持O1、O2、O3、O4、O5轴的复制，如果在설정매개변수时勾选了O6、O7轴，在执行复制点位时O6、O7轴不生效

**명령어名**：COPYPOS | **type**：118

### 매개변수 목록

| 매개변수名 | 유형 | 取值범위 | 설명 |
|--------|------|----------|------|
| cpaxles | int[] | 数组，每个元素代表相应轴号 | 选择需要复制的좌표轴데이터 |
| cpaxletype | int | [0,1] | 表示是否所有좌표轴데이터都复制。0：表示所有좌표轴데이터都复制，此时没有"cpaxles"节点；1：看"cpaxles"节点选择了什么 |
| cpdposname | string | P或E，局部或全局 | 目标위치변수名 |
| cpdpostype | int | [0,1] | 위치변수名标识。0：위치변수名固定，例：P001，G001；1：위치변수名不固定，例：P$I001，P$GI001，G$I001，G$GI001 |
| cpposname | string | P,GP,E,GE변수名 | 源위치변수名。当字符串为空时表示当前读取的是로봇当前위치的좌표 |
| cppostype | int | [0,1] | 위치변수名标识。0：위치변수名固定，例：P001，G001；1：위치변수名不固定，例：P$I001，P$GI001，G$I001，G$GI001 |
| logout | bool | true/false | 주석 처리 여부。true：在작업파일中将不执行该명령어；false：未주석 처리됨，可执行 |
| posidtype | int | [0,1] | 위치변수名标识。0：위치변수名固定，例：P001，G001；1：위치변수名不固定，例：P$I001，P$GI001，G$I001，G$GI001 |
| type | int | 118 | 复制点位在枚举数列type中为118 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |

### JSON 예시

```json
{
  "cpaxles": [2, 4, 6],
  "cpaxletype": 1,
  "cpdposname": "GE$GI001",
  "cpdpostype": 1,
  "cpposname": "P$GI001",
  "cppostype": 1,
  "logout": false,
  "type": 118,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 点位全加 (POSADDALL)

**설명**：在不同좌표계下给위치변수的若干个변수轴增加数值

**注意**：修改目标点位时注意超限问题和目标위치不可达，좌표无法转换问题

**명령어名**：POSADDALL | **type**：140

### 매개변수 목록

| 매개변수名 | 유형 | 取值범위 | 설명 |
|--------|------|----------|------|
| calData | string[] | 14个元素，每个元素为每个轴的增加值 | 轴增加数值。可手填也可为int或double변수유형，全局或局部 |
| caltype | int | [0,2] | 点位全조작유형。0：点位全加，1：点位全减，2：点位全改 |
| coord | string | RF、BF、TF、UF | 위치변수좌표계。RF：관절좌표，BF：直角좌표，TF：도구좌표，UF：사용자좌표 |
| datatype | int[] | 14个元素，与"calData"节点有对应关系 | 轴值是否改变。1：该轴有改变；2：该轴不改变 |
