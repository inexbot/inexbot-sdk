# 명령어-传送类

## 컨베이어工件检测开始

**명령어名**：CONVEYOR_CHECKPOS

**설명**：컨베이어工件检测开始

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| ID | int | 是 | 프로세스号，取值범위[1,9] |
| logout | bool | 是 | 주석 처리 여부：true=작업 파일에서 실행되지 않음；false=주석 처리되지 않음, 실행 가능 |
| type | int | 是 | 枚举值，CONVEYOR_CHECKPOS在枚举数列type中为107 |
| userParamInt | int | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

**JSON예시**：

```json
{
  "ID": 6,
  "logout": false,
  "type": 107,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 컨베이어跟踪开始

**명령어名**：CONVEYOR_ON

**설명**：컨베이어跟踪开始

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| ID | int | 是 | 프로세스号，取值범위[1,9] |
| ParaACC | object | 是 | 加속도，double유형手填值或int유형或double유형全局或局部변수或바인딩변수；取值범위[1,100]% |
| ParaV | object | 是 | 속도，double유형手填值或int유형或double유형全局或局部변수或바인딩변수；取值범위[2,1000]mm/s |
| logout | bool | 是 | 주석 처리 여부：true=작업 파일에서 실행되지 않음；false=주석 처리되지 않음, 실행 가능 |
| m_positionId | string | 否 | 点位，如："GP0001"；当填点位변수时사용，其余时刻为空 |
| m_postype | int | 否 | 当点位为工件点时值为1；取值범위[0,1] |
| posidname | string | 否 | 点位，如：GP$I001；当填的点位值为点位바인딩변수时사용，其余时刻为空 |
| posidtype | int | 否 | 点位유형：0=点位유형不为바인딩点位변수；1=点位유형为바인딩点位변수 |
| type | int | 是 | 枚举值，CONVEYOR_ON在枚举数列type中为105 |
| userParamInt | int | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

**JSON예시**：

```json
{
  "ID": 6,
  "ParaACC": {
    "data": 0.0,
    "secondvalue": 0,
    "value": 1,
    "varname": "GI007"
  },
  "ParaV": {
    "data": 0.0,
    "m_vUnit": 1,
    "secondvalue": 0,
    "value": 1,
    "varname": "GD003"
  },
  "logout": false,
  "m_positionId": "",
  "m_postype": 1,
  "posidname": "",
  "posidtype": 0,
  "type": 105,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 获取컨베이어跟踪위치

**명령어名**：CONVEYOR_POS

**설명**：获取컨베이어跟踪위치

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| ID | int | 是 | 프로세스号，取值범위[1,9] |
| dataVariable | object | 否 | 视觉附加데이터收个변수，int유형或double유형或string유형全局或局部或全局변수或바인딩변수；不사용时各个值都为空 |
| logout | bool | 是 | 주석 처리 여부：true=작업 파일에서 실행되지 않음；false=주석 처리되지 않음, 실행 가능 |
| m_positionId | string | 是 | 全局위치변수，GP변수；取值범위GP00001-GP9999 |
| type | int | 是 | 枚举值，CONVEYOR_POS在枚举数列type中为176 |
| userParamInt | int | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

**JSON예시**：

```json
{
  "ID": 6,
  "dataVariable": {
    "data": 0.0,
    "secondvalue": 1,
    "value": 5,
    "varname": "GD[I001]"
  },
  "logout": false,
  "m_positionId": "GP0019",
  "type": 176,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 删除컨베이어跟踪目标

**명령어名**：CONVEYOR_CLEAR

**설명**：删除컨베이어跟踪目标

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| ID | int | 是 | 프로세스号，取值범위[1,9] |
| logout | bool | 是 | 주석 처리 여부：true=작업 파일에서 실행되지 않음；false=주석 처리되지 않음, 실행 가능 |
| removeType | int | 是 | 删除범위：0=全部目标；1=本次目标；取值범위[0,1] |
| type | int | 是 | 枚举值，CONVEYOR_CLEAR在枚举数列type中为220 |
| userParamInt | int | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

**JSON예시**：

```json
{
  "ID": 6,
  "logout": false,
  "removeType": 1,
  "type": 220,
  "userParamInt": 0,
  "userParamString": ""
}
```
