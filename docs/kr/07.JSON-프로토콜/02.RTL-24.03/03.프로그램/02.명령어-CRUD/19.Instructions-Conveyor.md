## 컨베이어공작물감지시작

**명령어名**：CONVEYOR_CHECKPOS

**설명**：컨베이어공작물감지시작

| 매개변수名 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| ID | int | 예 | 프로세스号，값 범위범위[1,9] |
| logout | bool | 예 | 주석 처리 여부：true=작업 파일에서 실행되지 않음；false=주석 처리되지 않음, 실행 가능 |
| type | int | 예 | 열거형值，CONVEYOR_CHECKPOS在열거형数열type에서107 |
| userParamInt | int | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

**JSON예시**：

```json
{
  "ID": 6,
  "logout": false,
  "type": 107,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 컨베이어추적시작

**명령어名**：CONVEYOR_ON

**설명**：컨베이어추적시작

| 매개변수名 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| ID | int | 예 | 프로세스号，값 범위범위[1,9] |
| ParaACC | object | 예 | 加속도，double유형수동 입력 값或int유형或double유형전역或로컬변수或바인딩변수；값 범위범위[1,100]% |
| ParaV | object | 예 | 속도，double유형수동 입력 값或int유형或double유형전역或로컬변수或바인딩변수；값 범위범위[2,1000]mm/s |
| logout | bool | 예 | 주석 처리 여부：true=작업 파일에서 실행되지 않음；false=주석 처리되지 않음, 실행 가능 |
| m_positionId | string | 아니오 | 포인트，如："GP0001"；当填포인트변수时사용，其余时刻为空 |
| m_postype | int | 아니오 | 当포인트为공작물点时值为1；값 범위범위[0,1] |
| posidname | string | 아니오 | 포인트，如：GP$I001；当填的포인트值为포인트바인딩변수时사용，其余时刻为空 |
| posidtype | int | 아니오 | 포인트유형：0=포인트유형不为바인딩포인트변수；1=포인트유형为바인딩포인트변수 |
| type | int | 예 | 열거형值，CONVEYOR_ON在열거형数열type에서105 |
| userParamInt | int | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

**JSON예시**：

```json
{
  "ID": 6,
  "ParaACC": {
    "data": 0.0,
    "secondvalue": 0,
    "value": 1,
    "varname": "GI007"
  },
  "ParaV": {
    "data": 0.0,
    "m_vUnit": 1,
    "secondvalue": 0,
    "value": 1,
    "varname": "GD003"
  },
  "logout": false,
  "m_positionId": "",
  "m_postype": 1,
  "posidname": "",
  "posidtype": 0,
  "type": 105,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 가져오기컨베이어추적위치

**명령어名**：CONVEYOR_POS

**설명**：가져오기컨베이어추적위치

| 매개변수名 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| ID | int | 예 | 프로세스号，값 범위범위[1,9] |
| dataVariable | object | 아니오 | 비전附加데이터收번째변수，int유형或double유형或string유형전역或로컬或전역변수或바인딩변수；不사용时各번째值都为空 |
| logout | bool | 예 | 주석 처리 여부：true=작업 파일에서 실행되지 않음；false=주석 처리되지 않음, 실행 가능 |
| m_positionId | string | 예 | 전역위치변수，GP변수；값 범위범위GP00001-GP9999 |
| type | int | 예 | 열거형值，CONVEYOR_POS在열거형数열type에서176 |
| userParamInt | int | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

**JSON예시**：

```json
{
  "ID": 6,
  "dataVariable": {
    "data": 0.0,
    "secondvalue": 1,
    "value": 5,
    "varname": "GD[I001]"
  },
  "logout": false,
  "m_positionId": "GP0019",
  "type": 176,
  "userParamInt": 0,
  "userParamString": ""
}
```
