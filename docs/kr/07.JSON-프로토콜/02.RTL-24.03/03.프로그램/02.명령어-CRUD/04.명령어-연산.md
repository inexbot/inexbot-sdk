## 추가 (ADD)

**설명**：매개변수1및매개변수2相추가값저장입력结果변수

**명령어名**：ADD

| 필드 | 유형 | 설명 |
|------|------|------|
| desType1 | int | 변수1유형；이处가능채우기2 |
| desType2 | int | 변수2유형；이处가능채우기2 |
| logout | bool | 주석 처리 여부；true：에서작업파일에 복사합니다값을不실행해당명령어；false：未주석 처리됨，실행 가능 |
| paraSourceGroup | object | 结果변수；int유형변수또는double유형변수，全局또는局部변수 |
| paraValueGroup1 | object | 변수1；값또는int유형변수또는double유형변수，全局또는局部변수 |
| paraValueGroup2 | object | 변수2；값또는int유형변수또는double유형변수，全局또는局部변수 |
| type | int | ADD에서열거형序열에 복사합니다이(가)43 |
| userParamInt | int | 2차 개발客户사용자 정의매개변수（目이전없음意义） |
| userParamString | string | 2차 개발客户사용자 정의매개변수（目이전없음意义） |

**paraSourceGroup/paraValueGroup1/paraValueGroup2 结构**：

| 필드 | 유형 | 설명 |
|------|------|------|
| data | double | 값 |
| secondvalue | int | 변수유형标인식 |
| value | int | 변수참조标인식；0：사용data값；1：사용varname변수 |
| varname | string | 변수名 |

**예시**：

```json
{
  "desType1": 2,
  "desType2": 2,
  "logout": false,
  "paraSourceGroup": {"data": 0.0, "secondvalue": 0, "value": 1, "varname": "I001"},
  "paraValueGroup1": {"data": 2.0, "secondvalue": 0, "value": 0, "varname": ""},
  "paraValueGroup2": {"data": 5.0, "secondvalue": 0, "value": 0, "varname": ""},
  "type": 43,
  "userParamInt": 0,
  "userParamString": ""
}
```
