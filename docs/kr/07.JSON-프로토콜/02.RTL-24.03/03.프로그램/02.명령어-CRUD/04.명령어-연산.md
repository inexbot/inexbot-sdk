## 加 (ADD)

**설명**：매개변수1与매개변수2相加的值存入结果변수

**명령어名**：ADD

| 필드 | 유형 | 설명 |
|------|------|------|
| desType1 | int | 변수1的유형；此处只能填2 |
| desType2 | int | 변수2的유형；此处只能填2 |
| logout | bool | 주석 처리 여부；true：在작업파일中将不执行该명령어；false：未주석 처리됨，可执行 |
| paraSourceGroup | object | 结果변수；int유형변수或double유형변수，全局或局部변수 |
| paraValueGroup1 | object | 변수1；数值或int유형변수或double유형변수，全局或局部변수 |
| paraValueGroup2 | object | 변수2；数值或int유형변수或double유형변수，全局或局部변수 |
| type | int | ADD在枚举序列中为43 |
| userParamInt | int | 2차 개발客户自定义매개변수（目前无意义） |
| userParamString | string | 2차 개발客户自定义매개변수（目前无意义） |

**paraSourceGroup/paraValueGroup1/paraValueGroup2 结构**：

| 필드 | 유형 | 설명 |
|------|------|------|
| data | double | 数值 |
| secondvalue | int | 변수유형标识 |
| value | int | 변수引用标识；0：使用data数值；1：使用varname변수 |
| varname | string | 변수名 |

**예시**：

```json
{
  "desType1": 2,
  "desType2": 2,
  "logout": false,
  "paraSourceGroup": {"data": 0.0, "secondvalue": 0, "value": 1, "varname": "I001"},
  "paraValueGroup1": {"data": 2.0, "secondvalue": 0, "value": 0, "varname": ""},
  "paraValueGroup2": {"data": 5.0, "secondvalue": 0, "value": 0, "varname": ""},
  "type": 43,
  "userParamInt": 0,
  "userParamString": ""
}
```
