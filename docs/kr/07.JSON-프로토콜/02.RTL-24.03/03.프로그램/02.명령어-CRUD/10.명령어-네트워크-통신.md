# 명령어-网络통신类

---

## 전송데이터

**설명**：向个网络设备전송字符串信息或者변수值

**명령어名**：SENDMSG

| 매개변수 | 설명 | 유형 | 取值범위 |
|------|------|------|----------|
| ID | 프로세스号 | int | [1-9] |
| logout | 주석 처리 여부，true：작업 파일에서 실행되지 않음；false：주석 처리되지 않음, 실행 가능 | bool | - |
| sendChar | 전송的字符消息 | string | - |
| type | 전송데이터在枚举数列type中为94 | int | 94 |
| userParamInt | 2차 개발에서 사용자 정의, 현재 의미 없음 | int | - |
| userParamString | 2차 개발에서 사용자 정의, 현재 의미 없음 | string | - |

**JSON예시**：
```json
{
  "ID": 7,
  "logout": false,
  "sendChar": "UUUUUU",
  "type": 94,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 解析데이터

**설명**：该명령어用来解析外部设备传来的一组데이터，并将데이터存入多个변수

**명령어名**：PARSEMSG

| 매개변수 | 설명 | 유형 | 取值범위 |
|------|------|------|----------|
| ID | 프로세스号 | int | [1-9] |
| clearCache | 解析后是否清楚缓存区，0：否，1：是 | int | [0,1] |
| dataStorage | 记录提取데이터的数量；手填值int유형0或int유형변수，全局或局部；手填值0표시不记录提取데이터的数量 | object | - |
| dataVariable | 조회的데이터依次存放的首위치；int유형或double유형或string유형변수，全局或局部 | object | - |
| logout | 주석 처리 여부，true：작업 파일에서 실행되지 않음；false：주석 처리되지 않음, 실행 가능 | bool | - |
| type | 解析데이터在枚举数列type中为96 | int | 96 |
| userParamInt | 2차 개발에서 사용자 정의, 현재 의미 없음 | int | - |
| userParamString | 2차 개발에서 사용자 정의, 현재 의미 없음 | string | - |

**JSON예시**：
```json
{
  "ID": 3,
  "clearCache": 1,
  "dataStorage": {
    "data": 0.0,
    "secondvalue": 0,
    "value": 4,
    "varname": "GI009"
  },
  "dataVariable": {
    "data": 0.0,
    "secondvalue": 1,
    "value": 5,
    "varname": "GD[GI001]"
  },
  "logout": false,
  "type": 96,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 读取

**설명**：读取通过以太网或Modbus통신전송点位，并将点位存入到변수

**명령어名**：READCOMM

| 매개변수 | 설명 | 유형 | 取值범위 |
|------|------|------|----------|
| ID | 프로세스号 | int | [1-9] |
| commType | 통신方式，0：ETHERENT통신，1：MODBUS통신 | int | [0,1] |
| logout | 주석 처리 여부，true：작업 파일에서 실행되지 않음；false：주석 처리되지 않음, 실행 가능 | bool | - |
| m_varname | 点位存放个数；int유형변수，全局或局部 | string | - |
| positionId | 点位存放首个변수；P：点位局部변수，GP：点位全局변수 | string | - |
| type | 读取在枚举数列type中为110 | int | 110 |
| userParamInt | 2차 개발에서 사용자 정의, 현재 의미 없음 | int | - |
| userParamString | 2차 개발에서 사용자 정의, 현재 의미 없음 | string | - |

**JSON예시**：
```json
{
  "ID": 6,
  "commType": 1,
  "logout": false,
  "m_varname": "I001",
  "positionId": "GP0001",
  "type": 110,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 获取信息연결상태

**설명**：获取网络설정里某个프로세스号的연결상태

**명령어名**：MSG_CONNECTION_STATUS

| 매개변수 | 설명 | 유형 | 取值범위 |
|------|------|------|----------|
| ID | TCP통신时연결的프로세스号 | int | [1-9] |
| logout | 주석 처리 여부，true：작업 파일에서 실행되지 않음；false：주석 처리되지 않음, 실행 가능 | bool | - |
| type | 获取信息연결상태在枚举数列type中为95 | int | 95 |
| userParamInt | 2차 개발에서 사용자 정의, 현재 의미 없음 | int | - |
| userParamString | 2차 개발에서 사용자 정의, 현재 의미 없음 | string | - |
| variable | 상태存入변수名；bool유형변수，全局或局部 | object | - |

**JSON예시**：
```json
{
  "ID": 5,
  "logout": false,
  "type": 95,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {
    "data": 0.0,
    "secondvalue": 1,
    "value": 6,
    "varname": "GB[GI001]"
  }
}
```
