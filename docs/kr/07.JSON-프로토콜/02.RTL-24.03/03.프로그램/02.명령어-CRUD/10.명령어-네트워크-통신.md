# 명령어-网络통신클래스

---

## 전송데이터

**설명**：向번째网络设备전송문자열信息或者변수值

**명령어名**：SENDMSG

| 매개변수 | 설명 | 유형 | 取值범위 |
|------|------|------|----------|
| ID | 프로세스号 | int | [1-9] |
| logout | 주석 처리 여부，true：작업 파일에서 실행되지 않음；false：주석 처리되지 않음, 실행 가능 | bool | - |
| sendChar | 전송的字符메시지 | string | - |
| type | 전송데이터在열거형数열type中为94 | int | 94 |
| userParamInt | 2차 개발에서 사용자 정의, 현재 의미 없음 | int | - |
| userParamString | 2차 개발에서 사용자 정의, 현재 의미 없음 | string | - |

**JSON예시**：
```json
{
  "ID": 7,
  "logout": false,
  "sendChar": "UUUUUU",
  "type": 94,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 解析데이터

**설명**：该명령어用来解析外部设备传来的一组데이터，并将데이터存入多번째변수

**명령어名**：PARSEMSG

| 매개변수 | 설명 | 유형 | 取值범위 |
|------|------|------|----------|
| ID | 프로세스号 | int | [1-9] |
| clearCache | 解析后여부清楚캐시区，0：아니오，1：예 | int | [0,1] |
| dataStorage | 记录提取데이터的数量；수동 입력 값int유형0或int유형변수，全局或局部；수동 입력 값0표시不记录提取데이터的数量 | object | - |
| dataVariable | 조회的데이터依회存放的首위치；int유형或double유형或string유형변수，全局或局部 | object | - |
| logout | 주석 처리 여부，true：작업 파일에서 실행되지 않음；false：주석 처리되지 않음, 실행 가능 | bool | - |
| type | 解析데이터在열거형数열type中为96 | int | 96 |
| userParamInt | 2차 개발에서 사용자 정의, 현재 의미 없음 | int | - |
| userParamString | 2차 개발에서 사용자 정의, 현재 의미 없음 | string | - |

**JSON예시**：
```json
{
  "ID": 3,
  "clearCache": 1,
  "dataStorage": {
    "data": 0.0,
    "secondvalue": 0,
    "value": 4,
    "varname": "GI009"
  },
  "dataVariable": {
    "data": 0.0,
    "secondvalue": 1,
    "value": 5,
    "varname": "GD[GI001]"
  },
  "logout": false,
  "type": 96,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 읽기

**설명**：읽기通过以太网或Modbus통신전송포인트，并将포인트存入到변수

**명령어名**：READCOMM

| 매개변수 | 설명 | 유형 | 取值범위 |
|------|------|------|----------|
| ID | 프로세스号 | int | [1-9] |
| commType | 통신方式，0：ETHERENT통신，1：MODBUS통신 | int | [0,1] |
| logout | 주석 처리 여부，true：작업 파일에서 실행되지 않음；false：주석 처리되지 않음, 실행 가능 | bool | - |
| m_varname | 포인트存放번째数；int유형변수，全局或局部 | string | - |
| positionId | 포인트存放首번째변수；P：포인트局部변수，GP：포인트全局변수 | string | - |
| type | 읽기在열거형数열type中为110 | int | 110 |
| userParamInt | 2차 개발에서 사용자 정의, 현재 의미 없음 | int | - |
| userParamString | 2차 개발에서 사용자 정의, 현재 의미 없음 | string | - |

**JSON예시**：
```json
{
  "ID": 6,
  "commType": 1,
  "logout": false,
  "m_varname": "I001",
  "positionId": "GP0001",
  "type": 110,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 가져오기信息연결상태

**설명**：가져오기网络설정里某번째프로세스号的연결상태

**명령어名**：MSG_CONNECTION_STATUS

| 매개변수 | 설명 | 유형 | 取值범위 |
|------|------|------|----------|
| ID | TCP통신时연결的프로세스号 | int | [1-9] |
| logout | 주석 처리 여부，true：작업 파일에서 실행되지 않음；false：주석 처리되지 않음, 실행 가능 | bool | - |
| type | 가져오기信息연결상태在열거형数열type中为95 | int | 95 |
| userParamInt | 2차 개발에서 사용자 정의, 현재 의미 없음 | int | - |
| userParamString | 2차 개발에서 사용자 정의, 현재 의미 없음 | string | - |
| variable | 상태存入변수名；bool유형변수，全局或局部 | object | - |

**JSON예시**：
```json
{
  "ID": 5,
  "logout": false,
  "type": 95,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {
    "data": 0.0,
    "secondvalue": 1,
    "value": 6,
    "varname": "GB[GI001]"
  }
}
```
