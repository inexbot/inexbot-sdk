# 명령어-문자열클래스

---

## 1. 문자열추가 쓰기 (STRING_SPELL)

**설명**：在原有문자열변수或者新的字符변수中추가 쓰기需要的字符，就构成了一번째新的문자열변수。

**명령어名**：STRING_SPELL

**type 值**：223

### 매개변수 목록

| 매개변수名 | 유형 | 설명 | 예시值 |
|--------|------|------|--------|
| logout | bool | 주석 처리 여부。true：在작업파일中将不실행该명령어；false：未주석 처리됨，실행 가능 | false |
| type | int | 명령어유형，문자열추가 쓰기在열거형数열中为 223 | 223 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 | 0 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 | "" |
| variable | object | 변수유형及이름；string유형변수，全局或局部 | `{"data":0.0,"secondvalue":0,"value":8,"varname":"S001"}` |
| variableValue | object | 추가 쓰기的字符변수；string유형변수，全局或局部 | `{"data":0.0,"secondvalue":0,"value":8,"varname":"GS001"}` |

### JSON 예시

```json
{
  "logout": false,
  "type": 223,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {"data": 0.0, "secondvalue": 0, "value": 8, "varname": "S001"},
  "variableValue": {"data": 0.0, "secondvalue": 0, "value": 8, "varname": "GS001"}
}
```

## 2. 인덱스截取 (STRING_SLICE)

**설명**：截取一번째문자열변수中的其中一部分문자열，并把截取到部分문자열存到指定的변수。

**명령어名**：STRING_SLICE

**type 值**：224

### 매개변수 목록

| 매개변수名 | 유형 | 설명 | 예시值 |
|--------|------|------|--------|
| logout | bool | 주석 처리 여부。true：在작업파일中将不실행该명령어；false：未주석 처리됨，실행 가능 | false |
| type | int | 명령어유형，인덱스截取在열거형数열中为 224 | 224 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 | 0 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 | "" |
| startIndex | object | 起始인덱스위치；int유형수동 입력 값或者변수，全局或局部 | `{"data":2.0,"secondvalue":0,"value":0,"varname":""}` |
| endIndex | object | 종료인덱스위치；int유형수동 입력 값或者변수，全局或局部 | `{"data":8.0,"secondvalue":0,"value":0,"varname":""}` |
| dataVariable | object | 조회的데이터依회存放的위치；string유형변수，全局或局部 | `{"data":0.0,"secondvalue":1,"value":8,"varname":"GS[I001]"}` |
| variable | object | 被截取的문자열변수；string유형변수，全局或局部 | `{"data":0.0,"secondvalue":0,"value":8,"varname":"S001"}` |

### JSON 예시

```json
{
  "logout": false,
  "type": 224,
  "userParamInt": 0,
  "userParamString": "",
  "startIndex": {"data": 2.0, "secondvalue": 0, "value": 0, "varname": ""},
  "endIndex": {"data": 8.0, "secondvalue": 0, "value": 0, "varname": ""},
  "dataVariable": {"data": 0.0, "secondvalue": 1, "value": 8, "varname": "GS[I001]"},
  "variable": {"data": 0.0, "secondvalue": 0, "value": 8, "varname": "S001"}
}
```

## 3. 문자열分隔符拆分 (STRING_SPLIT)

**설명**：将문자열변수中的其中一번째字符拆分，并把拆分的字符依회存放到指定변수中。

**명령어名**：STRING_SPLIT

**type 值**：225

### 매개변수 목록

| 매개변수名 | 유형 | 설명 | 예시值 |
|--------|------|------|--------|
| logout | bool | 주석 처리 여부。true：在작업파일中将不실행该명령어；false：未주석 처리됨，실행 가능 | false |
| type | int | 명령어유형，문자열分隔符拆分在열거형数열中为 225 | 225 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 | 0 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 | "" |
| separator | object | 分隔符；string유형的수동 입력 값或변수，全局或局部 | `{"data":0.0,"secondvalue":0,"value":8,"varname":"GS001"}` |
| variable | object | 被提取的문자열변수名；string유형변수，全局或局部 | `{"data":0.0,"secondvalue":1,"value":8,"varname":"S[I001]"}` |
| dataVariable | object | 조회的데이터依회存放的首위치；string유형변수，全局或局部 | `{"data":0.0,"secondvalue":0,"value":8,"varname":"S001"}` |
| dataStorage | object | 记录提取데이터的数量；string유형的手填"0"或者int유형변수，全局或局部。string유형的手填"0"表示不使用 | `{"data":0.0,"secondvalue":1,"value":8,"varname":"GI[I001]"}` |

### JSON 예시

```json
{
  "dataStorage": {"data": 0.0, "secondvalue": 1, "value": 8, "varname": "GI[I001]"},
  "dataVariable": {"data": 0.0, "secondvalue": 0, "value": 8, "varname": "S001"},
  "logout": false,
  "separator": {"data": 0.0, "secondvalue": 0, "value": 8, "varname": "GS001"},
  "type": 225,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {"data": 0.0, "secondvalue": 1, "value": 8, "varname": "S[I001]"}
}
```

## 4. 문자열定비트 (STRING_LOCATE)

**설명**：조회문자열변수里面的一种字符所在的위치，并把위치和数量存到指定변수。

> **注意**：
> - 如果待인덱스변수在一번째문자열里面예不连续的字符，在실행定비트조회명령어时，字符的위치和번째数예없음法읽기出来的
> - 如果待인덱스변수在一번째문자열里面예连续的字符，在실행定비트조회명령어时，읽기出来的字符위치예제一번째인덱스변수的위치，데이터存放번째数为1

**명령어名**：STRING_LOCATE

**type 值**：226

### 매개변수 목록

| 매개변수名 | 유형 | 설명 | 예시值 |
|--------|------|------|--------|
| logout | bool | 주석 처리 여부。true：在작업파일中将不실행该명령어；false：未주석 처리됨，실행 가능 | false |
| type | int | 명령어유형，문자열定비트在열거형数열中为 226 | 226 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 | 0 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 | "" |
| variable | object | 被提取的문자열변수名；string유형변수，全局或局部 | `{"data":0.0,"secondvalue":0,"value":8,"varname":"S001"}` |
| startIndex | object | 需要搜索的字符；string유형수동 입력 값或변수，全局或局部 | `{"data":0.0,"secondvalue":0,"value":7,"varname":"RRRR"}` |
| dataVariable | object | 조회的데이터依회存放的首위치；int유형변수，全局或局部 | `{"data":0.0,"secondvalue":0,"value":8,"varname":"I001"}` |
| dataStorage | object | 记录提取데이터的数量；string유형的手填"0"或者int유형변수，全局或局部。string유형的手填"0"表示不使用 | `{"data":0.0,"secondvalue":1,"value":8,"varname":"GI[GI001]"}` |

### JSON 예시

```json
{
  "dataStorage": {"data": 0.0, "secondvalue": 1, "value": 8, "varname": "GI[GI001]"},
  "dataVariable": {"data": 0.0, "secondvalue": 0, "value": 8, "varname": "I001"},
  "logout": false,
  "startIndex": {"data": 0.0, "secondvalue": 0, "value": 7, "varname": "RRRR"},
  "type": 226,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {"data": 0.0, "secondvalue": 0, "value": 8, "varname": "S001"}
}
```

## 5. 문자열길이 (STRING_LENGTH)

**설명**：计算一번째문자열변수的길이，并把计算的길이存到변수。

**명령어名**：STRING_LENGTH

**type 值**：227

|--------|------|------|--------|

| type | int | 명령어유형，문자열길이在열거형数열中为 227 | 227 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 | 0 |

| dataVariable | object | 计算结果存储的结果；int유형변수，全局或局部 | `{"data":0.0,"secondvalue":0,"value":8,"varname":"I001"}` |

## 6. 문자열转非문자열 (STRING_TO)

**설명**：문자열변수转换成非문자열변수。

> **转换规则**：문자열변수转换成非문자열변수时只会将문자열변수的값部分转译到非문자열변수。在转译过程中会从문자열변수的제一번째字符시작转译，直到检测到的字符예非값的字符时转译종료，将检测到的값部分的字符转译到非字符변수。如果检测到的제一번째字符就예非값的字符，那문자열없음法转译성공。

**명령어名**：STRING_TO

**type 值**：228

### 매개변수 목록

| 매개변수名 | 유형 | 설명 | 예시值 |
|--------|------|------|--------|
| logout | bool | 주석 처리 여부。true：在작업파일中将不실행该명령어；false：未주석 처리됨，실행 가능 | false |
| type | int | 명령어유형，문자열转非문자열在열거형数열中为 228 | 228 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 | 0 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 | "" |
| variable | object | 待计算길이的변수；string유형변수，全局或局部 | `{"data":0.0,"secondvalue":0,"value":8,"varname":"S001"}` |
| nonVariable | object | 转译的目标변수；int유형或double유형或bool유형변수，局部或全局 | `{"data":0.0,"secondvalue":0,"value":8,"varname":"I001"}` |

### JSON 예시

```json
{
  "nonVariable": {"data": 0.0, "secondvalue": 0, "value": 8, "varname": "I001"},
  "logout": false,
  "type": 228,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {"data": 0.0, "secondvalue": 0, "value": 8, "varname": "S001"}
}
```

## 7. 非문자열转문자열 (TO_STRING)

|--------|------|------|--------|

| type | int | 명령어유형，非문자열转문자열在열거형数열中为 229 | 229 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 | 0 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 | "" |

| nonVariable | object | 需要转译的변수；int유형或double유형或bool유형변수，局部或全局 | `{"data":0.0,"secondvalue":1,"value":8,"varname":"GI[GI001]"}` |

```json
}
```
