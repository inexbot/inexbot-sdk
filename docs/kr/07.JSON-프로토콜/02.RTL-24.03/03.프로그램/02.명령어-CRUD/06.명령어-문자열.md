# 명령어-문자열클래스

## 1. 문자열추가 쓰기 (STRING_SPELL)

**설명**：기존문자열변수또는 새로운문자변수에 복사합니다추가 쓰기필요문자，构완료一번째갱신문자열변수。

**명령어名**：STRING_SPELL

**type 값을**：223

### 매개변수 목록

| 매개변수名 | 유형 | 설명 | 예시값을 |
|--------|------|------|--------|
| logout | bool | 주석 처리 여부。true：에서작업파일에 복사합니다값을不실행해당명령어；false：未주석 처리됨，실행 가능 | false |
| type | int | 명령어유형，문자열추가 쓰기에서열거형数열에 복사합니다이(가) 223 | 223 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 | 0 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 | "" |
| variable | object | 변수유형及이름；string유형변수，전역또는로컬 | `{"data":0.0,"secondvalue":0,"value":8,"varname":"S001"}` |
| variableValue | object | 추가 쓰기문자변수；string유형변수，전역또는로컬 | `{"data":0.0,"secondvalue":0,"value":8,"varname":"GS001"}` |

### JSON 예시

```json
{
  "logout": false,
  "type": 223,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {"data": 0.0, "secondvalue": 0, "value": 8, "varname": "S001"},
  "variableValue": {"data": 0.0, "secondvalue": 0, "value": 8, "varname": "GS001"}
}
```

## 2. 인덱스추출 (STRING_SLICE)

**설명**：추출一번째문자열변수에 복사합니다其에 복사합니다一부분문자열，위치와추출부분문자열지정된 변수에 저장합니다변수。

**명령어名**：STRING_SLICE

**type 값을**：224

### 매개변수 목록

| 매개변수名 | 유형 | 설명 | 예시값을 |
|--------|------|------|--------|
| logout | bool | 주석 처리 여부。true：에서작업파일에 복사합니다값을不실행해당명령어；false：未주석 처리됨，실행 가능 | false |
| type | int | 명령어유형，인덱스추출에서열거형数열에 복사합니다이(가) 224 | 224 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 | 0 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 | "" |
| startIndex | object | 起시작인덱스위치；int유형수동 입력 값또는者변수，전역또는로컬 | `{"data":2.0,"secondvalue":0,"value":0,"varname":""}` |
| endIndex | object | 종료인덱스위치；int유형수동 입력 값또는者변수，전역또는로컬 | `{"data":8.0,"secondvalue":0,"value":0,"varname":""}` |

}

## 6. 문자열转非문자열 (STRING_TO)

**설명**：문자열변수변환완료非문자열변수。

> **변환 규칙**：문자열변수변환완료非문자열변수일 때하면값을문자열변수값부분변환非문자열변수。변환 과정에서문자열변수제一번째문자시작변환，값이 아닌 문자가 감지될 때까지예非값문자일 때변환종료，감지된값부분의 문자를 대상 변수로 변환합니다非문자변수。만일확인제一번째문자예非값문자，那문자열없음法변환성공。

**명령어名**：STRING_TO

**type 값을**：228

### 매개변수 목록

| 매개변수名 | 유형 | 설명 | 예시값을 |
|--------|------|------|--------|
| logout | bool | 주석 처리 여부。true：에서작업파일에 복사합니다값을不실행해당명령어；false：未주석 처리됨，실행 가능 | false |
| type | int | 명령어유형，문자열转非문자열에서열거형数열에 복사합니다이(가) 228 | 228 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 | 0 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 | "" |
| variable | object | 계산 대상길이변수；string유형변수，전역또는로컬 | `{"data":0.0,"secondvalue":0,"value":8,"varname":"S001"}` |
| nonVariable | object | 변환대상변수；int유형또는double유형또는bool유형변수，로컬또는전역 | `{"data":0.0,"secondvalue":0,"value":8,"varname":"I001"}` |

### JSON 예시

```json
{
  "nonVariable": {"data": 0.0, "secondvalue": 0, "value": 8, "varname": "I001"},
  "logout": false,
  "type": 228,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {"data": 0.0, "secondvalue": 0, "value": 8, "varname": "S001"}
}
```
