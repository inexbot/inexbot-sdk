# 명령어-모션 제어클래스

## 외부 축모션

**설명**：및외부 축有닫기모션명령어，필수예에서존재외부 축이전제공사용할 수 있습니다。

---

### MOVJEXT - 외부 축点点모션

**명령어유형**：type = 33

#### 매개변수 설명

| 매개변수名 | 유형 | 가져오기값을범위 | 단위 | 설명 |
|--------|------|----------|------|------|
| ParaACC | double | [1, 100] | % | 추가속도 |
| ParaDEC | double | [1, 100] | % | 감소속도 |
| ParaEVJ | double | [1, 100] | - | 외부 축속도조정정리比率；값을이(가)문자열"0"일 때사용 안 함을 의미 |
| ParaV | double | [1, 100] | % | 속도（m_vUnit필수이(가)2） |
| ParaPL | double | [0, 5] | - | 평활 |
| ParaTIME | double | 自然数 | - | 제공이전실행일 때사이 |
| logout | bool | true/false | - | 주석 처리 여부；true：에서작업파일에 복사합니다값을不실행해당명령어 |
| userParamInt | int | - | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| posidname | string | - | - | 绑설정변수일 때포인트名 |
| positionId | string | - | - | 非绑설정변수일 때포인트名 |
| posidtype | int | 0또는1 | - | 포인트여부이(가)绑설정변수；1：绑설정변수，0：非绑설정변수 |
| RobotPos | object | - | - | 포인트정보（때사용绑설정변수포인트일 때，不사용해당节点） |

#### m_vUnit 속도단위설명

| 값을 | 설명 |
|----|------|
| 0 | cm/s |
| 1 | mm/s |
| 2 | %（만가능사용2） |

| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
