# 명령어-모션 제어클래스

## 외부 축모션

**설명**：与외부 축有关的모션명령어，필수예在존재외부 축的前提下才可以使用。

---

### MOVJEXT - 외부 축点到点모션

**명령어유형**：type = 33

#### 매개변수 설명

| 매개변수名 | 유형 | 取值범위 | 단위 | 설명 |
|--------|------|----------|------|------|
| ParaACC | double | [1, 100] | % | 加속도 |
| ParaDEC | double | [1, 100] | % | 减속도 |
| ParaEVJ | double | [1, 100] | - | 외부 축속도调整比率；值为문자열"0"时表示不使用 |
| ParaV | double | [1, 100] | % | 속도（m_vUnit필수为2） |
| ParaPL | double | [0, 5] | - | 평활 |
| ParaTIME | double | 自然数 | - | 提前실행时间 |
| logout | bool | true/false | - | 주석 처리 여부；true：在작업파일中将不실행该명령어 |
| userParamInt | int | - | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| posidname | string | - | - | 绑定변수时的포인트名 |
| positionId | string | - | - | 非绑定변수时的포인트名 |
| posidtype | int | 0或1 | - | 포인트여부为绑定변수；1：绑定변수，0：非绑定변수 |
| RobotPos | object | - | - | 포인트信息（当使用绑定변수포인트时，不使用该节点） |

#### m_vUnit 속도단위설명

| 值 | 설명 |
|----|------|
| 0 | cm/s |
| 1 | mm/s |
| 2 | %（仅可使用2） |

### EXTMOV - 외부 축随动모션

**명령어유형**：type = 144

#### 매개변수 설명

| 매개변수名 | 유형 | 取值범위 | 설명 |
|--------|------|----------|------|
| _pVarName | double | [-2000, 0) ∪ (0, 2000] | 속도值 |
| extaxle | int | 1, 2, 3 | 외부 축编号；1：O1，2：O2，3：O3 |
| movtype | int | 0, 1, 2 | 외부 축随动方式；0：随线속도动，1：恒速모션，2：绑定변수 |
| p_move_type | object | - | 当movtype为2时，绑定的변수为该节点 |
| logout | bool | true/false | 주석 처리 여부 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |

#### 예시

```json
{
    "_pVarName": {
        "data": 2.90,
        "secondvalue": 0,
        "value": 0,
        "varname": ""
    },
    "extaxle": 1,
    "logout": false,
    "movtype": 2,
    "p_move_type": {
        "data": 0.0,
        "secondvalue": 0,
        "value": 1,
        "varname": "I001"
    },
    "p_speed_proportion_factor": {
        "data": 1.0,
        "secondvalue": 0,
        "value": 0,
        "varname": ""
    },
    "type": 144,
    "userParamInt": 0,
    "userParamString": ""
}
```
