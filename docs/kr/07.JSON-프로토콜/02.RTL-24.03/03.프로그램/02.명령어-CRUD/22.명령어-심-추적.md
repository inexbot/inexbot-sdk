## 激光追踪开始

**설명**：激光追踪开始

**명령어名**：LASERTRACK_ON

**type 值**：85

### 매개변수表

| 매개변수名 | 유형 | 取值범위 | 설명 |
|--------|------|----------|------|
| ID | int | [1,99] | 跟踪파일号 |
| delay | double | [0,10] | 매개변수表延时 |
| segmentParam | int | [1,99] | 跟踪매개변수表号 |
| logout | bool | - | 주석 처리 여부：true=在작업파일中将不执行该명령어；false=未주석 처리됨，可执行 |
| type | int | - | 枚举值，LASERTRACK_ON 在枚举数列中为 85 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |

### JSON 예시

```json
{
  "ID": 2,
  "delay": 4.0,
  "logout": false,
  "segmentParam": 3,
  "type": 85,
  "userParamInt": 0,
  "userParamString": ""
}
```
