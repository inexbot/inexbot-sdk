# 명령어-프로그램제어클래스

## 开启스레드

**설명**：开启전역后台或者로컬后台任务

**명령어名**：PTHREAD_START

**type值**：90

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| jobName | string | 后台任务프로그램파일名 |
| localOrGlobal | int | 后台유형：0-로컬后台，1-전역后台 |
| logout | bool | 주석 처리 여부：true-작업 파일에서 실행되지 않음，false-未被주석실행 가능 |
| userParamInt | int | 2차 개발客户自定义매개변수（目前없음의미） |
| userParamString | string | 2차 개발客户自定义매개변수（目前없음의미） |

**요청예시**：
```json
{
    "jobName": "SFDFF",
    "localOrGlobal": 0,
    "logout": false,
    "type": 90,
    "userParamInt": 0,
    "userParamString": ""
}
```

## 弹窗명령어

**설명**：실행명령어时弹出所填写的提示内容窗口，버튼수량为选项수량，点击버튼会将변수值存入바인딩的변수

**명령어名**：WINDOW

**type值**：208

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| logout | bool | 주석 처리 여부：true-작업 파일에서 실행되지 않음，false-未被주석실행 가능 |
| userParamInt | int | 2차 개발客户自定义매개변수（目前없음의미） |
| userParamString | string | 2차 개발客户自定义매개변수（目前없음의미） |
| count | int | 选项수량，값 범위범위[0,3] |
| text1 | string | 提示内容 |
| text2 | string | 选项1内容 |
| text3 | string | 选项2内容 |
| text4 | string | 选项3内容 |
| value1 | int | 选项1对应변수值 |
| value2 | int | 选项2对应변수值 |
| value3 | int | 选项3对应변수值 |
| variable | object | 바인딩변수，구조：{data, secondvalue, value, varname} |

**variable구조**：

| 필드 | 유형 | 설명 |
|------|------|------|
| data | float | 데이터值 |
| secondvalue | int | 회要值 |
| value | int | 변수值 |
| varname | string | 변수名 |

**요청예시**：
```json
{
    "logout": false,
    "type": 208,
    "userParamInt": 0,
    "userParamString": "",
    "count": 3,
    "text1": "AAAAA",
    "text2": "1",
    "text3": "2",
    "text4": "3",
    "value1": 1111,
    "value2": 2222,
    "value3": 3333,
    "variable": {
        "data": 0.0,
        "secondvalue": 0,
        "value": 1,
        "varname": "I001"
    }
}
```

## 스레드상태

**설명**：查看현재所실행的스레드프로그램的상태，정지等于1，일시정지等于2，실행等于3

**명령어名**：PTHREAD_STATE

**type值**：209

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| logout | bool | 주석 처리 여부：true-작업 파일에서 실행되지 않음，false-未被주석실행 가능 |
| userParamInt | int | 2차 개발客户自定义매개변수（目前없음의미） |
| userParamString | string | 2차 개발客户自定义매개변수（目前없음의미） |
| jobName | string | 当选取后台유형时选择的프로그램名 |
| localOrGlobal | int | 프로그램유형：0-로컬后台，1-전역后台，2-主프로그램 |
| m_VariableValue | object | 存入변수，구조：{data, secondvalue, value, varname} |

**m_VariableValue구조**：

| 필드 | 유형 | 설명 |
|------|------|------|
| data | float | 데이터值 |
| secondvalue | int | 회要值 |
| value | int | 변수值 |
| varname | string | 변수名 |

**요청예시**：
```json
{
    "logout": false,
    "type": 209,
    "userParamInt": 0,
    "userParamString": "",
    "jobName": "SFDFF",
    "localOrGlobal": 0,
    "m_VariableValue": {
        "data": 0.0,
        "secondvalue": 0,
        "value": 1,
        "varname": "I001"
    }
}
```
