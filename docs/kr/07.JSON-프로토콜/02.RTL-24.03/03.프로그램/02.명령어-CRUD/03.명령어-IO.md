# 명령어-输入输出类

---

## DIN IO输入

**설명**：输入是相对于컨트롤러而言，由外部到컨트롤러。IO输入的명령어。

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| logout | bool | 주석 처리 여부；true：작업 파일에서 실행되지 않음；false：주석 처리되지 않음, 실행 가능 |
| type | int | DIN在枚举数列type中为17 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| ParaGroupNum | object | 输入组号；int유형；取值범위和一共多少组有关，IO板值*16/几路输入 |
| ParaGroupValue | object | 포트值存入；변수유형，全局或局部的bool或int型변수 |
| groupType | string | din유형；1路输入：IN#，4路输入：IGH#，8路输入：IG# |

**ParaGroupNum 结构**：

```json
{"data":2.0,"secondvalue":0,"value":0,"varname":""}
```

**ParaGroupValue 结构**：

```json
{"data":0.0,"secondvalue":0,"value":1,"varname":"I001"}
```

**예시**：

```json
{
    "logout": false,
    "type": 17,
    "userParamInt": 0,
    "userParamString": "",
    "ParaGroupNum": {"data": 2.0, "secondvalue": 0, "value": 0, "varname": ""},
    "ParaGroupValue": {"data": 0.0, "secondvalue": 0, "value": 1, "varname": "I001"},
    "groupType": "IGH#"
}
```

## DOUT IO输出

**설명**：将IO板上对应的IO포트置高或置低。IO输出的명령어。

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| logout | bool | 주석 처리 여부；true：작업 파일에서 실행되지 않음；false：주석 처리되지 않음, 실행 가능 |
| type | int | DOUT在枚举数列type中为16 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| ParaGroupNum | object | 输入组号；int유형；取值범위和一共多少组有关，IO板值*16/几路输出 |
| ParaGroupValue | object | 포트值输出；변수유형，全局或局部的bool或int型변수 |
| groupType | string | dout유형；1路输出：OT#，4路输出：OGH#，8路输出：OG# |
| errorHanding | int | 1：计时结束停止，0：输出值保持 |
| ParaGroupTime | object | 时间；double유형；범위[0,9999] |

**ParaGroupNum 结构**：

```json
{"data":2.0,"secondvalue":0,"value":0,"varname":""}
```

**ParaGroupValue 结构**：

```json
{"data":0.0,"secondvalue":0,"value":1,"varname":"I001"}
```

**ParaGroupTime 结构**：

```json
{"data":3.0,"secondvalue":0,"value":0,"varname":""}
```

**예시**：

```json
{
    "logout": false,
    "type": 16,
    "userParamInt": 0,
    "userParamString": "",
    "ParaGroupNum": {"data": 2.0, "secondvalue": 0, "value": 0, "varname": ""},
    "ParaGroupValue": {"data": 0.0, "secondvalue": 0, "value": 1, "varname": "I001"},
    "errorHanding": 1,
    "groupType": "OT#",
    "ParaGroupTime": {"data": 3.0, "secondvalue": 0, "value": 0, "varname": ""}
}
```

## PULSEOUT 脉冲输出

**설명**：按照设定的脉冲频率与个数，在R1 PWMIO板上的DB9端子的引脚4（PWM+）上进行输出。

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| logout | bool | 주석 처리 여부；true：작업 파일에서 실행되지 않음；false：주석 처리되지 않음, 실행 가능 |
| type | int | PULSEOUT在枚举数列type中为22 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| rate | int | 频率；int유형；범위[1,1000000] |
| sum | int | 个数；int유형；범위大于零 |

**예시**：

```json
{
    "logout": false,
    "type": 22,
    "userParamInt": 0,
    "userParamString": "",
    "rate": 10,
    "sum": 100
}
```

| ParaGroupValue | object | 포트值输出；변수유형，全局或局部的bool或int型변수 |
| groupType | string | dout유형；1路输出：OT#，4路输出：OGH#，8路输出：OG# |

**ParaGroupNum 结构**：
