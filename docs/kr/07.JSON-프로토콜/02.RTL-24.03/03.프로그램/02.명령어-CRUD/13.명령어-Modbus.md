# 명령어-MODBUS类

---

## MODBUS_OPEN - 打开Modbus연결

### 설명

打开Modbus통신연결

### 매개변수 목록

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| ID | int | 是 | Modbus主站프로세스号，取值범위[1,9] |
| logout | bool | 否 | 주석 처리 여부。true：在작업파일中将不执行该명령어；false：未주석 처리됨，可执行 |
| modType | string | 是 | 主站/从站。"Master"：主站；"Slave"：从站 |
| type | int | 是 | MODBUS_OPEN在枚举数列type中为210 |
| userParamInt | int | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

### 예시

```json
{
  "ID": 5,
  "logout": false,
  "modType": "Master",
  "type": 210,
  "userParamInt": 0,
  "userParamString": ""
}
```

## MODBUS_CONNECTION_STATUS - 获取Modbus연결상태

### 설명

将Modbus통신연결的상태存入변수，연결成功该변수值被赋值为1，연결失败选择的该변수被赋值为0

### 매개변수 목록

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| ID | int | 是 | Modbus主站프로세스号，取值범위[1,9] |
| logout | bool | 否 | 주석 처리 여부。true：在작업파일中将不执行该명령어；false：未주석 처리됨，可执行 |
| modType | string | 是 | 主站/从站。"Master"：主站；"Slave"：从站 |
| type | int | 是 | 获取Modbus연결상태在枚举数列type中为212 |
| userParamInt | int | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

### 예시

```json
{
  "ID": 1,
  "logout": false,
  "modType": "Master",
  "type": 212,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {"data": 0.0, "secondvalue": 1, "value": 6, "varname": "GB[GI001]"}
}
```

## MODBUS_READ - Modbus读상태

### 설명

读取Modbus中对应주소的数值。将获取到的상태存入到변수（bool유형변수或绑定변수，全局或局部）。

**주소유형설명：**

| 주소유형 | 설명 |
|----------|------|
| 3x | 读取Modbus中对应주소的值 |
| 4x | 同3x |
| 3x-bit | 读取所选주소中所存值得任一bit的상태 |
| 4x-bit | 同3x-bit |
| 0x | 可读可写的设备유형，相当于조작PLC的输出点 |
| 1x | 只读设备유형，相当于조작PLC的输入点 |

**주소유형为3X-bit时从站주소个数범위是[0,16]**

**주소유형为3X-bit，存到변수时候的17位为符号位，0表示正数，1表示负数**

**读取字符时会将读取到的所有值存入到一个변수，它跟整型、浮点型、布尔型변수不一样，不会根据설정的首주소，读从站주소个数顺延存入其它변수**

### 매개변수 목록

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| ID | int | 是 | Modbus主站프로세스号，取值범위[1,9] |
| addressNumber | object | 是 | 读从站주소个数。手填或者int유형변수或绑定변수，全局或局部。读取的주소个数，如果설정的读从站주소个数为3，执行读명령어后会从Modbus首주소开始读取3个주소（包含首주소一共读3个） |
| addressType | string | 是 | 주소유형。可选取值：3x，4x，3x-bit，4x-bit，0x，1x |
| dataType | string | 是 | 데이터유형。当데이터存放的首个변수选I或GI时，从站的데이터유형可选short和ushort（分别对应短整型和无符号短整型，前者범위为[-32768,32767]，后者的범위为[0,65535]）。데이터存放首个변수选I或GI데이터유형时需要和从站的데이터유형一致，否则可能导致데이터出错 |
| dataVariable | object | 是 | 데이터存放的首个변수。所有변수或绑定변수都可。将读取到的매개변수存放到변수，如果选择的首个변수为I001，读取的주소个数为3，执行读명령어会把매개변수值依次存入到I001、I002、I003 |
| firstAddress | object | 是 | 从站寄存器首주소。手填或int유형변수或绑定변수，全局或局部。根据명령어매개변수里面설정的首주소读取매개변수值，如果매개변수里面설정的首주소是5，在执行读명령어后会从Modbus第5个주소开始读 |
| logout | bool | 否 | 주석 처리 여부。true：在작업파일中将不执行该명령어；false：未주석 처리됨，可执行 |
| modType | string | 是 | 主站/从站。"Master"：主站；"Slave"：从站 |
| type | int | 是 | MODBUS_READ在枚举数列type中为213 |
| userParamInt | int | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

### 예시

```json
{
  "ID": 3,
  "addressNumber": {"data": 0.0, "secondvalue": 1, "value": 4, "varname": "GI[I001]"},
  "addressType": "4x-bit",
  "dataType": "short",
  "dataVariable": {"data": 0.0, "secondvalue": 1, "value": 9, "varname": "GS[GI001]"},
  "firstAddress": {"data": 0.0, "secondvalue": 1, "value": 4, "varname": "GI[GI001]"},
  "logout": false,
  "modType": "Master",
  "type": 213,
  "userParamInt": 0,
  "userParamString": ""
}
```

## MODBUS_WRITE - Modbus写조작

### 설명

该명령어用于通过Modbus将변수写入从站寄存器中对应위치的주소中

**주소유형설명：**

| 주소유형 | 설명 |
|----------|------|
| 4x | 将변수里的值写入주소码 |
| 4x-bit | 将변수组成16位二进制码写入주소码中 |
| 0x | 将변수值写入주소码，只有"0"，"1"两种상태，변수有值的话写入1，변수没值写入0 |

### 매개변수 목록

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| ID | int | 是 | Modbus主站프로세스号，取值범위[1,9] |
| addressNumber | object | 是 | 读从站주소个数。手填或者int유형변수或绑定변수，全局或局部。读取的주소个数，如果설정的读从站주소个数为3，执行读명령어后会从Modbus首주소开始读取3个주소（包含首주소一共读3个） |
| addressType | string | 是 | 주소유형。可选取值：4x，4x-bit，0x |
| dataVariable | object | 是 | 데이터存放的首个변수，写入的변수유형。手填或者所有변수或绑定변수都可 |
| firstAddress | object | 是 | 从站寄存器首주소。手填或int유형변수或绑定변수，全局或局部。开始写入的第一个주소，如果首주소为5，执行写명령어会从modbus第5个주소开始写入 |
| logout | bool | 否 | 주석 처리 여부。true：在작업파일中将不执行该명령어；false：未주석 처리됨，可执行 |
| modType | string | 是 | 主站/从站。"Master"：主站；"Slave"：从站 |
| type | int | 是 | MODBUS_WRITE在枚举数列type中为214 |
| userParamInt | int | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

### 예시

```json
{
  "ID": 5,
  "addressNumber": {"data": 0.0, "secondvalue": 0, "value": 1, "varname": "I001"},
  "addressType": "0x",
  "dataVariable": {"data": 0.0, "secondvalue": 1, "value": 4, "varname": "GI[GI001]"},
  "firstAddress": {"data": 0.0, "secondvalue": 1, "value": 1, "varname": "I[GI001]"},
  "logout": false,
  "modType": "Master",
  "type": 214,
  "userParamInt": 0,
  "userParamString": ""
}
```
