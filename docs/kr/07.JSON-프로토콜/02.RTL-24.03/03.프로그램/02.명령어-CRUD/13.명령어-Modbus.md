## MODBUS_OPEN - 열기Modbus연결

### 설명

열기Modbus통신연결

### 매개변수 목록

| 매개변수이름 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| ID | int | 예 | Modbus마스터프로세스번호，가져오기값을범위[1,9] |
| logout | bool | 아니오 | 주석 처리 여부。true：에서작업파일에 복사합니다값을아님실행해당명령어；false：미주석 처리됨，실행 가능 |
| modType | string | 예 | 마스터/슬레이브。"Master"：마스터；"Slave"：슬레이브 |
| type | int | 예 | MODBUS_OPEN에서열거형수열type에 복사합니다이(가)210 |
| userParamInt | int | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

### 예시

```json
{
  "ID": 5,
  "logout": false,
  "modType": "Master",
  "type": 210,
  "userParamInt": 0,
  "userParamString": ""
}
```

## MODBUS_READ - Modbus읽기상태

### 설명

읽기Modbus에 복사합니다해당주소값。값을가져오기상태에 저장합니다변수（bool유형변수또는绑설정변수，전역또는로컬）。

**주소유형설명：**

| 주소유형 | 설명 |
|----------|------|
| 3x | 읽기Modbus에 복사합니다해당주소값 |
| 4x | 同3x |
| 3x-bit | 읽기所선택 시주소에 복사합니다所저장값을得任1bit상태 |
| 4x-bit | 同3x-bit |
| 0x | 가능읽기쓰기 가능장치유형，즉조작PLC출력포인트 |
| 1x | 읽기 전용장치유형，즉조작PLC입력포인트 |

**주소유형이(가)3X-bit일 때슬레이브주소번째수범위예[0,16]**

**주소유형이(가)3X-bit，저장변수일 때候17비트이(가)符번호비트，0의미양수，1의미음수**

**읽기문자일 때하면값을읽기모든값을에 저장합니다1번째변수，이는정수형、부동소수점형、불리언형변수달리，설정값에 따라설정首주소，읽기슬레이브주소번째수자동으로 이어지지 않습니다다른 변수에 저장변수**

### 매개변수 목록

| 매개변수이름 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| ID | int | 예 | Modbus마스터프로세스번호，가져오기값을범위[1,9] |
| addressNumber | object | 예 | 읽기슬레이브주소번째수。手채우기또는者int유형변수또는绑설정변수，전역또는로컬。읽기주소번째수，만일설정읽기슬레이브주소번째수이(가)3，실행읽기명령어뒤하면부터Modbus首주소시작읽기3번째주소（포함首주소1共읽기3번째） |
| addressType | string | 예 | 주소유형。선택가져오기값을：3x，4x，3x-bit，4x-bit，0x，1x |
| dataType | string | 예 | 데이터유형。때데이터저장首번째변수선택 시I또는GI일 때，슬레이브데이터유형선택short와/과ushort（각각短정수형와/과없음符번호短정수형，전자범위이(가)[-32768,32767]，후자범위이(가)[0,65535]）。데이터저장首번째변수선택 시I또는GI데이터유형일 때필요와/과슬레이브데이터유형1致，아니오则가능가능내보내기致데이터출력오류 |
| dataVariable | object | 예 | 데이터저장首번째변수。모든변수또는绑설정변수사용 가능。값을읽기매개변수저장변수，선택한首번째변수이(가)I001，읽기주소번째수이(가)3，실행읽기명령어하면매개변수값을依회에 저장합니다I001、I002、I003 |
| firstAddress | object | 예 | 슬레이브레지스터首주소。手채우기또는int유형변수또는绑설정변수，전역또는로컬。에 따라명령어매개변수里面설정首주소읽기매개변수값을，만일매개변수里面설정首주소예5，에서실행읽기명령어뒤하면부터Modbus제5번째주소시작읽기 |
| logout | bool | 아니오 | 주석 처리 여부。true：에서작업파일에 복사합니다값을아님실행해당명령어；false：미주석 처리됨，실행 가능 |
| modType | string | 예 | 마스터/슬레이브。"Master"：마스터；"Slave"：슬레이브 |
| type | int | 예 | MODBUS_READ에서열거형수열type에 복사합니다이(가)213 |
| userParamInt | int | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

### 예시

```json
{
  "ID": 3,
  "addressNumber": {"data": 0.0, "secondvalue": 1, "value": 4, "varname": "GI[I001]"},
  "addressType": "4x-bit",
  "dataType": "short",
  "dataVariable": {"data": 0.0, "secondvalue": 1, "value": 9, "varname": "GS[GI001]"},
  "firstAddress": {"data": 0.0, "secondvalue": 1, "value": 4, "varname": "GI[GI001]"},
  "logout": false,
  "modType": "Master",
  "type": 213,
  "userParamInt": 0,
  "userParamString": ""
}
```

## MODBUS_WRITE - Modbus쓰기조작

### 설명

해당명령어사용하여于사용하여Modbus값을변수쓰기슬레이브레지스터에 복사합니다해당위치주소에 복사합니다

**주소유형설명：**

| 주소유형 | 설명 |
|----------|------|
| 4x | 값을변수里값쓰기주소적재 |
| 4x-bit | 값을변수취합하여16비트이진 코드쓰기주소적재에 복사합니다 |
| 0x | 값을변수값을쓰기주소적재，有"0"，"1"两종류상태，변수값이 있으면쓰기1，변수값이 없으면쓰기0 |

### 매개변수 목록

| 매개변수이름 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| ID | int | 예 | Modbus마스터프로세스번호，가져오기값을범위[1,9] |
| addressNumber | object | 예 | 읽기슬레이브주소번째수。手채우기또는者int유형변수또는绑설정변수，전역또는로컬。읽기주소번째수，만일설정읽기슬레이브주소번째수이(가)3，실행읽기명령어뒤하면부터Modbus首주소시작읽기3번째주소（포함首주소1共읽기3번째） |
| addressType | string | 예 | 주소유형。선택가져오기값을：4x，4x-bit，0x |
| dataVariable | object | 예 | 데이터저장首번째변수，쓰기변수유형。手채우기또는者모든변수또는绑설정변수사용 가능 |
| firstAddress | object | 예 | 슬레이브레지스터首주소。手채우기또는int유형변수또는绑설정변수，전역또는로컬。시작쓰기제1번째주소，만일首주소이(가)5，실행쓰기명령어하면부터modbus제5번째주소시작쓰기 |
| logout | bool | 아니오 | 주석 처리 여부。true：에서작업파일에 복사합니다값을아님실행해당명령어；false：미주석 처리됨，실행 가능 |
| modType | string | 예 | 마스터/슬레이브。"Master"：마스터；"Slave"：슬레이브 |
| type | int | 예 | MODBUS_WRITE에서열거형수열type에 복사합니다이(가)214 |
| userParamInt | int | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

### 예시

```json
{
  "ID": 5,
  "addressNumber": {"data": 0.0, "secondvalue": 0, "value": 1, "varname": "I001"},
  "addressType": "0x",
  "dataVariable": {"data": 0.0, "secondvalue": 1, "value": 4, "varname": "GI[GI001]"},
  "firstAddress": {"data": 0.0, "secondvalue": 1, "value": 1, "varname": "I[GI001]"},
  "logout": false,
  "modType": "Master",
  "type": 214,
  "userParamInt": 0,
  "userParamString": ""
}
```
