# 명령어-항목件제어클래스

## 호출子프로그램 (CALL)

**설명**：호출另一번째프로그램，호출프로그램실행완료뒤반환主프로그램CALL명령어一줄계속실행。

**명령어名**：CALL

| 매개변수名 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| callInParaVarData | array | 아니오 | 입력매개변수번째数及선택；有0-12번째요소。요소형식：`{"data":0.0,"secondvalue":0,"value":2,"varname":"D001"}` |
| callOutParaVarData | array | 아니오 | 출력매개변수번째数及선택；有0-12번째요소。요소형식：`{"data":0.0,"secondvalue":0,"value":2,"varname":"D001"}` |
| jobName | string | 예 | 호출프로그램이름，不가능예본身，호출프로그램에 복사합니다须有매개변수声明 |
| logout | bool | 아니오 | 주석 처리 여부；true：에서작업파일에 복사합니다값을不실행해당명령어；false：未주석 처리됨，실행 가능 |
| type | int | 예 | CALL에서열거형数열에 복사합니다이(가)15 |
| userParamInt | int | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

**예시**：有两번째매개변수

```json
{
    "callInParaVarData": [
        {
            "data": 0.0,
            "secondvalue": 0,
            "value": 2,
            "varname": "D001"
        },
        {
            "data": 0.0,
            "secondvalue": 0,
            "value": 2,
            "varname": "D002"
        }
    ],
    "callOutParaVarData": [
        {
            "data": 0.0,
            "secondvalue": 0,
            "value": 5,
            "varname": "GD003"
        },
        {
            "data": 0.0,
            "secondvalue": 0,
            "value": 5,
            "varname": "GD004"
        }
    ],
    "jobName": "QQQQ",
    "logout": false,
    "type": 15,
    "userParamInt": 0,
    "userParamString": ""
}
```

## 종료 (RETURN)

**설명**：종료현재실행작업파일，예를 들어：子프로그램、뒤台프로그램。

**명령어名**：RETURN

| 매개변수名 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| logout | bool | 아니오 | 주석 처리 여부；true：에서작업파일에 복사합니다값을不실행해당명령어；false：未주석 처리됨，실행 가능 |
| type | int | 예 | RETURN에서열거형数열에 복사합니다이(가)251 |
| userParamInt | int | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

**예시**：

```json
{
    "logout": false,
    "type": 251,
    "userParamInt": 0,
    "userParamString": ""
}
```

## 호출Lua파일 (CALL_LUAFILE)

**설명**：매개변수설정义완료之뒤，直연결업로드Lua파일，그 후삽입호출Lua파일명령어，명령어실행종료뒤하면쓰기완료모든매개변수변수전송입력와/과출력。

**명령어名**：CALL_LUAFILE

| 매개변수名 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| logout | bool | 아니오 | 주석 처리 여부；true：에서작업파일에 복사합니다값을不실행해당명령어；false：未주석 처리됨，실행 가능 |
| type | int | 예 | CALL_LUAFILE에서열거형数열에 복사합니다이(가)218 |
| userParamInt | int | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| inParaVarData | array | 아니오 | 전송입력매개변수번째数及선택；요소번째数[0-12]。요소형식：`{"data":0.0,"secondvalue":0,"value":1,"varname":"I001"}` |
| luafileName | string | 예 | 호출lua파일 |
| outParaVarData | array | 아니오 | 출력매개변수번째数及선택；요소번째数[0-12]。요소형식：`{"data":0.0,"secondvalue":0,"value":1,"varname":"I001"}` |
| textEditType | bool | 아니오 | 文본컴파일유형；true：文본유형，false：语句유형 |

**예시**：

```json
{
    "logout": false,
    "type": 218,
    "userParamInt": 0,
    "userParamString": "",
    "inParaVarData": [
        {
            "data": 0.0,
            "secondvalue": 0,
            "value": 1,
            "varname": "I001"
        },
        {
            "data": 2.0,
            "secondvalue": 0,
            "value": 0,
            "varname": ""
        }
    ],
    "luafileName": "demo.lua",
    "outParaVarData": [
        {
            "data": 3.0,
            "secondvalue": 0,
            "value": 0,
            "varname": ""
        },
        {
            "data": 6.0,
            "secondvalue": 0,
            "value": 0,
            "varname": ""
        }
    ],
    "textEditType": true
}
```

## 만일 (IF)

**설명**：사용하여判해제항목件여부완료立来실행一대상명령어，判해제항목件不满足일 때，실행ELSEIF语句。IF와/과ENDIF구성에 대해사용，不가능单独삭제또는변경수정，하면내보내기致프로그램실행출력오류。ELSE IF불가脱离IF单独사용。ELSE불가脱离IF单独사용。

**명령어名**：IF

| 매개변수名 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| logout | bool | 아니오 | 주석 처리 여부；true：에서작업파일에 복사합니다값을不실행해당명령어；false：未주석 처리됨，실행 가능 |
| type | int | 예 | IF에서열거형数열에 복사합니다이(가)9 |
| userParamInt | int | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| ifconditionArrayFINAL | object | 예 | 의미if判해제；{}이내이(가)if判해제데이터 |

**ifconditionArrayFINAL 구조**：

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| conditionGroup | array | 요소번째数의미有多少번째判해제 |
| logic | array | if判해제및if判해제之사이닫기系，按배열이내요소순서대로；1：또는，0：및 |
| logicGroup | array | conditionGroup노드요소判해제항목件；1：또는，0：및，null의미해당레이어没有判해제 |

**conditionGroup 요소구조**：

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| logicType | int | if判해제항목件比较방식；1：==，2：<，3：>，4：<=，5：>=，6：!= |
| paraGroupOne | object | 변수1，형식：`{"data":0.0,"secondvalue":0,"value":1,"varname":"I002"}` |
| paraGroupTwo | object | 변수2，형식：`{"data":0.0,"secondvalue":0,"value":1,"varname":"I002"}` |

**예시**：

```json
{
    "logout": false,
    "type": 9,
    "userParamInt": 0,
    "userParamString": "",
    "ifconditionArrayFINAL": {
        "conditionGroup": [
            [
                {
                    "logicType": 2,
                    "paraGroupOne": {
                        "data": 0.0,
                        "secondvalue": 0,
                        "value": 1,
                        "varname": "I001"
                    },
                    "paraGroupTwo": {
                        "data": 0.0,
                        "secondvalue": 0,
                        "value": 0,
                        "varname": ""
                    }
                }
            ],
            [
                {
                    "logicType": 1,
                    "paraGroupOne": {
                        "data": 0.0,
                        "secondvalue": 0,
                        "value": 1,
                        "varname": "I001"
                    },
                    "paraGroupTwo": {
                        "data": 0.0,
                        "secondvalue": 0,
                        "value": 0,
                        "varname": ""
                    }
                }
            ],
            [
                {
                    "logicType": 6,
                    "paraGroupOne": {
                        "data": 0.0,
                        "secondvalue": 0,
                        "value": 1,
                        "varname": "I002"
                    },
                    "paraGroupTwo": {
                        "data": 0.0,
                        "secondvalue": 0,

    }
}
```

## 대기 (WAIT)

**설명**：항목件완료立之이전프로그램상태대기상태，항목件완료立뒤계속실행대기명령어뒤面명령어。

**명령어名**：WAIT

| 매개변수名 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| logout | bool | 아니오 | 주석 처리 여부；true：에서작업파일에 복사합니다값을不실행해당명령어；false：未주석 처리됨，실행 가능 |
| type | int | 예 | WAIT에서열거형数열에 복사합니다이(가)8 |
| userParamInt | int | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| m_filterTime | double | 아니오 | 滤波일 때사이，입력信번호필요满足일 때사이；[0,9999]s。滤波일 때사이필요小于대기일 때사이 |
| paraTime | object | 아니오 | 대기일 때사이；double유형；[0.01,9999]s |
| para_wait_result_ | object | 아니오 | 대기결과，满足항목件이(가)true；bool유형 |

## WHILE 循环 (WHILE)

**설명**：判해제항목件이(가)真예反복원실행循环语句里面명령어。삽입WHILE명령어同일 때하면同일 때삽입ENDWHILE명령어。若필요삭제WHILE명령어请同일 때删掉其해당ENDWHILE명령어，아니오则하면내보내기致프로그램없음法실행。때프로그램열기头이(가)WHILE且最뒤一样명령어이(가)ENDWHILE일 때，请에서프로그램열기头또는结尾삽입一항목TIMER（지연）명령어。아니오则때WHILE명령어항목件不满足일 때하면내보내기致프로그램陷입력死机。때WHILE이내部명령어没有모션클래스명령어또는에서某종류경우가능가능하면陷입력死循环일 때，请에서WHILE및ENDWHILE사이삽입一항목TIMER（지연）명령어，아니오则때WHILE명령어항목件满足일 때가능가능하면내보내기致프로그램陷입력死机。때WHILE명령어항목件满足일 때，하면循环실행WHILE및ENDWHILE两항목명령어之사이명령어。에서실행WHILE명령어之이전若判해제항목件不满足，에서실행WHILE명령어일 때하면直연결跳转ENDWHILE명령어而不실행WHILE및ENDWHILE之사이명령어；若에서실행WHILE및ENDWHILE之사이명령어过程에 복사합니다，判해제항목件변경완료不满足，하면계속실행，直실행ENDWHILE줄，不再循环而예계속실행ENDWHILE아래명령어。

**명령어名**：WAIT_POS
