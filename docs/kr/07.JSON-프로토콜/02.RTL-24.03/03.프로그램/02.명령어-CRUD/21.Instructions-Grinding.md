# 명령어-그라인딩프로세스클래스

---

## 棱边그라인딩

**명령어名：POLISH_EDGE**

**설명:** 棱边그라인딩

**type：** 109（在열거형序열中）

### 매개변수 목록

| 매개변수 | 유형 | 설명 | 값 범위범위 |
|--------|------|------|----------|
| ParaACC | double/int | 가속도 | [1,100]% |
| ParaAHEAD | double/int | 提前실행，与 aheadtype 有关 | [1,100]% |
| ParaDEC | double/int | 감속도 | [1,100]% |
| ParaPL | int | 평활 | [0,5] |
| ParaTIME | double/int | 提前실행时间 | - |
| ParaV | double/int | 속도 | [1,1000] mm/s |
| aheadtype | int | 提前유형 | 0:提前时间；1:提前距离；2:提前进도 |
| logout | bool | 주석 처리 여부 | true:不실행该명령어；false:실행 가능 |
| polish | int | 그라인딩회数 | [1,99] |
| polishAngle | double | 角도 | [-180,180] 도 |
| polishID | int | 그라인딩프로세스号 | [1,99] |
| posidname | string | 포인트 | P或GP바인딩변수 |
| posidtype | int | 포인트유형 | 0:포인트변수；1:포인트바인딩변수 |
| positionId | string | 포인트 | 当 posidtype 为 0 时使用 |
| userParamInt | int | 2차 개발自定义，目前없음의미 | - |
| userParamString | string | 2차 개발自定义，目前없음의미 | - |

### JSON 예시

```json
{
  "ParaACC": {"data": 0.0, "secondvalue": 0, "value": 1, "varname": "I002"},
  "ParaAHEAD": {"data": 0.0, "secondvalue": 1, "value": 1, "varname": "GD[I001]"},
  "ParaDEC": {"data": 0.0, "secondvalue": 0, "value": 1, "varname": "GI008"},
  "ParaPL": {"data": 0.0, "secondvalue": 1, "value": 1, "varname": "I[GI001]"},
  "ParaV": {"data": 0.0, "m_vUnit": 1, "secondvalue": 0, "value": 1, "varname": "I001"},
  "aheadtype": 2,
  "logout": false,
  "polish": 89,
  "polishAngle": 70.0,
  "polishID": 30,
  "posidname": "GP$GI001",
  "posidtype": 1,
  "positionId": "",
  "type": 109,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 그라인딩계속

**명령어名：POLISH_CONTINUE**

**설명:** 그라인딩계속

**type：** 131（在열거형序열中）

### 매개변수 목록

| 매개변수 | 유형 | 설명 | 값 범위범위 |
|--------|------|------|----------|
| angle | double | 角도 | [-180,180] 도 |
| logout | bool | 주석 처리 여부 | true:不실행该명령어；false:실행 가능 |
| polishID | int | 프로세스号 | [1,99] |
| time | double | 提前실행时间 | - |
| times | int | 그라인딩회数 | [1,99] |
| userParamInt | int | 2차 개발自定义，目前없음의미 | - |
| userParamString | string | 2차 개발自定义，目前없음의미 | - |

### JSON 예시

```json
{
  "angle": 99,
  "logout": false,
  "polishID": 21,
  "time": 999,
  "times": 19,
  "type": 131,
  "userParamInt": 0,
  "userParamString": ""
}
```
