# 명령어-스프레이 프로세스클래스

---

## 스프레이시작

**명령어名：SPRAY_ON**

**type值：122**

### 매개변수 설명

| 매개변수名 | 유형 | 取值범위 | 설명 |
|--------|------|----------|------|
| anaID | int | [0, 99] | 模拟量组号 |
| atom | double | [0, 10] | 雾化模拟量(V)，模拟量组号为0时수정 |
| colorID | int | - | 颜色ID |
| flow | double | [0, 10] | 流量模拟量(V)，模拟量组号为0时수정 |
| gunID | int | [1, 2] | 枪 |
| logout | bool | true/false | 주석 처리 여부：true작업 파일에서 실행되지 않음，false未被注释실행 가능 |
| point | int | - | 포인트 |
| sector | double | [0, 10] | 扇形模拟量(V)，模拟量组号为0时수정 |
| seqID | int | [1, 99] | 时序组号 |
| traID | int | - | 궤적ID |
| type | int | 122 | SPRAY_ON在열거형序열type中为122 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |

### JSON예시

```json
{
  "anaID": 0,
  "atom": 6.0,
  "colorID": 1,
  "flow": 8.0,
  "gunID": 2,
  "logout": false,
  "point": 1,
  "sector": 7.0,
  "seqID": 39,
  "traID": 1,
  "type": 122,
  "userParamInt": 0,
  "userParamString": ""
}
```
