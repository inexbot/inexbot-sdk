# 명령어-변수 유형

---

## 1. 대입 (SET)

### 설명

정의된 정수형, 부동소수점형, 불리언형 및 문자열 변수에 값을 대입합니다. 문자열 추가 명령어입니다.

### 매개변수 목록

| 매개변수 | 유형 | 설명 |
|--------|------|------|
| logout | bool | 주석 처리 여부; true: 작업 파일에서 실행되지 않음; false: 주석 처리되지 않음, 실행 가능 |
| type | int | 명령어 유형, 값은 41 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| variable | object | 변수; int 유형 또는 bool 유형 또는 double 유형 또는 string 유형 변수, 전역 또는 로컬 |
| variableValue | object | 변수 값; 수동 입력 값 또는 bool 유형 또는 int 유형 또는 double 유형 변수 |

### variable 구조

| 필드 | 유형 | 설명 |
|------|------|------|
| varname | string | 변수명 |
| value | int | 변수 값 |
| data | double | 부동소수점 데이터 |
| secondvalue | int | 예비 값 |

### variableValue 구조

| 필드 | 유형 | 설명 |
|------|------|------|
| varname | string | 변수명 |
| value | int | 변수 값 |
| data | double | 부동소수점 데이터 |
| secondvalue | int | 예비 값 |

### 예시

```json
{
  "logout": false,
  "type": 41,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {
    "data": 0.0,
    "secondvalue": 0,
    "value": 8,
    "varname": "I001"
  },
  "variableValue": {
    "data": 0.0,
    "secondvalue": 0,
    "value": 8,
    "varname": "GD009"
  }
}
```

## 2. 파일 쓰기 (FORCESET)

### 설명

캐시된 데이터를 하드디스크에 저장합니다. 프로그램 실행 중 모든 계산, 대입 작업은 캐시의 값을 변경하며 시스템 파일에 저장되지 않습니다. 메모리의 전역 숫자 변수를 파일에 강제로 기록하려면 FORCESET 명령어를 사용할 수 있습니다.

### 매개변수 목록

| 매개변수 | 유형 | 설명 |
|--------|------|------|
| logout | bool | 주석 처리 여부; true: 작업 파일에서 실행되지 않음; false: 주석 처리되지 않음, 실행 가능 |
| type | int | 명령어 유형, 값은 42 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| variable | object | 변수명; int 유형 또는 bool 유형 또는 double 유형, 전역 |

### variable 구조

| 필드 | 유형 | 설명 |
|------|------|------|
| varname | string | 변수명 |
| value | int | 변수 값 |
| data | double | 부동소수점 데이터 |
| secondvalue | int | 예비 값 |

### 예시

```json
{
  "logout": false,
  "type": 42,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {
    "data": 0.0,
    "secondvalue": 0,
    "value": 8,
    "varname": "GI001"
  }
}
```
