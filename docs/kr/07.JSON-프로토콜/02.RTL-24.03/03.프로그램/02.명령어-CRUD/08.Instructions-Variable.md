# 指令-变量类

---

## 1. 赋值 (SET)

### 설명

给定义的정수형、부동소수점형、불리언형和문자열变量赋值。문자열追加指令。

### 매개변수 목록

| 매개변수 | 유형 | 설명 |
|--------|------|------|
| logout | bool | 주석 처리 여부；true：작업 파일에서 실행되지 않음；false：주석 처리되지 않음, 실행 가능 |
| type | int | 명령어 유형，值为 41 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| variable | object | 变量；int类型或bool类型或double类型或string类型变量，全局或局部 |
| variableValue | object | 변수 값；수동 입력 값或bool类型或int类型或double类型变量 |

### variable 结构

| 字段 | 类型 | 说明 |
|------|------|------|
| varname | string | 변수명 |
| value | int | 변수 값 |
| data | double | 부동소수점 데이터 |
| secondvalue | int | 예비 값 |

### variableValue 结构

| 字段 | 类型 | 说明 |
|------|------|------|
| varname | string | 변수명 |
| value | int | 변수 값 |
| data | double | 부동소수점 데이터 |
| secondvalue | int | 예비 값 |

### 예시

```json
{
  "logout": false,
  "type": 41,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {
    "data": 0.0,
    "secondvalue": 0,
    "value": 8,
    "varname": "I001"
  },
  "variableValue": {
    "data": 0.0,
    "secondvalue": 0,
    "value": 8,
    "varname": "GD009"
  }
}
```

## 2. 写入文件 (FORCESET)

### 설명

캐시된 데이터를 하드디스크에 저장。在程序运行过程中，모든 계산, 할당 작업은 캐시의 값을 변경하며 시스템 파일에 저장되지 않습니다。메모리의 전역 숫자 변수를 파일에 강제로 쓰려면 다음을 사용할 수 있습니다 FORCESET 指令。

### 매개변수 목록

| 매개변수 | 유형 | 설명 |
|--------|------|------|
| logout | bool | 주석 처리 여부；true：작업 파일에서 실행되지 않음；false：주석 처리되지 않음, 실행 가능 |
| type | int | 명령어 유형，值为 42 |
| userParamInt | int | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| variable | object | 변수명；int类型或bool类型或double类型，全局 |

### variable 结构

| 字段 | 类型 | 说明 |
|------|------|------|
| varname | string | 변수명 |
| value | int | 변수 값 |
| data | double | 부동소수점 데이터 |
| secondvalue | int | 예비 값 |

### 예시

```json
{
  "logout": false,
  "type": 42,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {
    "data": 0.0,
    "secondvalue": 0,
    "value": 8,
    "varname": "GI001"
  }
}
```
