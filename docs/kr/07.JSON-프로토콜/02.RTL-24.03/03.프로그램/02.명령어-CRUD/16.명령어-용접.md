# 명령어-용접 제어

## 용접 시작 (ARCON)

**설명:** 본 명령어는 아크 발생 동작을 실행합니다. 파일 번호를 선택한 후에는 매개변수가 기본적으로 용접 프로세스 인터페이스에서 설정·저장된 매개변수에 따라 실행됩니다. 임시 프로세스 매개변수 스위치가 켜져 있을 경우, 명령어의 매개변수가 우선 적용됩니다. 용접 시작은 용접 종료와 함께 사용해야 합니다.

**명령어명:** ARCON

**type:** 23

| 매개변수명 | 유형 |값 범위 범위 | 설명 |
|--------|------|----------|------|
| anamorphismFlag | int | [0,1] | 아크 발생 그라데이션 플래그 사용 여부; 0: 사용 안 함, 1: 사용 |
| arconUseAbled | int | [0,2] | 아크 발생 매개변수 플래그 사용 여부; 0: 임시 매개변수 사용 안 함, 1: 사용, 2: 사용 안 함 |
| generatingOfArcAgainFlag | int | [0,1] | 재아크 발생 플래그; 0: 사용 안 함, 1: 사용 |
| generatingOfArcCurrent | double | [0-1000]A | 아크 발생 전류; 99999.0: 임시 변수 미사용, 프로세스 구성 매개변수 사용 |
| generatingOfArcTime | double | [0-5]s | 아크 발생 시간; 99999.0: 임시 변수 미사용, 프로세스 구성 매개변수 사용 |
| generatingOfArcVoltage | double | [-1000,1000]V | 아크 발생 전압; 99999.0: 임시 변수 미사용, 프로세스 구성 매개변수 사용 |
| logout | bool | true/false | 주석 처리 여부; true: 작업파일에서 본 명령어를 실행하지 않음 |
| startUpAgainFlag | int | [0,1] | 재시작 플래그 사용 여부; 0: 사용 안 함, 1: 사용 |
| useTemp | int | [0,1] | 임시 프로세스 매개변수 플래그 사용 여부; 0: 사용 안 함, 1: 사용 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| variable | object | [1-99] | 사용할 용접 매개변수 프로세스 번호; int 직접 입력 또는 int 유형 변수(전역 또는 로컬) |
| weldCurrent | double | [0-1000]A | 용접 전류 |
| weldVoltage | double | - | 용접 전압 |

**JSON 예시:**

```json
{
  "anamorphismFlag": 1,
  "arconUseAbled": 1,
  "generatingOfArcAgainFlag": 1,
  "generatingOfArcCurrent": 3.0,
  "generatingOfArcTime": 2.0,
  "generatingOfArcVoltage": 36.0,
  "logout": false,
  "startUpAgainFlag": 1,
  "type": 23,
  "useTemp": 1,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {"data": 0.0, "secondvalue": 1, "value": 4, "varname": "GI[I001]"},
  "weldCurrent": 89.0,
  "weldVoltage": 78.0
}
```

## ARCOFF 용접 종료

**설명:** 본 명령어는 아크 소호 동작을 실행합니다. 파일 번호를 선택한 후에는 매개변수가 기본적으로 용접 프로세스 인터페이스에서 설정·저장된 매개변수에 따라 실행됩니다. 임시 프로세스 매개변수 스위치가 켜져 있을 경우, 명령어의 매개변수가 우선 적용됩니다. 용접 시작은 용접 종료와 함께 사용해야 합니다.

**명령어명:** ARCOFF

**type:** 24

| 매개변수명 | 유형 |값 범위 범위 | 설명 |
|--------|------|----------|------|
| arcoffUseAbled | int | [0,2] | 아크 소호 매개변수 플래그 사용 여부; 0: 임시 매개변수 사용 안 함, 1: 사용, 2: 사용 안 함 |
| logout | bool | true/false | 주석 처리 여부; true: 작업파일에서 본 명령어를 실행하지 않음 |
| quenchingAnamorphismFlag | int | [0,2] | 아크 소호 그라데이션 사용 여부; 0: 임시 매개변수 사용 안 함, 1: 사용, 2: 사용 안 함 |
| quenchingOfArcCurrent | double | [0,1000]A | 아크 소호 전류 |
| quenchingOfArcTime | double | [0,5]s | 아크 소호 시간 |
| quenchingOfArcVoltage | double | [-1000,1000]V | 아크 소호 전압 |
| useTemp | int | [0,1] | 임시 프로세스 매개변수 플래그 사용 여부; 0: 사용 안 함, 1: 임시 매개변수 사용 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| variable | object | - | 사용할 용접 매개변수 프로세스 번호 |

**JSON 예시:**

```json
{
  "arcoffUseAbled": 1,
  "logout": false,
  "quenchingAnamorphismFlag": 1,
  "quenchingOfArcCurrent": 2.0,
  "quenchingOfArcTime": 3.0,
  "quenchingOfArcVoltage": 1.0,
  "type": 24,
  "useTemp": 1,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {"data": 1.0, "secondvalue": 0, "value": 0, "varname": ""}
}
```

## WVON위빙 시작

**설명:** 본 명령어가 실행되면위빙(weave welding)이 시작됩니다. 본 명령어를 실행하기 전에 먼저 용접 시작 ARCON 명령어를 실행해 주십시오. 임시 매개변수를 사용하지 않을 경우, data는 99999로 설정해야 합니다.

**명령어명:** WVON

**type:** 26

| 매개변수명 | 유형 |값 범위 범위 | 설명 |
|--------|------|----------|------|
| logout | bool | true/false | 주석 처리 여부; true: 작업파일에서 본 명령어를 실행하지 않음 |
| useTemp | int | [0,1] | 임시 프로세스 매개변수 플래그 사용 여부; 0: 사용 안 함, 1: 임시 매개변수 사용 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| variable | object | [1-99] | 사용할 위빙 매개변수 프로세스 번호; int 직접 입력 또는 int 유형 변수(전역 또는 로컬) |
| weldLeftDwellTime | object | [0,15]s | 좌측 정지 시간; double 유형 수동 입력 값, double 또는 int 유형 변수(전역 또는 로컬) |
| weldRightDwellTime | object | [0,15]s | 우측 정지 시간; double 유형 수동 입력 값, double 또는 int 유형 변수(전역 또는 로컬) |
| weldSwingAmplitude | object | (0,50]mm | 摆动 진폭; double 유형 수동 입력 값, double 또는 int 유형 변수(전역 또는 로컬) |
| weldSwingFrequency | object | (0,20]HZ | 摆动 주파수; double 유형 수동 입력 값, double 또는 int 유형 변수(전역 또는 로컬) |

**JSON 예시:**

```json
{
  "logout": false,
  "type": 26,
  "useTemp": 1,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {"data": 0.0, "secondvalue": 0, "value": 1, "varname": "I006"},
  "weldLeftDwellTime": {"data": 0.0, "secondvalue": 1, "value": 1, "varname": "GD[I001]"},
  "weldRightDwellTime": {"data": 0.0, "secondvalue": 1, "value": 1, "varname": "D[I001]"},
  "weldSwingAmplitude": {"data": 0.0, "secondvalue": 1, "value": 1, "varname": "I[GI001]"},
  "weldSwingFrequency": {"data": 0.0, "secondvalue": 0, "value": 1, "varname": "GD001"}
}
```
