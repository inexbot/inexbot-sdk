# 명령어-용접제어类

## 용접开始

**설명：** 该명령어可执行起弧조작。选择好파일标号以后，매개변수默认是根据용접프로세스界面设定保存的매개변수来실행。打开使用临时프로세스매개변수开关，以명령어中매개변수为准。용접开始要和용접结束一起使用。

**명령어名：** ARCON

**type：** 23

| 매개변수名 | 유형 | 取值범위 | 설명 |
|--------|------|----------|------|
| anamorphismFlag | int | [0,1] | 使用起弧渐变标志；0：不使用，1：使用 |
| arconUseAbled | int | [0,2] | 使用起弧매개변수标志；0：不使用临时매개변수，1：使用，2：不使用 |
| generatingOfArcAgainFlag | int | [0,1] | 再起弧标志；0：不使用，1：使用 |
| generatingOfArcCurrent | double | [0-1000]A | 起弧电流；99999.0：不使用临时변수，使用프로세스구성매개변수 |
| generatingOfArcTime | double | [0-5]s | 起弧时间；99999.0：不使用临时변수，使用프로세스구성매개변수 |
| generatingOfArcVoltage | double | [-1000,1000]V | 起弧电压；99999.0：不使用临时변수，使用프로세스구성매개변수 |
| logout | bool | true/false | 주석 처리 여부；true：在작업파일中将不执行该명령어 |
| startUpAgainFlag | int | [0,1] | 使用再启动标志；0：不使用，1：使用 |
| useTemp | int | [0,1] | 是否使用临时프로세스매개변수标志；0：不使用，1：使用 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| variable | object | [1-99] | 要使用的용접매개변수프로세스号；手填int或int유형변수，全局或局部 |
| weldCurrent | double | [0-1000]A | 용접电流 |
| weldVoltage | double | - | 용접电压 |

**JSON예시：**

```json
{
  "anamorphismFlag": 1,
  "arconUseAbled": 1,
  "generatingOfArcAgainFlag": 1,
  "generatingOfArcCurrent": 3.0,
  "generatingOfArcTime": 2.0,
  "generatingOfArcVoltage": 36.0,
  "logout": false,
  "startUpAgainFlag": 1,
  "type": 23,
  "useTemp": 1,
  "userParamInt": 0,
  "userParamString": "",
  "variable": {"data": 0.0, "secondvalue": 1, "value": 4, "varname": "GI[I001]"},
  "weldCurrent": 89.0,
  "weldVoltage": 78.0
}
```

## ARCSET 용접설정

**설명：** 该명령어可以설정용접时的电流电压。

|--------|------|----------|------|
| arcsetRamp | int | [0,1] | 渐变方式；0：不使用，1：时间 |
| logout | bool | true/false | 주석 처리 여부；true：在작업파일中将不执行该명령어 |
| useTemp | int | [0,1] | 是否使用临时프로세스매개변수标志；0：不使用，1：使用临时매개변수 |

| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |

| weldSetRampTime | object | [0,100000]ms | 渐变时间；int유형手填值或변수，全局或局部 |

## TIGWELDON 鱼鳞焊开始

**설명：** 执行该명령어开始鱼鳞焊轨迹，执行该명령어前请先실행용접开始ARCON명령어。

**명령어名：** TIGWELDON

**type：** 66

| 매개변수名 | 유형 | 取值범위 | 설명 |
|--------|------|----------|------|
| headTailType | int | [0,3] | 首位段；0：首尾段용접优化空走距离，1：首尾段空走优化空走距离，2：首段용접，3：首段空走 |
| l1 | double | [0.01,9999] | 용접距离；typeOFTIG节点值为2时需要此节点 |
| l1speed | double | [1,200]% | 용접속도；typeOFTIG节点值为2时需要此节点 |
| l2 | double | [0.01,9999]mm | 空走距离 |
| l2speed | double | [1,1000]% | 空走속도 |
| logout | bool | true/false | 주석 처리 여부；true：在작업파일中将不执行该명령어 |
| timeOFTIG | double | [0.01,6]s | 点焊时间；typeOFTIG节点值为1时需要此节点 |
| typeOFTIG | int | [0,1] | 매개변수유형；1:点焊时间，2：용접距离 |
| userParamInt | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
