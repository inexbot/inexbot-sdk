# 명령어-通用모션

## 1. MOVJ 点到点모션

**설명**：点到点모션为관절모션，单位为%。

### 매개변수 설명

| 매개변수名 | 유형 | 取值범위 | 单位 | 설명 |
|--------|------|----------|------|------|
| ParaACC | double | [1,100] | % | 加속도 |
| ParaDEC | double | [1,100] | % | 减속도 |
| ParaV | double | [1,100] | % | 속도，m_vUnit必须为2 |
| ParaPL | double | [0,5] | - | 平滑변수 |
| ParaTIME | double | 自然数 | - | 提前执行时间 |
| type | int | 1 | - | 명령어유형，1表示MOVJ명령어 |
| logout | bool | - | - | 주석 처리 여부，true表示不执行 |
| userParamInt | int | - | - | 2차 개발自定义매개변수 |
| userParamString | string | - | - | 2차 개발自定义매개변수 |
| posidname | string | - | - | 绑定변수时的点位名 |
| positionId | string | - | - | 非绑定변수时的点位名 |
| posidtype | int | 0或1 | - | 点位유형：1=绑定변수，0=非绑定변수 |

### RobotPos 点位信息

## 2. MOVL 直线모션

**설명**：直线모션속도单位为mm。

### 매개변수 설명

| 매개변수名 | 유형 | 取值범위 | 单位 | 설명 |
|--------|------|----------|------|------|
| ParaACC | double | [1,100] | % | 加속도 |
| ParaDEC | double | [1,100] | % | 减속도 |
| ParaV | double | [1,1000] | mm/s | 속도，m_vUnit必须为1 |
| ParaPL | double | [0,5] | - | 平滑변수 |
| ParaTIME | double | 自然数 | - | 提前执行时间 |
| type | int | 2 | - | 명령어유형，2表示MOVL명령어 |
| logout | bool | - | - | 주석 처리 여부，true表示不执行 |
| userParamInt | int | - | - | 2차 개발自定义매개변수 |
| userParamString | string | - | - | 2차 개발自定义매개변수 |
| posidname | string | - | - | 绑定변수时的点位名 |
| positionId | string | - | - | 非绑定변수时的点位名 |
| posidtype | int | 0或1 | - | 点位유형：1=绑定변수，0=非绑定변수 |

### m_vUnit 속도单位설명

| 值 | 单位 |
|----|------|
| 0 | cm/s |
| 1 | mm/s |
| 2 | 百分比 |

> 直线모션时只能使用 1 (mm/s)。

### RobotPos 点位信息

## 3. MOVC 圆弧모션

**설명**：使用时需要一条直线或点到点명령어和两条圆弧명령어，这三个点确定一个圆弧。

### 매개변수 설명

| 매개변수名 | 유형 | 取值범위 | 单位 | 설명 |
|--------|------|----------|------|------|
| ParaACC | double | [1,100] | % | 加속도 |
| ParaDEC | double | [1,100] | % | 减속도 |
| ParaV | double | [1,1000] | mm/s | 속도，m_vUnit必须为1 |
| ParaPL | double | [0,5] | - | 平滑변수 |
| ParaTIME | double | 自然数 | - | 提前执行时间 |
| ParaSPIN | double | 0或1 | - | 姿态매개변수：0=示教姿态，1=姿态倾角不变 |
| type | int | 3 | - | 명령어유형，3表示MOVC명령어 |
| logout | bool | - | - | 주석 처리 여부，true表示不执行 |
| userParamInt | int | - | - | 2차 개발自定义매개변수 |
| userParamString | string | - | - | 2차 개발自定义매개변수 |
| posidname | string | - | - | 绑定변수时的点位名 |
| positionId | string | - | - | 非绑定변수时的点位名 |
| posidtype | int | 0或1 | - | 点位유형：1=绑定변수，0=非绑定변수 |

## 4. MOVCA 插入整圆

**설명**：整圆모션需要一条点到点或者直线모션，加两个整圆모션명령어。

| ParaACC | double | [1,100] | % | 加속도 |

| ParaSPIN | double | 0,1,2 | - | 姿态매개변수：0=姿态不变，1=六轴不转，2=六轴旋转 |

| logout | bool | - | - | 주석 처리 여부，true表示不执行 |
| userParamInt | int | - | - | 2차 개발自定义매개변수 |
| userParamString | string | - | - | 2차 개발自定义매개변수 |
| posidname | string | - | - | 绑定변수时的点位名 |
| positionId | string | - | - | 非绑定변수时的点位名 |
| posidtype | int | 0或1 | - | 点位유형：1=绑定변수，0=非绑定변수 |

## 5. MOVS 插入曲线

**설명**：曲线모션为S型모션轨迹，需要一条点到点或直线명령어，再加四条曲线모션명령어。

| userParamInt | int | - | - | 2차 개발自定义매개변수 |
| userParamString | string | - | - | 2차 개발自定义매개변수 |
| posidname | string | - | - | 绑定변수时的点位名 |
