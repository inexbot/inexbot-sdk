# 명령어-좌표계클래스

---

## 1. 전환도구手 (SWITCHTOOL)

**설명:** 전환도구手，0为없음도구手

**명령어名：** `SWITCHTOOL`

### 매개변수 설명

| 매개변수名 | 유형 | 取值범위 | 설명 |
|--------|------|----------|------|
| `_variable` | object | - | 변수유형；手填int유형或int유형，全局或局部 |
| `_variable.data` | float | - | 数据值 |
| `_variable.secondvalue` | int | - | 副值 |
| `_variable.value` | int | [0, 999] | 도구手编号 |
| `_variable.varname` | string | - | 변수명 |
| `logout` | bool | - | 주석 처리 여부：true=작업 파일에서 실행되지 않음；false=주석 처리되지 않음, 실행 가능 |
| `type` | int | 48 | 전환도구手在열거형数열type中为48 |
| `userParamInt` | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| `userParamString` | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |

### 예시

```json
{
  "_variable": {
    "data": 0.0,
    "secondvalue": 0,
    "value": 4,
    "varname": "GI009"
  },
  "logout": false,
  "type": 48,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 3. 사용자 좌표转换 (USERCOORD_TRANS)

**설명:** 将B、C사용자 좌표系叠加（×），结果置入A사용자 좌표系

**명령어名：** `USERCOORD_TRANS`

### 매개변수 설명

| 매개변수名 | 유형 | 取值범위 | 설명 |
|--------|------|----------|------|
| `CoordA` | int | [1, 999] | 사용자 좌표A；结果存入该사용자 좌표系，这里예사용자 좌표系序号 |
| `CoordB` | int | [1, 999] | 사용자 좌표B |
| `CoordC` | int | [1, 999] | 사용자 좌표C |
| `logout` | bool | - | 주석 처리 여부：true=작업 파일에서 실행되지 않음；false=주석 처리되지 않음, 실행 가능 |
| `type` | int | 117 | 사용자 좌표转换在열거형数열type中为117 |
| `userParamInt` | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| `userParamString` | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |

### 예시

```json
{
  "CoordA": 8,
  "CoordB": 18,
  "CoordC": 28,
  "logout": false,
  "type": 117,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 4. 전환외부 축 (SWITCHSYNC)

**설명:** 在프로그램실행中전환현재使用的외부 축

**명령어名：** `SWITCHSYNC`

### 매개변수 설명

| 매개변수名 | 유형 | 取值범위 | 설명 |
|--------|------|----------|------|
| `logout` | bool | - | 주석 처리 여부：true=작업 파일에서 실행되지 않음；false=주석 처리되지 않음, 실행 가능 |
| `syncgroup` | int | [1, 3] | 要전환到的외부 축的组号 |
| `type` | int | 50 | 전환외부 축在열거형数열type中为50 |
| `userParamInt` | int | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| `userParamString` | string | - | 2차 개발에서 사용자 정의, 현재 의미 없음 |

### 예시

```json
{
  "logout": false,
  "syncgroup": 2,
  "type": 50,
  "userParamInt": 0,
  "userParamString": ""
}
```
