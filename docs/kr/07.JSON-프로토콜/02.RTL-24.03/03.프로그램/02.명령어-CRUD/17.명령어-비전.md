# 명령어-비전명령类

## 비전开始

**설명**：执行비전开始명령어后컨트롤러与相机연결

**명령어名**：VISION_RUN

| 매개변수 | 유형 | 必填 | 설명 |
|------|------|------|------|
| ID | int | 是 | 프로세스号，取值범위[1,99] |
| logout | bool | 是 | 주석 처리 여부：true=在작업파일中将不执行该명령어；false=未주석 처리됨，可执行 |
| type | int | 是 | VISION_RUN在枚举数列type中为79 |
| userParamInt | int | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

**JSON예시**：
```json
{
  "ID": 14,
  "logout": false,
  "type": 79,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 飞拍开始

**설명**：로봇도구모션经过拍照点时，快速触发相机拍照，在拍照过程中物体或相机处于모션상태

**명령어名**：ONFLY_INIT

| 매개변수 | 유형 | 必填 | 설명 |
|------|------|------|------|
| Delay | object | 是 | 记录좌표延时，全局或局部double유형手填值或int/double유형변수或绑定변수，取值범위[0,1000] |
| axisNum | object | 否 | 轴，string유형手填值或int유형全局或局部변수或绑定변수 |
| _posIdType | int | 是 | phonePos节点是点位변수还是绑定点位변수：0=点位변수；1=绑定点位변수 |
| IOPort | object | 否 | IO포트，int유형或绑定변수유형，取值범위[1,1000] |
| IOValue | object | 否 | 포트值，bool유형手填值或변수或绑定변수 |
| cameraTriggerDelayTime | object | 是 | 相机触发延时，double유형手填值或int유형，取值범위[0,1000] |
| craftID | object | 是 | 프로세스号，int유형，取值범위[1,99] |
| logout | bool | 是 | 주석 처리 여부：true=在작업파일中将不执行该명령어；false=未주석 처리됨，可执行 |
| triggerAngleSpacing | object | 否 | 触发角度间隔，double유형手填值或int/double유형全局或局部변수或绑定변수，取值범위[-360,360] |
| triggerCount | object | 否 | 触发次数，int유형手填值或全局或局部绑定변수或변수，取值범위[0,10]；不使用时`{"data":0.0,"secondvalue":0,"value":1,"varname":"[-]"}` |
| triggerStartPos | object | 否 | 触发起始위치，double유형手填值或全局或局部int/double유형변수或绑定변수 |
| phonePos | string | 是 | 拍照点，string유형，全局或局部P点位或E点位변수或绑定변수 |
| radius | object | 是 | 拍照点범위半径，取值범위[0.001,1000] |
| type | int | 是 | ONFLY_INIT在枚举数列type中为239 |
| userParamInt | int | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| visionCraftID | object | 否 | 비전프로세스号 |

## 飞拍计算도구

**설명**：飞拍计算도구作用于机械臂夹持工件时可能出现偏心위치，工件有旋转角。通过计算获取并切换新的도구之后，工件中心放置到放料위치点，不偏心无旋转

**명령어名**：ONFLY_CALC_TOOL

| 매개변수 | 유형 | 必填 | 설명 |
|------|------|------|------|
| craftID | object | 是 | 프로세스号，int유형手填值或全局或局部변수或绑定변수，取值범위[1,99] |
| grabTool | object | 是 | 抓取도구，int유형手填值或全局或局部변수或绑定변수，取值범위[1,999] |
| logout | bool | 是 | 주석 처리 여부：true=在작업파일中将不执行该명령어；false=未주석 처리됨，可执行 |
| newTool | object | 是 | 新도구，int유형手填值或全局或局部변수或绑定변수，取值범위[1,999] |
| type | int | 是 | ONFLY_CALC_TOOL在枚举数列type中为242 |
| userParamInt | int | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 否 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

**JSON예시**：
```json
{
  "craftID": {"data": 0.0, "secondvalue": 1, "value": 4, "varname": "GI[GI001]"},
  "grabTool": {"data": 0.0, "secondvalue": 1, "value": 1, "varname": "I[GI001]"},
  "logout": false,
  "newTool": {"data": 0.0, "secondvalue": 1, "value": 4, "varname": "GI[I006]"},
  "type": 242,
  "userParamInt": 0,
  "userParamString": ""
}
```
