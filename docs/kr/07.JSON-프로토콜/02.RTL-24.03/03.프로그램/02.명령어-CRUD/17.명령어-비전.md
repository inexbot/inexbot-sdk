# 명령어-비전명령클래스

## 비전시작

**설명**：실행비전시작명령어后컨트롤러与相机연결

**명령어名**：VISION_RUN

| 매개변수 | 유형 | 필수 | 설명 |
|------|------|------|------|
| ID | int | 예 | 프로세스号，取值범위[1,99] |
| logout | bool | 예 | 주석 처리 여부：true=在작업파일中将不실행该명령어；false=未주석 처리됨，실행 가능 |
| type | int | 예 | VISION_RUN在열거형数열type中为79 |
| userParamInt | int | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

**JSON예시**：
```json
{
  "ID": 14,
  "logout": false,
  "type": 79,
  "userParamInt": 0,
  "userParamString": ""
}
```

## 가져오기비전위치번째数

**설명**：记录相机전송的포인트번째数，将포인트번째数存入到选择的변수

**명령어名**：VISION_POSNUM

| 매개변수 | 유형 | 필수 | 설명 |
|------|------|------|------|
| ID | int | 예 | 프로세스号，取值범위[1,99] |
| M_STRPOSNUM | object | 예 | 全局값변수，全局int변수或绑定변수，取值범위[1,990] |
| logout | bool | 예 | 주석 처리 여부：true=在작업파일中将不실행该명령어；false=未주석 처리됨，실행 가능 |
| type | int | 예 | VISION_POSNUM在열거형数열type中为81 |
| userParamInt | int | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

**M_STRPOSNUM结构**：
```json
{"data": 0.0, "secondvalue": 1, "value": 4, "varname": "GI[GI001]"}
```

**JSON예시**：
```json
{
  "ID": 31,
  "M_STRPOSNUM": {"data": 0.0, "secondvalue": 1, "value": 4, "varname": "GI[GI001]"},
  "logout": false,
  "type": 81,
  "userParamInt": 0,
  "userParamString": ""
}
```

| userParamString | string | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

**dataVariable结构**：
```json
{"data": 0.0, "secondvalue": 1, "value": 9, "varname": "GS[I001]"}
```

```json
{
```

## 飞拍시작

**설명**：로봇도구모션经过拍照点时，快速触发相机拍照，在拍照过程中物体或相机处于모션상태

**명령어名**：ONFLY_INIT

| 매개변수 | 유형 | 필수 | 설명 |
|------|------|------|------|
| Delay | object | 예 | 记录좌표延时，全局或局部double유형수동 입력 값或int/double유형변수或绑定변수，取值범위[0,1000] |
| axisNum | object | 아니오 | 축，string유형수동 입력 값或int유형全局或局部변수或绑定변수 |
| _posIdType | int | 예 | phonePos节点예포인트변수还예绑定포인트변수：0=포인트변수；1=绑定포인트변수 |
| IOPort | object | 아니오 | IO포트，int유형或绑定변수유형，取值범위[1,1000] |
| IOValue | object | 아니오 | 포트值，bool유형수동 입력 값或변수或绑定변수 |
| cameraTriggerDelayTime | object | 예 | 相机触发延时，double유형수동 입력 값或int유형，取值범위[0,1000] |
| craftID | object | 예 | 프로세스号，int유형，取值범위[1,99] |
| logout | bool | 예 | 주석 처리 여부：true=在작업파일中将不실행该명령어；false=未주석 처리됨，실행 가능 |
| triggerAngleSpacing | object | 아니오 | 触发角도间隔，double유형수동 입력 값或int/double유형全局或局部변수或绑定변수，取值범위[-360,360] |
| triggerCount | object | 아니오 | 触发회数，int유형수동 입력 값或全局或局部绑定변수或변수，取值범위[0,10]；不使用时`{"data":0.0,"secondvalue":0,"value":1,"varname":"[-]"}` |
| triggerStartPos | object | 아니오 | 触发起始위치，double유형수동 입력 값或全局或局部int/double유형변수或绑定변수 |
| phonePos | string | 예 | 拍照点，string유형，全局或局部P포인트或E포인트변수或绑定변수 |
| radius | object | 예 | 拍照点범위半径，取值범위[0.001,1000] |
| type | int | 예 | ONFLY_INIT在열거형数열type中为239 |
| userParamInt | int | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| visionCraftID | object | 아니오 | 비전프로세스号 |

## 飞拍计算도구

**설명**：飞拍计算도구作用于机械臂夹持공작물时可能出现偏心위치，공작물有旋转角。通过计算가져오기并전환新的도구之后，공작물中心放置到放料위치点，不偏心없음旋转

| 매개변수 | 유형 | 필수 | 설명 |
|------|------|------|------|

| logout | bool | 예 | 주석 처리 여부：true=在작업파일中将不실행该명령어；false=未주석 처리됨，실행 가능 |
| newTool | object | 예 | 新도구，int유형수동 입력 값或全局或局部변수或绑定변수，取值범위[1,999] |
| type | int | 예 | ONFLY_CALC_TOOL在열거형数열type中为242 |
| userParamInt | int | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |
| userParamString | string | 아니오 | 2차 개발에서 사용자 정의, 현재 의미 없음 |

**JSON예시**：
