# 프로그램실행제어

## 프로그램실행总数清零

**명령어：** `0x5600`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 选择로봇，取值범위 [1, 4] |

```json
{
  "robot": 1
}
```

## 실행프로그램

对프로그램进行조작。

### 전송开始실행프로그램

**명령어：** `0x5601`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要조작的로봇 번호，取值범위 [1, 4] |
| jobname | string | 프로그램名 |
| suffixname | string | 프로그램名后缀名称，取值：.JBR 主프로그램，.JBP 后台局部프로그램，.JBPG 后台全局프로그램 |
| stepRunSum | int | 一共多少步（可选） |
| callerName | string | 访问者名字（可选） |
| stepRun | bool | 是否单步실행（可选） |
| index | int | 索引（可选） |
| noFileRun | bool | 无파일실행（可选） |
| notJudgeSafePos | bool | 不判断安全위치（可选） |
| line | int | 프로그램开始실행的行号 |
| globalStart | bool | 是否为多机모드 |
| continueRun | int | 是否为继续실행：0 不继续실행，1 继续실행 |
| startOver | int | 是否为重头실행：0 继续실행，1 重头실행 |

```json
{
  "robot": 1,
  "jobname": "TTT",
  "suffixname": ".JBR",
  "stepRunSum": 2,
  "callerName": "tttt",
  "stepRun": false,
  "index": 0,
  "noFileRun": false,
  "notJudgeSafePos": false,
  "line": 1,
  "globalStart": false,
  "continueRun": 0,
  "startOver": 0
}
```

> 注：可选필드不使用无影响

## 프로그램执行상태설정

호스트 컴퓨터需要改变当前컨트롤러프로그램执行모드时전송。

**명령어：** `0x5603`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 选择로봇，取值범위 [1, 4] |
| type | int | 프로그램执行상태：0 单步실행，1 暂停，2 停止但不下电，5 停止但不下电（停止优先使用） |

```json
{
  "robot": 1,
  "type": 5
}
```

> 注意：对于单步，没有全部로봇；对于暂停和停止，只有多机모드下有全部로봇，此时将 "robot" 节点설정成 0

## 追加모션모드

追加모션모드是在不使用프로그램的条件下실행프로그램。

使用方法：
1. 打开追加모션모드
2. 追加모션모드插入명령어向量
3. 追加모션모드설정

### 打开追加모션모드

**명령어：** `0x5640`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要조작的로봇 번호，取值범위 [1, 4] |
| open | bool | 是否开启追加모션모드，true 打开，false 关闭 |

```json
{
  "robot": 1,
  "open": true
}
```

### 조회追加모션모드

**명령어：** `0x5641`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要조작的로봇 번호，取值범위 [1, 4] |

```json
{
  "robot": 1
}
```

**컨트롤러回复**

**명령어：** `0x5642`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要조작的로봇 번호，取值범위 [1, 4] |
| open | bool | 是否开启追加모션모드，true 打开，false 关闭 |

```json
{
  "robot": 1,
  "open": true
}
```

### 追加모션插入명령어向量

**명령어：** `0x5643`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要조작的로봇 번호，取值범위 [1, 4] |
| isContinue | bool | true 只수신并存储 pos，false 수신 pos 并使得로봇执行所有 pos |
| data | array | 명령어 json 列表，格式为 cmd->toJson() |

```json
{
  "data": [
    { "cmd": "toJson()" },
    { "cmd": "toJson()" },
    { "cmd": "toJson()" }
  ],
  "isContinue": false,
  "robot": 1
}
```

> 명령어详情请参阅명령어页面

## 自启动

自启动프로그램，설정后开机自启动。

### 설정自启动프로그램

**명령어：** `0x5647`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要조회的로봇 번호，取值범위 [1, 4] |
| enable | bool | 是否使能，true 使能，false 不使用 |
| count | int | 실행次数 |
| speed | int | 실행속도 |
| jobname | string | 프로그램名字 |

```json
{
  "robot": 1,
  "enable": false,
  "count": 1,
  "speed": 15,
  "jobname": "Q1"
}
```

### 조회自启动프로그램

**명령어：** `0x5648`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要조회的로봇 번호，取值범위 [1, 4] |

```json
{
  "robot": 1
}
```

**컨트롤러回复**

**명령어：** `0x5649`

| 필드 | 유형 | 설명 |
|------|------|------|
| robot | int | 要조회的로봇 번호，取值범위 [1, 4] |
| enable | bool | 是否使能 |
| count | int | 실행次数 |
| speed | int | 실행속도 |
| jobname | string | 프로그램名字 |

```json
{
  "robot": 1,
  "enable": false,
  "count": 1,
  "speed": 15,
  "jobname": "Q1"
}
```

> 需补充试실행和单步相关内容
