# 팔레타이징프로세스

## 1 그리퍼

### 1.1 호스트 컴퓨터설정그리퍼매개변수

**설명：** 그리퍼설정예설정팔레타이징过程에 복사합니다사용하여도구。两번째吸盘分别가져오기料（一번째吸盘吸완료뒤再전환另一번째吸盘가져오기料），那么필요설정완료两번째그리퍼；两번째吸盘同일 때가져오기料，则설정완료一번째그리퍼；만일两번째吸盘分别放料（一번째吸盘放완료뒤전환另一번째吸盘放料），이일 때필요설정两번째그리퍼；两번째吸盘同일 때放料，则설정완료一번째그리퍼。

**명령어：** `0x8000`

| 매개변수名 | 유형 | 설명 | 가져오기값을범위 |
|--------|------|------|----------|
| robot | int | 현재로봇 | [1,4] |
| craftID | int | 프로세스번호 | - |
| pallet.gripperNum | int | 그리퍼개수 | [1,4] |
| pallet.gripper | array | 그리퍼도구번호배열，제i번째数代表그리퍼i도구번호 | - |

```json
{
    "robot": 1,
    "craftID": 1,
    "pallet": {
        "gripperNum": 2,
        "gripper": [1, 3, 4, 1]
    }
}
```
