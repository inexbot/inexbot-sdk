# 팔레타이징프로세스

## 1 抓手

### 1.1 호스트 컴퓨터설정抓手매개변수

**설명：** 抓手설정就是설정팔레타이징过程中用到的도구。两个吸盘分别取料（一个吸盘吸完后再切换到另一个吸盘取料），那么要설정成两个抓手；两个吸盘同时取料，则설정成一个抓手；如果两个吸盘分别放料（一个吸盘放完后切换到另一个吸盘放料），此时需要설정两个抓手；两个吸盘同时放料，则설정成一个抓手。

**명령어：** `0x8000`

| 매개변수名 | 유형 | 설명 | 取值범위 |
|--------|------|------|----------|
| robot | int | 当前로봇 | [1,4] |
| craftID | int | 프로세스号 | - |
| pallet.gripperNum | int | 抓手数目 | [1,4] |
| pallet.gripper | array | 抓手도구号数组，第i个数代表抓手i的도구号 | - |

```json
{
    "robot": 1,
    "craftID": 1,
    "pallet": {
        "gripperNum": 2,
        "gripper": [1, 3, 4, 1]
    }
}
```

| pallet.realPos | array | 工件点，double유형，6个元素 | - |
| pallet.shiftPos | array | 辅助点，double유형，6个元素 | - |

```json
{
    "robot": 1,
    "craftID": 1,
    "pallet": {
        "floorNum": 1,
        "enterPos": [9999.0, 9999.0, 9999.0, 9999.0, 9999.0, 9999.0],
```
