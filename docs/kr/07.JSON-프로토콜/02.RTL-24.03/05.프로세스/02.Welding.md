# 용접프로세스

용접的相关통신协议主要包含용접装备、电流电压、용접매개변수、摆焊매개변수、용접IO포트、多层多道용접、相贯线，关于용접프로세스的详细介绍请参考용접프로세스手册。

---

## 1. 용접装备

용접装备설정包含基础기능、微调整、防碰撞、再시작/再启弧和其他模块。

### 1.1 호스트 컴퓨터설정용접装备

**명령어：** `0x8300`

설명：용접过程中，通过发生断弧、防碰撞、切모드정지等方式断弧时，此기능可在断弧处接着용접。自动再시작、半自动再시작时要确保断弧处和回退目标위치之间没有障碍物。

#### basic_functions（基础기능）

| 매개변수名 | 유형 | 설명 | 取值범위 |
|--------|------|------|----------|
| arc_detection_time | double | 电弧检测时间 | [0,5]s |
| arc_depletion_detection_time | double | 电弧耗尽检测时间 | [0,5]s |
| arc_detection_ensure_time | double | 电弧检测确认时间 | [0,5]s |
| pre_give_gas_time | double | 提前送气时间 | [0,5]s |
| close_time | double | 关气时间（负数为提前关气，正数为延后关气） | [-5,5]s |
| weld_interruption_detection | bool | 용접中断检测 | true：中断检测；false：不中断检测 |
| arc_fly | bool | 飞줄起弧 | true：飞줄起弧；false：不飞줄起弧 |
| air_close_mode | int | 关气方式 | 0: 延迟关气，1: 提前关气 |

#### micro_adjustment（微调整）

| 매개변수名 | 유형 | 설명 | 取值범위 |
|--------|------|------|----------|
| single_adjustment_current | double | 용접电流单회调整量 | [0,1000]A |
| single_adjustment_voltage | double | 용접电压单회调整量 | [-1000,1000]V |

#### restart（再시작/再起弧）

| 매개변수名 | 유형 | 설명 | 取值범위 |
|--------|------|------|----------|
| restart_times | int | 再시작회数 | [0,1000] |
| restart_distance | double | 再시작距离 | [0,100]mm |
| restart_speed | double | 再시작속도 | [0,100]mm/s |
| re_arc_times | int | 再起弧회数 | [1,1000] |
| restart_enable | bool | 再시작활성화 | - |
| re_arc_enable | bool | 再起弧활성화 | - |
| restart_mode | int | 再시작모드 | - |

#### antiCollision（防碰撞）

| 매개변수名 | 유형 | 설명 | 取值범위 |
|--------|------|------|----------|
| enable | bool | 防碰撞활성화 | true: 开启；false：닫기 |
| inputPort | int | 防碰撞IO포트 | - |
| inputValue | int | 防碰撞出发电平 | 0/1 |
| stopTime | double | 防碰撞快速정지时间 | [50,200]s |
| outputPort | int | 防碰撞상태출력포트 | - |
| outputValue | int | 防碰撞상태출력电平 | 0/1 |
| shieldFlag | bool | 屏蔽防碰撞 | true: 开启；false：닫기 |
| shieldTime | double | 屏蔽时间 | [0,1000000]s |

#### other（其他）

| 매개변수名 | 유형 | 설명 | 取值범위 |
|--------|------|------|----------|
| break_back_time | double | 断弧回抽时间 | [0,5]s |
| weld_finish_time | double | 용접완료回抽时间 | [0,5]s |
| break_back_enable | bool | 断弧回抽활성화 | true: 开启；false：닫기 |
| weld_finish_function | bool | 용접완료回抽기능 | true: 开启；false：닫기 |
| simulation_set_zero | bool | 灭弧模拟量置零기능 | true: 开启；false：닫기 |
| collisionButtonFlag.error_enable_switch | bool | 오류활성화开关 | - |

**예시：**


## 2. 电流电压

### 2.1 전류-전압 매칭설정

**명령어：** `0x8310`

| 매개변수名 | 유형 | 설명 | 取值범위 |
|--------|------|------|----------|
| robot | int | 현재로봇 | [1,4] |

#### current_control（电流제어）

| 매개변수名 | 유형 | 설명 | 取值범위 |
|--------|------|------|----------|
| set_current | double[] | 설정电流 | [0,10]A |
| actual_current | double[] | 实际电流 | [0,999]A |

#### voltage_control（电压제어）

| 매개변수名 | 유형 | 설명 | 取值범위 |
|--------|------|------|----------|
| set_voltage | double[] | 설정电压 | [0,10]V |
| actual_voltage | double[] | 实际电压 | [0,999]V |

```json
{
  "current_control": {
    "set_current": [1, 2, 3, 4, 5, 6, 7, 8],
    "actual_current": [1, 2, 3, 4, 5, 6, 7, 8]
  },
  "voltage_control": {
    "set_voltage": [1, 2, 3, 4, 5, 6, 7, 8],
    "actual_voltage": [1, 2, 3, 4, 5, 6, 7, 8]
  },
  "robot": 1
}
```

## 3. 용접매개변수

### 3.1 호스트 컴퓨터설정용접매개변수

**명령어：** `0x8320`

| 매개변수名 | 유형 | 설명 | 取值범위 |
|--------|------|------|----------|
| num | int | 프로세스号（从1시작，可설정1-99번째） | [1,99] |
| notes | string | 注释 | - |
| robot | int | 현재로봇 | [1,4] |

#### arcoff_parameters（收弧매개변수）

| 매개변수名 | 유형 | 설명 | 取值범위 |
|--------|------|------|----------|
| arcoff_use_param | bool | 여부使用收弧매개변수 | true：使用；false：不使用 |
| arcoff_voltage | double | 收弧电压 | [-1000,1000]V |
| arcoff_current | double | 收弧电流 | [0,1000]A |
| arcoff_time | double | 收弧时间 | [0,5]s |
| ramp_time | double | 渐变时间 | [0,100000]ms |
| ramp_enable | bool | 渐变활성화 | - |
| ramp_mode | int | 收弧渐变方式 | 0：时间渐变 |

#### weld_parameters（용접매개변수）

| 매개변수名 | 유형 | 설명 | 取值범위 |
|--------|------|------|----------|
| weld_current | double | 용접电流 | [0,1000]A |
| weld_voltage | double | 용접电压 | [-1000,1000]V |

#### arcon_parameters（起弧매개변수）

| 매개변수名 | 유형 | 설명 | 取值범위 |
|--------|------|------|----------|
| arcon_use_param | bool | 여부使用起弧매개변수 | true：使用；false：不使用 |
| arcon_current | double | 起弧电流 | [0,1000]A |
| arcon_voltage | double | 起弧电压 | [-1000,1000]V |
| arcon_time | double | 起弧时间 | [0,5]s |
| arcon_ramp_time | double | 渐变时间 | [0,1000000]ms |
| arcon_ramp_enable | bool | 起弧渐变활성화 | - |
| arcon_ramp_mode | int | 起弧渐变方式 | 0：时间渐变 |

```json
    "arcoff_time": 1.1,

    "ramp_enable": true,
    "ramp_mode": 1
  },
```

## 4. 摆焊매개변수

inexbot摆焊支持七种摆动方式：正弦摆、Z字摆、圆形摆、외부 축定点摆、L型摆、三角摆、8字摆。摆动주파수、摆动幅도、起始方向、水平偏角、竖直偏角等매개변수可调。

### 4.1 호스트 컴퓨터 전송설정摆焊매개변수

**명령어：** `0x8330`

| 매개변수名 | 유형 | 설명 | 取值범위 |
|--------|------|------|----------|
| num | int | 摆焊파일序号（从1시작） | [1,99] |
| note | string | 注释 | - |
| robot | int | 현재로봇 | [1,4] |

#### parameter（摆焊매개변수）

| 매개변수名 | 유형 | 설명 | 取值범위 |
|--------|------|------|----------|
| weaving_type | int | 摆动유형 | [0,6]，0:正弦，1:Z字形，2:圆形摆，3:외부 축定点摆，4:L型摆，5:三角摆，6:8字摆 |
| weaving_freq | int | 摆动주파수 | (0,20]Hz |
| weaving_ampli | int | 摆动幅도 | (0,50]mm |
| isMove | int | 여부이동/停留 | 0：停留，1：이동 |
| right_standing_time | int | 右停留时间（选择Z字形摆、외부 축定点摆、L型摆、三角摆时需要설정） | [0,15] |
| left_standing_time | int | 左停留时间（选择Z字形摆、외부 축定点摆、L型摆、三角摆时需要설정） | [0,15] |
| init_direction | int | 起始方向 | 0：+1，1：-1 |
| horizontal_angle | int | 水平偏角 | [-180,180] |
| vertical_angle | int | 竖直偏角 | [-180,180] |
| left_angle | int | 左仰角（选择L型摆、三角摆时需要설정） | [0,90] |
| right_angle | int | 右仰角（选择L型摆、三角摆时需要설정） | [0,90] |

```json
{
  "num": 1,
  "parameter": {
    "note": "예시",
    "weaving_type": 0,
    "weaving_freq": 20,
    "weaving_ampli": 50,
    "isMove": 0,
    "right_standing_time": 15,
    "left_standing_time": 15,
    "init_direction": 1,
```
