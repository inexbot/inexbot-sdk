# 寻位跟踪

## 跟踪

### 跟踪传感器유형

#### 설정跟踪传感器유형

- **명령字：** `0x8740`
- **方向：** 호스트 컴퓨터 전송

| 매개변수名 | 유형 | 取值범위 | 설명 |
|--------|------|----------|------|
| fileNum | int | [1,99] | 跟踪파일号 |
| robot | int | [1,4] | 로봇号 |
| sensorType | int | [0,2] | 跟踪유형：0-线激光，1-电弧，2-弧压 |

**요청예시：**

```json
{
  "fileNum": 1,
  "robot": 1,
  "sensorType": 0
}
```

#### 조회跟踪传感器유형

### 电弧补偿매개변수

#### 설정电弧跟踪左右补偿매개변수

- **명령字：** `0x8750`
- **方向：** 호스트 컴퓨터 전송

| 매개변수名 | 유형 | 取值범위 | 설명 |
|--------|------|----------|------|
| craftid | int | [1,99] | 跟踪파일号 |
| robot | int | [1,4] | 로봇号 |
| accFactor | double | [0.1,10] | 补偿加속도倍数，对应当前모션명령어加속도 |
| algorithmType | int | [0,1] | 偏差提取유형：0-均值，1-手填值 |
| beginCycleNum | int | [1,100] | 开始采样周期数 |
| differential | double | [0,1] | 微分系数 |
| integral | double | [0,1] | 积分系数 |
| maxSingleLen | double | [0,10] | 每次最大补偿量 |
| scale | double | [0,1] | 比例系数 |
| switchon | bool | true/false | 补偿开关：true-打开，false-关闭 |

**요청예시：**

```json
{
  "craftid": 1,
  "lateralCompensation":
  {
    "accFactor": 1.0,
    "algorithmType": 0,
    "beginCycleNum": 4,
    "differential": 0.0,
    "integral": 0.00050,
    "maxSingleLen": 2.0,
    "scale": 0.050,
    "switchon": true
  },
  "robot": 1
}
```
