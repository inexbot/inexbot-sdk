# 심 검출추적

## 추적

### 추적传센서유형

#### 설정추적传센서유형

- **명령字：** `0x8740`
- **방향：** 호스트 컴퓨터 전송

| 매개변수名 | 유형 | 값 범위범위 | 설명 |
|--------|------|----------|------|
| fileNum | int | [1,99] | 추적파일号 |
| robot | int | [1,4] | 로봇号 |
| sensorType | int | [0,2] | 추적유형：0-线레이저，1-아크，2-弧压 |

**요청예시：**

```json
{
  "fileNum": 1,
  "robot": 1,
  "sensorType": 0
}
```

#### 조회추적传센서유형

- **명령字：** `0x8741`

### 아크补偿매개변수

#### 설정아크추적左右补偿매개변수

- **명령字：** `0x8750`
- **방향：** 호스트 컴퓨터 전송

| 매개변수名 | 유형 | 값 범위범위 | 설명 |
|--------|------|----------|------|
| craftid | int | [1,99] | 추적파일号 |
| robot | int | [1,4] | 로봇号 |
| accFactor | double | [0.1,10] | 补偿加속도倍数，对应현재모션명령어加속도 |
| algorithmType | int | [0,1] | 편차提取유형：0-均值，1-수동 입력 값 |
| beginCycleNum | int | [1,100] | 시작采样주기数 |
| differential | double | [0,1] | 微分系数 |
| integral | double | [0,1] | 积分系数 |
| maxSingleLen | double | [0,10] | 每회최대补偿量 |
| scale | double | [0,1] | 比例系数 |
| switchon | bool | true/false | 补偿开关：true-열기，false-닫기 |

**요청예시：**

```json
{
  "craftid": 1,
  "lateralCompensation":
  {
    "accFactor": 1.0,
    "algorithmType": 0,
    "beginCycleNum": 4,
    "differential": 0.0,
    "integral": 0.00050,
    "maxSingleLen": 2.0,
    "scale": 0.050,
    "switchon": true
  },
  "robot": 1
}
```
