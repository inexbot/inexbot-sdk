# 컨베이어

## 0x8200 호스트 컴퓨터 전송，轨道传送위치설정

### 매개변수 설명

| 매개변수名 | 유형 | 설명 | 取值범위 |
|--------|------|------|----------|
| conveyorID | int | 컨베이어프로세스号 | [1,9] |
| position | object | 위치매개변수对象 | - |
| robot | int | 로봇号 | [1,4] |

### position 子매개변수

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| receLatestPos | double | 最迟수신X值 |
| trackRangeXMax | double | 跟踪最大범위 |
| trackStartXPoint | double | 跟踪开始X点 |

### 요청 예시

```json
{
  "conveyorID": 1,
  "position": {
    "receLatestPos": 0.0,
    "trackRangeXMax": 0.0,
    "trackStartXPoint": 0.0
  },
  "robot": 1
}
```

## 0x8210 호스트 컴퓨터 전송，컨베이어传送매개변수설정

### 매개변수 설명

| 매개변수名 | 유형 | 설명 | 取值범위 |
|--------|------|------|----------|
| conveyor | object | 컨베이어매개변수对象 | - |
| conveyorID | int | 컨베이어프로세스号 | [1,9] |
| robot | int | 로봇号 | [1,4] |
| track | object | 轨道매개변수对象 | - |

### conveyor 子매개변수

| 매개변수名 | 유형 | 설명 | 取值범위 |
|--------|------|------|----------|
| encoderDirection | int | 编码器方向：1正向，-1负向 | - |
| encoderResolution | double | 分辨率（线/mm） | [0,10000] |
| encoderValue | string | 编码器值 | - |
| encoder_num | int | 编码器编号 | - |
| encoder_sum | int | 编码器总数 | - |
| maxEncoderVal | string | 编码器最大值 | - |
| minEncoderVal | string | 编码器最小值 | - |
| posRecordMode | int | 위치모드：0编码器，1恒速설정 | [0,1] |
| speed | double | 编码器속도（mm/s） | - |

### track 子매개변수

| 매개변수名 | 유형 | 설명 | 单位 |
|--------|------|------|------|
| arch_height | int | 门型高度 | mm |
| arch_straight_len | int | 门型直线距离 | mm |

### 요청 예시

```json
{
  "conveyor": {
    "encoderDirection": 1,
    "encoderResolution": 44.33,
    "encoderValue": "0",
    "encoder_num": 1,
    "encoder_sum": 0,
    "maxEncoderVal": "2147483647",
    "minEncoderVal": "-2147483648",
    "posRecordMode": 0,
    "speed": 0.0
  },
  "conveyorID": 1,
  "robot": 1,
  "track": {
```

## 0x8213 컨베이어传送위치检查매개변수설정

### 매개변수 설명

| 매개변수名 | 유형 | 설명 | 取值범위 |
|--------|------|------|----------|
| conveyorID | int | 컨베이어프로세스号 | [1,9] |
| detectSrc | object | 检测源매개변수对象 | - |
| robot | int | 로봇号 | [1,4] |

### detectSrc 子매개변수

| 매개변수名 | 유형 | 설명 | 取值범위 |
|--------|------|------|----------|
| DI_capturePos | int | 绑定触发信号IO | 与当前IO数量有关 |
| filter_data_missing_triggered_latch_data_ | bool | 是否过滤未触发锁存的信号 | - |
| sensorTrg | int | 传感器触发方式：0下降沿触发，1上升沿触发 | [0,1] |
| type | int | 检测和识别유형：0传感器，1비전，2传感器+비전，3全局변수+modbus，4全局변수+비전 | [0,4] |
| visionID | int | 비전프로세스号 | [1,9] |
| vision_io_filter_type | int | 비전过滤유형：0信号触发过滤，1信号触发过滤，2无过滤 | [0,2] |
| vision_latch_encoder_value_type | int | 비전锁存编码器值유형：0비전触发锁存，1收到데이터锁存 | [0,1] |

### 요청 예시

```json
{
  "conveyorID": 1,
  "detectSrc": {
    "DI_capturePos": 0,
    "filter_data_missing_triggered_latch_data_": true,
    "sensorTrg": 0,
    "sensor_safe_trigger_interval": 1.0,
    "type": 2,
    "visionID": 1,
    "vision_io_filter_type": 0,
    "vision_latch_encoder_value_type": 1
  },
  "robot": 1
}
```
