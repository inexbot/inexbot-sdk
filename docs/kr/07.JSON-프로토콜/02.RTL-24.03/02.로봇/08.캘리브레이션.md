# 캘리브레이션

## 四点캘리브레이션

설명：로봇末梢移动到并캘리브레이션四个点，计算后可得杆长度和轴영점偏移，若认可计算结果，可将计算结果填入DH매개변수。

### 호스트 컴퓨터四点캘리브레이션计算요청

**명령어：** `0x3700`

| 매개변수名 | 유형 | 범위/설명 |
|--------|------|-----------|
| ParamOf4Points | double[] | 长度为2，L1和L2长度。两매개변수分别为로봇末梢从标记点A移动到标记点C时X的移动距离和Y的移动距离 |

```json
{
  "ParamOf4Points": [85.0, 70.0]
}
```

### 호스트 컴퓨터N号点캘리브레이션요청

**명령어：** `0x3701`

| 매개변수名 | 유형 | 범위/설명 |
|--------|------|-----------|
| pointNum | int | 범위[0,3]，캘리브레이션点编号 |
| status | int | 범위[0,1]，캘리브레이션상태。0表示取消캘리브레이션，1表示캘리브레이션 |

```json
{
  "pointNum": 0,
  "status": 1
}
```

### 컨트롤러回复호스트 컴퓨터N号点캘리브레이션위치信息

**명령어：** `0x3702`

| 매개변수名 | 유형 | 범위/설명 |
|--------|------|-----------|
| pointNum | int | 범위[0,3]，캘리브레이션点编号 |
| pos | double[] | 长度为4，위치信息列表 |

```json
{
  "pointNum": 0,
  "pos": [0.0, 0.0, 0.0, 0.0]
}
```

### 컨트롤러回复호스트 컴퓨터캘리브레이션计算结果

**명령어：** `0x3703`

| 매개변수名 | 유형 | 범위/설명 |
|--------|------|-----------|
| dtheta | double[] | 长度为2，轴영점偏移列表。值全为-1表示计算失败 |
| length | double[] | 长度为2，杆长度列表。值全为-1表示计算失败 |
| result | bool | 是否计算成功 |

```json
{
  "dtheta": [-1.0, -1.0],
  "length": [-1.0, -1.0],
  "result": false
}
```

**명령어：** `0x3705`

| 매개변수名 | 유형 | 범위/설명 |
|--------|------|-----------|
| apply | bool | 计算结果填入是否成功 |

```json

```

### 호스트 컴퓨터조회캘리브레이션信息

**명령어：** `0x3706`

| 매개변수名 | 유형 | 범위/설명 |
|--------|------|-----------|
| dtheta | double[] | 长度为2，轴영점偏移列表。null表示没有计算结果 |
| length | double[] | 长度为2，杆长度列表。null表示没有计算结果 |
| result | bool | 是否计算成功 |
| status | int[] | 点位캘리브레이션상태列表。0表示未캘리브레이션，1表示已캘리브레이션 |

```json
{
  "dtheta": [null, null],
  "length": [null, null],
  "result": false,
  "status": [0, 1, 0, 1]
}
```

## 20点캘리브레이션

설명：一种用于校准로봇的方法，通过在特定的20个点上进行캘리브레이션，以确保로봇的准确性和稳定性。

### 호스트 컴퓨터캘리브레이션计算요청

**명령어：** `0x3710`

| 매개변수名 | 유형 | 범위/설명 |
|--------|------|-----------|
| calibrationType | int | 캘리브레이션点方法。2表示2点法，12表示12点法，15表示15点法，20表示20点法有校准영점，21表示21点法无校准영점 |
| noCalZero | bool | 是否为校准영점 |
| robot | int | 범위[1,4]，当前로봇号 |
| tool | int | 범위[1,999]，当前도구编号 |

```json
{
  "calibrationType": 20,
  "noCalZero": false,
  "robot": 1,
  "tool": 1
}
```

### 호스트 컴퓨터캘리브레이션N号点요청

**명령어：** `0x3711`

| 매개변수名 | 유형 | 범위/설명 |
|--------|------|-----------|
| calibrationType | int | 캘리브레이션点方法。2表示2点法，12表示12点法，15表示15点法，20表示20点法有校准영점，21表示21点法无校准영점 |
| point | int | 캘리브레이션点编号。2点法时범위[0,1]，12点法时범위[0,11]，15点法时범위[0,14]，20点法时범위[0,19] |
| robot | int | 범위[1,4]，当前로봇号 |
| tool | int | 범위[1,999]，当前도구编号 |

```json
{
  "calibrationType": 20,
  "point": 1,
  "robot": 1,
  "tool": 1
}
```

### 호스트 컴퓨터설정地轨协同매개변수

**명령어：** `0x3739`

| 매개변수名 | 유형 | 범위/설명 |

| trackMap | int[] | 长度为3，地轨各轴方向。0表示未选择方向，1表示X方向，2表示Y轴方向，3表示Z轴方向。超过当前轴数的部分默认为0 |

```json

  "calibrateResult": true,
  "conversionRatio": [58.0, 1.0, 1.0],
  "trackMap": [1, 0, 0]
```
