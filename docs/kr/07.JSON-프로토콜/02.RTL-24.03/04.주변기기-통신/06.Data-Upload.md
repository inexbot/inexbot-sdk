# 数据업로드

## 1. 数据传输开关

### 1.1 호스트 컴퓨터 설정数据传输开关

**명령어:** `0x7330`

| 매개변수名 | 유형 | 描述 |
|--------|------|------|
| enable | bool | 数据传输开关 |
| robot | int | 현재로봇 번호 |

```json
{
  "enable": true,
  "robot": 1
}
```

## 2. 数据업로드매개변수

### 2.1 호스트 컴퓨터 설정数据업로드매개변수

**명령어:** `0x7333`

| 매개변수名 | 유형 | 범위/값 범위 | 描述 |
|--------|------|-----------|------|
| dataCollectTime | int | [0, 999999]s | 数据采集주기 |
| dataUploadTime | int | [0, 999999]s | 数据업로드주기 |
| fileSuffix | string | "csv", "txt" | 文件형식 |
| filepath | string | - | 文件경로 |
| password | string | - | 密码 |
| robot | int | [1, 4] | 로봇 번호 |
| sendInfoFileFlag | bool | - | 예아니오전송설명文档 |
| serverIP | string | - | 服务器IP |
| serverPort | int | (0, 65535] | 포트 번호 |
| uploadMode | string | 默认"FTP" | 업로드방식 |
| username | string | - | 用户名 |

```json
{
  "dataCollectTime": 1.0,
  "dataUploadTime": 30.0,
  "fileSuffix": "csv",
  "filepath": "",
  "password": "",
  "robot": 1,
  "sendInfoFileFlag": false,
  "serverIP": "192.168.1.14",
  "serverPort": 21,
  "uploadMode": "FTP",
  "username": ""
}
```

## 3. 数据형식

> **설명:** 호스트 컴퓨터 설정与조회分四회进줄

### 3.1 호스트 컴퓨터 설정数据형식

**명령어:** `0x7336`

| 매개변수名 | 유형 | 범위 | 描述 |
|--------|------|------|------|
| format | string | - | 数据형식，형식의미：Robot-R1_年-月-日_时-分-s，详细설명参考数据형식 |
| robot | int | [1, 4] | 현재로봇 번호 |
| type | int | [1, 4] | 현재설정的형식유형 |

**type 값 범위설명：**

| 值 | 描述 |
|----|------|
| 1 | 설명文档파일명 |
| 2 | 설명文档文件内容 |
| 3 | 数据文档파일명 |
| 4 | 数据文档文件内容 |

```json
{
  "format": "Robot-R1_ $Y%-$m%-$d%_ $H%-$M%-$S%",
  "robot": 1,
  "type": 1
}
```

### 3.2 호스트 컴퓨터 조회数据형식
