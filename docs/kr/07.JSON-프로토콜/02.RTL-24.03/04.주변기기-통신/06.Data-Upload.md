# 데이터업로드

## 1. 데이터传输스위치

### 1.1 호스트 컴퓨터 설정데이터传输스위치

**명령어:** `0x7330`

| 매개변수이름 | 유형 | 설명 |
|--------|------|------|
| enable | bool | 데이터传输스위치 |
| robot | int | 현재로봇 번호 |

```json
{
  "enable": true,
  "robot": 1
}
```

**명령어:** `0x7331`

| 매개변수이름 | 유형 | 설명 |
|--------|------|------|
| robot | int | 현재로봇 번호 |

```json
{
  "robot": 1
}
```

### 1.3 컨트롤러 응답데이터传输스위치

**명령어:** `0x7332`

| 매개변수이름 | 유형 | 설명 |
|--------|------|------|

## 2. 데이터업로드매개변수

### 2.1 호스트 컴퓨터 설정데이터업로드매개변수

**명령어:** `0x7333`

| 매개변수이름 | 유형 | 범위/값 범위 | 설명 |
|--------|------|-----------|------|
| dataCollectTime | int | [0, 999999]s | 데이터采集주기 |
| dataUploadTime | int | [0, 999999]s | 데이터업로드주기 |
| fileSuffix | string | "csv", "txt" | 파일형식 |
| filepath | string | - | 파일경로 |
| password | string | - | 密码 |
| robot | int | [1, 4] | 로봇 번호 |
| sendInfoFileFlag | bool | - | 예아니오전송설명文档 |
| serverIP | string | - | 서버IP |
| serverPort | int | (0, 65535] | 포트 번호 |
| uploadMode | string | 默认"FTP" | 업로드방식 |
| username | string | - | 用户이름 |

```json
{
  "dataCollectTime": 1.0,
  "dataUploadTime": 30.0,
  "fileSuffix": "csv",
  "filepath": "",
  "password": "",
  "robot": 1,
  "sendInfoFileFlag": false,
  "serverIP": "192.168.1.14",
  "serverPort": 21,
  "uploadMode": "FTP",
  "username": ""
}
```

## 3. 데이터형식

> **설명:** 호스트 컴퓨터 설정与조회分四회进줄

### 3.1 호스트 컴퓨터 설정데이터형식

**명령어:** `0x7336`

| 매개변수이름 | 유형 | 범위 | 설명 |
|--------|------|------|------|
| format | string | - | 데이터형식，형식의미：Robot-R1_年-月-日_时-分-s，详细설명参考데이터형식 |
| robot | int | [1, 4] | 현재로봇 번호 |
| type | int | [1, 4] | 현재설정的형식유형 |

**type 값 범위설명：**

| 값 | 설명 |
|----|------|
| 1 | 설명文档파일명 |
| 2 | 설명文档파일内容 |
| 3 | 데이터文档파일명 |
| 4 | 데이터文档파일内容 |

```json
{
  "format": "Robot-R1_ $Y%-$m%-$d%_ $H%-$M%-$S%",
  "robot": 1,
  "type": 1
}
```

### 3.2 호스트 컴퓨터 조회데이터형식

**명령어:** `0x7337`

| 매개변수이름 | 유형 | 범위 | 설명 |
|--------|------|------|------|
| robot | int | [1, 4] | 현재로봇 번호 |
| type | int | [1, 4] | 현재설정的형식유형 |

```json
{
  "robot": 1,

}
```

### 3.3 컨트롤러 응답데이터형식
