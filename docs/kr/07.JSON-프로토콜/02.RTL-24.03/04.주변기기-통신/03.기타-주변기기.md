# 其他外设

## 编码器

### 설명

호스트 컴퓨터설정외부 축编码器溢值매개변수

**명령어：** `0x7200`

#### 매개변수 목록

| 필드名 | 유형 | 범위 | 설명 |
|--------|------|------|------|
| externalAxis | array | - | 외부 축编码器最大值最小值매개변수 목록，长度为当前외부 축轴数 |
| encode | array | - | 编码器最大值最小值매개변수 목록，长度为该轴从动轴数目+1 |
| max | double | [-2^32, 2^32] | 编码器最大值 |
| min | double | [-2^32, 2^32] | 编码器最小值 |
| externalGroupNum | int | [1, 3] | 要설정매개변수的轴组号 |
| robot | int | [1, 4] | 当前로봇号 |

#### 요청 예시

```json
{
  "externalAxis": [
    {
      "encode": [
        {
          "max": "100.0",
          "min": "-100.0"
        }
      ]
    }
  ],
  "externalGroupNum": 1,
  "robot": 2
}
```

## 伺服内部매개변수

### 호스트 컴퓨터설정伺服매개변수

**명령어：** `0x7240`

#### 매개변수 목록

| 필드名 | 유형 | 범위 | 설명 |
|--------|------|------|------|
| robot | int | [1, 4] | 当前로봇号 |
| servoNum | int | - | 需要설정的轴号 |
| servo | object | - | 伺服内部매개변수 목록；不同机型获取到매개변수不一致 |
| value | int/double | - | 설정的매개변수值 |
| temporary_save | int | [0, 1] | 是否为临时储存；0表示非临时储存，1表示临时储存 |
