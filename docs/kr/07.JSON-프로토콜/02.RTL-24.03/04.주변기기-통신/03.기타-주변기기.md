# 기타外设

## 엔코더

### 설명

호스트 컴퓨터설정외부 축엔코더溢값매개변수

**명령어：** `0x7200`

#### 매개변수 목록

| 필드이름 | 유형 | 범위 | 설명 |
|--------|------|------|------|
| externalAxis | array | - | 외부 축엔코더최대값최소값매개변수 목록，길이을/를현재외부 축축수 |
| encode | array | - | 엔코더최대값최소값매개변수 목록，길이을/를该축종속 축수目+1 |
| max | double | [-2^32, 2^32] | 엔코더최대값 |
| min | double | [-2^32, 2^32] | 엔코더최소값 |
| externalGroupNum | int | [1, 3] | 要설정매개변수的축그룹 번호 |
| robot | int | [1, 4] | 현재로봇번호 |

#### 요청 예시

```json
{
  "externalAxis": [
    {
      "encode": [
        {
          "max": "100.0",
          "min": "-100.0"
        }
      ]
    }
  ],
  "externalGroupNum": 1,
  "robot": 2
}
```

## 서보内部매개변수

### 설명

컨트롤러전송서보内部매개변수时会十번째매개변수을/를1组，分多회전송给호스트 컴퓨터

### 호스트 컴퓨터설정서보매개변수

**명령어：** `0x7240`

#### 매개변수 목록

| 필드이름 | 유형 | 범위 | 설명 |
|--------|------|------|------|
| robot | int | [1, 4] | 현재로봇번호 |
| servoNum | int | - | 필요설정的축번호 |
| servo | object | - | 서보内部매개변수 목록；아님同机型가져오기到매개변수아님1致 |
| value | int/double | - | 설정的매개변수값 |
| temporary_save | int | [0, 1] | 여부을/를临时储存；0의미非临时储存，1의미临时储存 |

#### 요청 예시
