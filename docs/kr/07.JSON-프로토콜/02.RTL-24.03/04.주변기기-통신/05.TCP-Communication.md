# Tcp通讯

## 호스트 컴퓨터 설정Tcp通讯参数

**명령어:** `0x7320`

### 매개변수 목록

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| client | object | 아니오 | 클라이언트 매개변수，当 `type` 为 1 时该节点存在 |
| client.frameHeader | string | 아니오 | 数据帧头，留空为没有 |
| client.ip | string | 아니오 | 需要连接的服务器IP |
| client.numberSystem | int | 아니오 | 接收到的数据解析进制：0-十进制，1-十六进制 |
| client.port | int | 아니오 | 通讯端口，范围 (0, 65535] |
| client.separator | string | 아니오 | 数据分隔符 |
| client.terminator | string | 아니오 | 数据帧尾，留空为没有 |
| server | object | 아니오 | 서버 매개변수，当 `type` 为 0 时该节点存在 |
| server.frameHeader | string | 아니오 | 数据帧头，留空为没有 |
| server.ip | string | 아니오 | 当前用作服务器IP |
| server.numberSystem | int | 아니오 | 接收到的数据解析进制：0-十进制，1-十六进制 |
| server.port | int | 아니오 | 通讯端口，范围 (0, 65535] |
| server.separator | string | 아니오 | 数据分隔符 |
| server.terminator | string | 아니오 | 数据帧尾，留空为没有 |
| craft | int | 예 | 工艺号，范围 [1, 9] |
| robot | int | 예 | 当前로봇 번호，范围 [1, 4] |
| type | int | 예 | TCP通讯方式：0-服务器，1-客户端，2-配置文件 |

### 요청 예시

```json
{
  "client": {
    "frameHeader": "@",
    "ip": "192.168.1.111",
    "numberSystem": 0,
    "port": 9000,
    "separator": ",",
    "terminator": "!"
  },
  "craft": 1,
  "robot": 1,
  "server": {
    "frameHeader": "@",
    "ip": "192.168.0.229",
    "numberSystem": 0,
    "port": 9001,
    "separator": ",",
    "terminator": "!"
  },
  "type": 1
}
```

## 호스트 컴퓨터 조회Tcp通讯参数

### 매개변수 목록

| 字段 | 类型 | 必填 | 说明 |

| craft | int | 예 | 工艺号，范围 [1, 9] |
| robot | int | 예 | 로봇 번호，范围 [1, 4] |
| type | int | 예 | TCP通讯方式：0-服务器，1-客户端，2-配置文件；为 2 时控制器发送서버 매개변수 |

  "type": 2
}
