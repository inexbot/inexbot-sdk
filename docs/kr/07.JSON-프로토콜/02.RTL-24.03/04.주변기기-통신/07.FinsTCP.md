# finsTCP

## 1. 호스트 컴퓨터 설정 finsTCP 매개변수

**명령어:** `0x7340`

### 매개변수 목록

| 매개변수 | 유형 | 설명 | 범위/형식 |
|--------|------|------|-----------|
| IP | string | IP 주소 | - |
| PLCReadAddress | string | PLC 읽기首주소 | "DM" 开头 + 숫자，범위 [1, 9999] |
| PLCWriteAddress | string | PLC 쓰기首주소 | "DM" 开头 + 숫자，범위 [1, 9999] |
| craftNum | int | 프로세스号 | 범위 [1, 9] |
| localReadAddress | int[] | 本机읽기首주소열表 | 범위 [1, 990]，제一비트为全局불리언首주소，제二비트为全局정수형首주소 |
| localWriteAddress | int[] | 本机쓰기首주소열表 | 범위 [1, 990]，제一비트为全局불리언首주소，제二비트为全局정수형首주소 |
| port | int | 포트 번호 | 범위 [0, 65535] |

### 요청 예시

```json
{
  "IP": "192.168.1.10",
  "PLCReadAddress": "DM2000",
  "PLCWriteAddress": "DM1000",
  "craftNum": 1,
  "localReadAddress": [500, 500],
  "localWriteAddress": [100, 100],
  "port": 9600
}
```

## 3. 컨트롤러가 호스트 컴퓨터에 응답 finsTCP 매개변수

**명령어:** `0x7342`

### 매개변수 목록

| 매개변수 | 유형 | 설명 | 범위/형식 |
|--------|------|------|-----------|
| IP | string | IP 주소 | - |
| PLCReadAddress | string | PLC 읽기首주소 | "DM" 开头 + 숫자，범위 [1, 9999] |
| PLCWriteAddress | string | PLC 쓰기首주소 | "DM" 开头 + 숫자，범위 [1, 9999] |
| craftNum | int | 프로세스号 | 범위 [1, 9] |
| localReadAddress | int[] | 本机읽기首주소열表 | 범위 [1, 990]，제一비트为全局불리언首주소，제二비트为全局정수형首주소 |
