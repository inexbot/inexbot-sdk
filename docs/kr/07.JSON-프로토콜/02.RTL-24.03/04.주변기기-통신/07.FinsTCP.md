# finsTCP

## 1. 호스트 컴퓨터 설정 finsTCP 参数

**명령어:** `0x7340`

### 매개변수 목록

| 매개변수 | 유형 | 설명 | 范围/格式 |
|--------|------|------|-----------|
| IP | string | IP 地址 | - |
| PLCReadAddress | string | PLC 读取首地址 | "DM" 开头 + 数字，范围 [1, 9999] |
| PLCWriteAddress | string | PLC 写入首地址 | "DM" 开头 + 数字，范围 [1, 9999] |
| craftNum | int | 工艺号 | 范围 [1, 9] |
| localReadAddress | int[] | 本机读取首地址列表 | 范围 [1, 990]，第一位为全局布尔首地址，第二位为全局정수형首地址 |
| localWriteAddress | int[] | 本机写入首地址列表 | 范围 [1, 990]，第一位为全局布尔首地址，第二位为全局정수형首地址 |
| port | int | 포트 번호 | 范围 [0, 65535] |

### 요청 예시

```json
{
  "IP": "192.168.1.10",
  "PLCReadAddress": "DM2000",
  "PLCWriteAddress": "DM1000",
  "craftNum": 1,
  "localReadAddress": [500, 500],
  "localWriteAddress": [100, 100],
  "port": 9600
}
```

## 3. 컨트롤러가 호스트 컴퓨터에 응답 finsTCP 参数

**명령어:** `0x7342`

### 매개변수 목록

| 매개변수 | 유형 | 설명 | 范围/格式 |
|--------|------|------|-----------|
| IP | string | IP 地址 | - |
| PLCReadAddress | string | PLC 读取首地址 | "DM" 开头 + 数字，范围 [1, 9999] |
| PLCWriteAddress | string | PLC 写入首地址 | "DM" 开头 + 数字，范围 [1, 9999] |
| craftNum | int | 工艺号 | 范围 [1, 9] |
| localReadAddress | int[] | 本机读取首地址列表 | 范围 [1, 990]，第一位为全局布尔首地址，第二位为全局정수형首地址 |
| localWriteAddress | int[] | 本机写入首地址列表 | 范围 [1, 990]，第一位为全局布尔首地址，第二位为全局정수형首地址 |
| port | int | 포트 번호 | 范围 [0, 65535] |
| status | int | 通讯状态 | 范围 [0, 1]，0 表示未连接，1 表示已连接 |

### 응답 예시

```json
{
  "IP": "192.168.1.10",
  "PLCReadAddress": "DM2000",
  "PLCWriteAddress": "DM1000",
  "craftNum": 1,
  "localReadAddress": [500, 500],
  "localWriteAddress": [100, 100],
  "port": 9600,
  "status": 0
}
```
