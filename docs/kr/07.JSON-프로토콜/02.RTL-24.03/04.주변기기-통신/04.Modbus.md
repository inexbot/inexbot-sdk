# Modbus

## 1. 호스트 컴퓨터설정modbus主站매개변수

**명령어：** `0x7300`

### 매개변수 목록

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| masterStation | object | 是 | 主站매개변수，包含RTU매개변수和TCP매개변수 |
| startAddress | bool | 否 | 주소偏移；true表示주소不变，false表示주소自动-1 |

#### masterStation 매개변수

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| type | string | 是 | 主站유형；取值"TCP","RTU" |
| processNumber | int | 是 | 需要설정的프로세스号；범위[1,9] |
| RTU | object | 否 | RTU매개변수 |
| TCP | object | 否 | TCP매개변수 |

#### RTU 매개변수

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| baudrate | int | 是 | 波特率 |
| checkBit | string | 是 | 校验位；取值"N","E","O" |
| dataBit | int | 是 | 데이터位；범위[5,8] |
| port | int | 是 | 포트 번호 |
| slaveId | int | 是 | 从站ID；범위[0,65535] |
| stopBit | int | 是 | 停止位；범위[0,1] |

#### TCP 매개변수

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| IP | string | 是 | IP주소 |
| endian_type | int | 是 | Float大小端；0表示AB CD，1表示CD AB，2表示BA DC，3表示DC BA |
| port | int | 是 | 포트 번호；범위[0,65535] |

### 요청 예시

```json
{
  "masterStation": {
    "RTU": {
      "baudrate": 115200,
      "checkBit": "E",
      "dataBit": 5,
      "port": 2,
      "slaveId": 1,
      "stopBit": 1
    },
    "TCP": {
      "IP": "192.168.1.14",
      "endian_type": 1,
      "port": 503
    },
    "processNumber": 1,
    "type": "TCP"
  },
  "startAddress": false
}
```
