# OPC UA

## 호스트 컴퓨터에서 OPC UA 매개변수 설정

**명령어:** `0x7350`

### 매개변수 설명

| 필드 | 유형 | 설명 |
|------|------|------|
| client | object | 클라이언트 매개변수 |
| client.enable | bool | 연결 활성화 |
| client.ip | string | 서버 IP 주소 |
| client.port | int | 포트 번호, 범위 [0,65535] |
| client.resource | string | 서버 이름 |
| server | object | 서버 매개변수 |
| server.enable | bool | 연결 활성화 |
| server.ip | string | 컨트롤러를 서버로 사용할 때의 IP 주소 |
| server.port | int | 포트 번호, 범위 [0,65535] |

### 요청 예시

```json
{
  "client":
  {
    "enable":false,
    "ip":"192.168.1.240",
    "port":49400,
    "resource":"ProsysServer"
  },
  "server":
  {
    "enable":false,
    "ip":"192.168.0.229",
    "port":4840
  }
}
```
