# IO

## 상태提示

> 설명：提示포트범위最大均为当前绑定的IO数量；输出方式범위均为[0,2]，0为0，1为1，2为闪烁

## IO复位

> 설명："enable"与"value"节点长度均为128，实际有效데이터为当前IO的总포트数，超过部分默认为0

| enable | array | 是否复位；bool유형，长度128 |

## IO구성

> 설명：串口模拟IO매개변수（若EtherCAT IO板有模拟IO，则该串口模拟IO将无效）

### 호스트 컴퓨터설정IO구성매개변수

**명령어：** `0x7130`

**매개변수 설명：**

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| serialAnalog | object | 串行模拟매개변수 |
| serialAnalog.baudRate | int | 波特率 |
| serialAnalog.port | int | 포트 번호，범위(0,65535] |
| serialAnalog.type | string | 模拟유형<br/>"DAC_ANAIO"：DAC模拟IO板<br/>"SUPER_ANAIO"：格控模拟IO板 |
| simuNum | int | 虚拟IO数量，범위[0,4] |

**요청 예시：**

```json
{
  "serialAnalog":
  {
    "baudRate":115200,
    "port":2,
    "type":"SUPER_ANAIO"
  },
  "simuNum":2
}
```

## 安全설정

### 호스트 컴퓨터설정安全설정매개변수

**명령어：** `0x7140`

**매개변수 설명：**

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| quickStop | object | 紧急停止매개변수 |
| quickStop.enable | bool | 紧急停止使能 |
| quickStop.port1 | int | 紧急停止1数字输入포트 |
| quickStop.port2 | int | 紧急停止2数字输入포트 |
| quickStop.shieldFlag1 | bool | 屏蔽紧急停止1 |
| quickStop.shieldFlag2 | bool | 屏蔽紧急停止2 |
| quickStop.shieldTime | int | 屏蔽紧急停止时间，범위[0,100000)s |
| quickStop.time | double | 快速停止时间，범위[50,200]ms |

**매개변수 설명：**

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| port | array | 强制数字输入的포트 번호；int유형 |
| status | array | 强制数字输入的值；int유형 |

**응답 예시：**

```json
{
  "port":[2,6,12],
  "status":[0,1,0]
}
```

### 호스트 컴퓨터설정强制模拟输入

**명령어：** `0x7193`

**매개변수 설명：**

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| enable | int | 是否强制模拟输入，범위[0,1] |
| port | int | 强制模拟输入的포트 번호 |
| status | double | 强制模拟输入的当前值（该节点数值不小于0） |

**요청 예시：**

```json
{
  "enable":1,
  "port":2,
  "status":9.0
}
```

### 컨트롤러回复호스트 컴퓨터强制模拟输入포트

**명령어：** `0x7195`

**매개변수 설명：**

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| port | array | 强制数字输入的포트 번호；int유형 |
| status | array | 强制数字输入的值；double유형 |

**응답 예시：**

```json
{

  "status":[5.0,9.0]
}
```
