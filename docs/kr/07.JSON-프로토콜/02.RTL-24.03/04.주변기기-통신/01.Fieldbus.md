# 总线

## CanOpen

### 0x7000 호스트 컴퓨터 조회该硬件的can支持的功能

**명령어:** `0x7000`

**요청 매개변수:**

| 매개변수 | 유형 | 설명 |
|--------|------|------|
| canPort | UInt | can포트 번호 |
| function | Int | 该硬件can支持的功能，范围[0,3]<br/>0-标准帧，1-扩展帧，2-远程帧，3-过滤id |

**요청 예시:**

```json
{
  "canPort": 1,
  "function": 0
}
```

---

### 0x7001 컨트롤러가 호스트 컴퓨터에 응답조회该硬件的can支持的功能结果

**명령어:** `0x7001`

**응답 매개변수:**

| 매개변수 | 유형 | 설명 |
|--------|------|------|
| canPort | UInt | can포트 번호 |
| support | Bool | 该硬件的can예아니오支持这功能 |

**응답 예시:**

```json
{
  "canPort": 1,
  "support": true
}
```

---

### 0x7002 호스트 컴퓨터 설정can参数

**명령어:** `0x7002`

**요청 매개변수:**

| 매개변수 | 유형 | 설명 |
|--------|------|------|
| canBaud | UInt | can通讯波特率 |
| canPort | UInt | can通讯端口 |
| enableRecvFiltert | Bool | 예아니오开启过滤ID |
| frameFormat | Int | 帧格式，范围[0,1]<br/>0-标准帧，1-扩展帧 |
| recvFilterId | UInt | 设置过滤ID |

**요청 예시:**

```json
{
  "canBaud": 115200,
  "canPort": 1,
  "enableRecvFiltert": true,
  "frameFormat": 0,
  "recvFilterId": 1
}
```

### 0x7004 컨트롤러가 호스트 컴퓨터에 응답接收一帧can数据结果

**명령어:** `0x7004`

**응답 매개변수:**

| 매개변수 | 유형 | 설명 |
|--------|------|------|
| canPort | UInt | can通讯端口 |
| data | UChar[] | 接收到的数据，长度为Length节点的值 |
| Length | UInt | 接收到的数据长度 |
| recvCanID | UInt | 接收到的帧ID |

**응답 예시:**

```json
{
  "canPort": 1,
  "data": [10],
  "Length": 1,
  "recvCanID": 1
}
```

  "useRemoteFrame": true

## ENI

### 0x7010 호스트 컴퓨터 조회ENI文件列表

**명령어:** `0x7010`

**요청 매개변수:**

| 매개변수 | 유형 | 설명 |
|--------|------|------|
| num | Int | 获取ENI文件数目，范围[1,10]<br/>1-获取当前使用的ENI文件，10-获取所有ENI文件 |

**요청 예시:**

```json
{
  "num": 10
}
```

---

### 0x7011 컨트롤러가 호스트 컴퓨터에 응답ENI文件列表

**명령어:** `0x7011`

**응답 매개변수:**

| 매개변수 | 유형 | 설명 |
|--------|------|------|
| ENIfilelist | String[] | 控制器ENI文件列表 |
| ENIfilenum | Int | ENI文件数目 |
| absolutepath | String | 控制器中ENI文件所在文件夹 |

**응답 예시:**

```json
{
  "ENIfilelist": [
    "INEXBOT-IO-R4-1.xml",
    "DST_X503-1.json",
    "DST_X503-1.xml",
    "KunWeiTech_FT_Sensors-1.xml",
    "P140000107-1.0.1.1-ECXML-1.xml"
  ],
  "ENIfilenum": 10,
  "absolutepath": "eni/"
}
```
