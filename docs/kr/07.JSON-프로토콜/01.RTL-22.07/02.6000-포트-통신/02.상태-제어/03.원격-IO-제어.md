# 远程IO제어

## 1. IO기능界面설정

### 1.1 IO기능界面설정（示教盒전송）

**명령码**: `0x2F01  IO_CONTROL_SET`

**매개변수 설명**:

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| inPort | object | 是 | 远程IO기능포트绑定 |
| inValue | object | 是 | 포트值，IO매개변수，0：低电平有效，1：高电平有效 |
| program | array | 是 | 远程IO프로그램DIN和value绑定，共10个 |
| robot | int | 是 | 로봇号 |

**inPort 对象매개변수**:

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| start | int | 启动，绑定포트 |
| stop | int | 停止，绑定포트 |
| pause | int | 暂停，绑定포트 |
| faultReset | int | 清除报警，绑定포트 |
| clearoutagekeep | int | 清除断电保持데이터，绑定포트 |

**inValue 对象매개변수**:

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| start | int | 启动 |
| stop | int | 停止 |
| pause | int | 暂停 |
| faultReset | int | 清除报警 |
| clearoutagekeep | int | 清除断电保持데이터 |

**program 数组매개변수**:

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| port | int | 포트 번호 |
| value | int | 포트值，0或1 |

**요청 예시**:

```json
{
  "inPort": {
    "clearoutagekeep": 16,
    "faultReset": 4,
    "pause": 3,
    "start": 1,
    "stop": 2
  },
  "inValue": {
    "clearoutagekeep": 1,
    "faultReset": 1,
    "pause": 1,
    "start": 1,
    "stop": 1
  },
  "program": [
    {"port": 5, "value": 1},
    {"port": 6, "value": 0},
    {"port": 7, "value": 0},
    {"port": 8, "value": 1},
    {"port": 9, "value": 1},
    {"port": 0, "value": 1},
    {"port": 0, "value": 1},
    {"port": 0, "value": 1},
    {"port": 0, "value": 1},
    {"port": 0, "value": 1}
  ],
  "robot": 1
}
```

### 2.4 安全点위치通知

**명령码**: `0x2F16  IO_CONTROL_NOTIS_SAFEPOS`

**설명**: 실행작업파일时，开始安全使能，不在安全点时컨트롤러 응답전송

**매개변수 설명**:

| 매개변수名 | 유형 | 必填 | 설명 |
|--------|------|------|------|
| robot | int | 是 | 当前로봇 |
| isSync | bool | 是 | false:로봇不在安全区，true:외부 축不在安全区 |
| currentPos | array | 是 | 当前좌표 |
| safePos | array | 是 | 홈 위치位 |

**응답 예시**:

```json
{
  "robot": 1,
  "isSync": false,
  "currentPos": [0, 0.1, 2, 3.3, 44, 555.55, 6, 7.7, 88],
  "safePos": [0, 0.1, 2, 3.3, 88, 555.55, 6, 7.7, 88]
}
```
