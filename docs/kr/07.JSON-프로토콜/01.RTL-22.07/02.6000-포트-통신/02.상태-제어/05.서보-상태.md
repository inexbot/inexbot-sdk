## 2. 伺服상태설정及获取

### 2.1 호스트 컴퓨터改变伺服的상태

| 매개변수 | 유형 | 설명 | 取值범위 |
|------|------|------|----------|
| robot | int | 选择로봇 | [1, 4] |
| status | int | 伺服상태 | 0: 真实伺服, 1: 虚拟伺服, 2: 无伺服 |

| 项目 | 值 |
|------|-----|
| 명령字 | 0x2001 SERVO_STATUS_SET |

```json
{
  "robot": 1,
  "status": 0
}
```

### 2.2 호스트 컴퓨터조회伺服상태

| 项目 | 值 |
|------|-----|
| 명령字 | 0x2002 SERVO_STATUS_INQUIRE |

```json
{
  "robot": 1
}
```

### 2.3 컨트롤러 응답伺服상태

| 매개변수 | 유형 | 설명 | 取值범위 |
|------|------|------|----------|
| robot | int | 选择로봇 | [1, 4] |
| mode | int | 실행모드 | 0: 单机모드, 1: 多机모드 |
| status | int | 伺服상태 | 0: 停止, 1: 就绪, 2: 错误, 3: 실행 |

| 项目 | 值 |
|------|-----|
| 명령字 | 0x2003 SERVO_STATUS_RESPOND |

```json
{
  "mode": 0,
  "robot": 1,
  "status": 0
}
```

## 6. 清除伺服错误

### 6.1 티치 펜던트清除伺服错误

| 매개변수 | 유형 | 설명 | 取值범위 |
|------|------|------|----------|
| robot | int | 选择로봇 | [1, 4] |

| 项目 | 值 |
|------|-----|
| 명령字 | 0x3201 FAULT_RESET |

```json
{
  "robot": 1
}
```

### 6.2 清除伺服错误后，반환清除结果

| 매개변수 | 유형 | 설명 |
|------|------|------|
| robot | int | 选择로봇 |
| clearErrflag | bool | true: 清除伺服错误成功, false: 清除失败 |

| 项目 | 值 |
|------|-----|
| 명령字 | 0x3202 FAULT_RESET_RESULT |

```json
{
  "robot": 1,
  "clearErrflag": true
}
```
