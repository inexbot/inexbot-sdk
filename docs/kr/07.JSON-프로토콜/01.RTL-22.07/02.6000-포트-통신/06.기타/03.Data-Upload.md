# 数据上传

## 设置数据上传开关

**命令码**: `0x5721`  `DATAUPLOAD_ENABLE_SET`

### 发送

```json
{
  "robot": 1,
  "enable": true
}
```

| 매개변수 | 유형 | 설명 |
|------|------|------|
| robot | int | 로봇 번호 |
| enable | bool | 数据예아니오上传（true-上传，false-不上传） |

---

## 设置数据传输参数

**命令码**: `0x5724`  `DATAUPLOAD_PARM_SET`

### 发送

```json
{
  "robot": 1,
  "uploadMode": "FTP",
  "fileSuffix": ".csv",
  "serverIP": "192.168.1.100",
  "serverPort": 21,
  "username": "admin",
  "password": "password",
  "filepath": "/data",
  "dataCollectTime": 60.0,
  "dataUploadTime": 300.0,
  "sendInfoFileFlag": true
}
```

| 매개변수 | 유형 | 설명 |
|------|------|------|
| robot | int | 로봇 번호 |
| uploadMode | string | 上传方式（如 FTP） |
| fileSuffix | string | 文件格式（如 .csv） |
| serverIP | string | 服务器 IP 地址 |
| serverPort | int | 포트 번호 |
| username | string | 用户名 |
| password | string | 密码 |
| filepath | string | 路径 |
| dataCollectTime | double | 数据采集周期（秒） |
| dataUploadTime | double | 数据上传周期（秒） |
| sendInfoFileFlag | bool | 예아니오发送说明文件 |

## 设置上传数据格式

### 发送

```json
{
  "robot": 1,
  "type": 1,
  "format": "datafile"
}
```

| 매개변수 | 유형 | 설명 |
|------|------|------|
| robot | int | 로봇 번호 |
| type | int | 数据类型（1, 2, 3, 4） |
| format | string | 说明文档文件名 |
