# 데이터 업로드

## 데이터 업로드 스위치 설정

**명령 코드**: `0x5721`  `DATAUPLOAD_ENABLE_SET`

### 전송

```json
{
  "robot": 1,
  "enable": true
}
```

| 매개변수 | 유형 | 설명 |
|------|------|------|
| robot | int | 로봇 번호 |
| enable | bool | 데이터 업로드 여부 (true-업로드, false-업로드 안 함) |

## 데이터 전송 매개변수 설정

**명령 코드**: `0x5724`  `DATAUPLOAD_PARM_SET`

### 전송

```json
{
  "robot": 1,
  "uploadMode": "FTP",
  "fileSuffix": ".csv",
  "serverIP": "192.168.1.100",
  "serverPort": 21,
  "username": "admin",
  "password": "password",
  "filepath": "/data",
  "dataCollectTime": 60.0,
  "dataUploadTime": 300.0,
  "sendInfoFileFlag": true
}
```

| 매개변수 | 유형 | 설명 |
|------|------|------|
| robot | int | 로봇 번호 |
| uploadMode | string | 업로드 방식 (예: FTP) |
| fileSuffix | string | 파일 형식 (예: .csv) |
| serverIP | string | 서버 IP 주소 |
| serverPort | int | 포트 번호 |
| username | string | 사용자 이름 |
| password | string | 비밀번호 |
| filepath | string | 경로 |
| dataCollectTime | double | 데이터 수집 주기 (초) |
| dataUploadTime | double | 데이터 업로드 주기 (초) |
| sendInfoFileFlag | bool | 설명 파일 전송 여부 |

### 전송

```json
  "robot": 1,
  "type": 1,
  "format": "datafile"
}
```

| 매개변수 | 유형 | 설명 |
|------|------|------|
| robot | int | 로봇 번호 |

| format | string | 설명 문서 파일명 |
