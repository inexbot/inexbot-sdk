# 恩普伺服

## 충돌 감지

**0x3203 NEP_FUNCTION_SET**

요청 매개변수:

| 매개변수 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| robot | int | 예 | 로봇 번호 |
| collisionDetectionFlag | bool | 예 | 충돌 감지 활성화 |
| dragTeachFlag | bool | 예 | 드래그 티치 활성화 |
| collisionThreshold | int[6] | 예 | 각 축 충돌 임계값 |

요청 예시:

```json
{
    "robot": 1,
    "collisionDetectionFlag": true,
    "dragTeachFlag": false,
    "collisionThreshold": [1300, 1300, 1300, 1300, 1300, 1300]
}
```

요청 매개변수:

|--------|------|------|------|
| robot | int | 예 | 로봇 번호 |

| dragTeachFlag | bool | 예 | 드래그 티치 활성화 |

응답 예시:

    "dragTeachFlag": false,

## 충돌 해제

**0x3206 NEP_FUNCTION_CLEAR**

요청 매개변수:

| 매개변수 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| robot | int | 예 | 로봇 번호 |

요청 예시:

```json
{
    "robot": 1
}
```

**0x3207 NEP_FUNCTION_CLEAR_RESPOND**

컨트롤러 응답응답 매개변수:

| 매개변수 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| robot | int | 예 | 로봇 번호 |
| clear | bool | 예 | true: 충돌 해제成功 false: 충돌 해제失败 |

응답 예시:

```json
{
    "robot": 1,
    "clear": true
}
```
