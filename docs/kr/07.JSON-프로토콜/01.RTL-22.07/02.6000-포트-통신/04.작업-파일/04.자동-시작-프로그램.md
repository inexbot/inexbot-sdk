# 자동 시작 프로그램 실행

## 티치 펜던트 설정

**레지스터:** `0x5081` `SELF_START_RUN_SET`

### 매개변수 설명

| 매개변수  | 유형   | 필수 | 설명         | 예시 값 |
|---------|--------|------|--------------|--------|
| robot   | int    | 예   | 로봇 번호     | 1      |
| enable  | bool   | 예   | 활성화 여부     | false  |
| count   | int    | 예   | 실행 횟수     | 1      |
| speed   | int    | 예   | 실행 속도     | 15     |
| jobname | string | 예   | 작업 파일 이름 | "Q1"   |

### JSON 예시

```json
{
    "robot": 1,
    "enable": false,
    "count": 1,
    "speed": 15,
    "jobname": "Q1"
}
```
