# Lua 프로그램

Lua 파일은 `~/robot/job/lua` 디렉터리에 저장됩니다.

파일 업로드 및 파일 목록 조회는 SCP 방식을 사용합니다.

## 프로그램 실행

**호스트 컴퓨터 전송: 0x2511**

```json
{
  "fileName": "xxx.lua"
}
```

**컨트롤러 응답: 0x2512**

```json
{
  "fileName": "xxx.lua",
  "result": true,
  "error": "오류 원인"
}
```

| 매개변수 | 유형 | 필수 | 설명 |
|------|------|------|------|
| fileName | string | 예 | Lua 파일명 |
| result | bool | 예 | 실행 성공 여부 |
| error | string | 예 | 오류 원인, 성공 시 빈 문자열 |

## 프로그램 정지

**호스트 컴퓨터 전송: 0x2513**

```json
{
  "fileName": "xxx.lua"
}
```

| 매개변수 | 유형 | 필수 | 설명 |
|------|------|------|------|
| fileName | string | 예 | Lua 파일명 |

---

## 현재 실행 상태

**호스트 컴퓨터 전송: 0x2515**

```json
{}
```

**컨트롤러 응답: 0x2516**

```json
{
  "fileNames": [
    {
      "fileName": "xxx.lua",
      "status": 1
    }
  ]
}
```

| 매개변수 | 유형 | 필수 | 설명 |
|------|------|------|------|
| fileNames | array | 예 | 실행 중인 파일 목록 |
| fileNames[].fileName | string | 예 | Lua 파일명 |
| fileNames[].status | int | 예 | 1-일시정지, 2-실행 |
