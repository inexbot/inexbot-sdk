# 원격 프로그램 설정

## 원격 모드 연결 상태

### Modbus, 외부 IO 연결 상태 조회

**명령어**: `0x5032 REMOTE_CONNECT_INQUIRE`

요청 본문:

```json
{}
```

### 연결 상태 반환

**명령어**: `0x5033 REMOTE_CONNECT_RESPOND`

| 매개변수 | 유형 | 설명 |
|------|------|------|
| ModbusConnect | bool | Modbus 연결 상태: `false` 미연결, `true` 연결됨 |
| ExternIOConnect | bool | 외부 IO 연결 상태: `false` 미연결, `true` 연결됨 |
| type | int | 연결 유형: 0:NAN, 1:RTU, 2:TCP |

응답 본문:

```json
{
  "ModbusConnect": false,
  "ExternIOConnect": true,
  "type": 2
}
```

## 원격 작업 파일 설정

### 원격 작업 파일 설정

**명령어**: `0x2C01 REMOTE_JOBFILE_SET`

| 매개변수 | 유형 | 설명 |
|------|------|------|
| robot | int | 로봇 번호 |
| jobFile | array | 작업 파일 목록 |

**jobFile 배열 요소**:

| 매개변수 | 유형 | 설명 |
|------|------|------|
| name | string | 작업 파일 이름 |
| times | int | 실행 횟수 |

요청 본문:

```json
{
  "robot": 1,
  "jobFile": [
    {"name": "Q1", "times": 1},
    {"name": "Q2", "times": 1},
    {"name": "Q3", "times": 1},
    {"name": "Q4", "times": 1},
    {"name": "Q5", "times": 1}
    // ... 최대 10개
  ]
}
```

### 원격 작업 파일 조회

**명령어**: `0x2C02 REMOTE_JOBFILE_INQUIRE`

요청 본문:

```json
{
  "robot": 1
}
```

### 작업 파일 조회 결과 반환

**명령어**: `0x2C03 REMOTE_JOBFILE_RESPOND`

응답 본문:

```json
{
  "robot": 1,
  "jobFile": [
    {"name": "Q1", "times": 3},
    {"name": "Q2", "times": 3},
    {"name": "", "times": 1},
    {"name": "Q4", "times": 2},
    {"name": "", "times": 1}
  ]
}
```

## 원격 모드 매개변수 설정

### 원격 모드 매개변수 설정

**명령어**: `0x2C04 REMOTE_PARA_SET`

| 매개변수 | 유형 | 설명 |
|------|------|------|
| robot | int | 로봇 번호 |
| remoteDefaultSpeed | int | 원격 모드 속도, 범위 1~100 |
| reserveIsStart | int | 예약 후 시작: `1` 시작, `0` 해제 |
| reserveTrgTime | int | 시작 확인 시간, 범위 200~1000ms |
| waitingTime | int | IO 반복 트리거 시간 |

요청 본문:

```json
{
  "robot": 1,
  "remoteDefaultSpeed": 15,
  "reserveIsStart": 1,
  "reserveTrgTime": 200,
  "waitingTime": 500
}
```
