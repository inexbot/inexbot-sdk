# TCP통신

## 설정网络매개변수

*0x4180 MSGCOMM_PARAM_SET*

### 客户端

**요청 매개변수**

| 필드 | 유형 | 설명 |
|------|------|------|
| frameHeader | string | 帧头 |
| ip | string | IP 주소 |
| numberSystem | int | 0: 十进制, 1: 十六进制 |
| port | int | 포트 번호 |
| separator | string | 分隔符 |
| terminator | string | 종료符 |
| craft | int | 프로세스号 (1~9) |
| robot | int | 로봇 번호 |
| type | int | 0: 服务器, 1: 客户端 |

**요청 예시**

```json
{
  "client": {
    "frameHeader": "@",
    "ip": "192.168.1.111",
    "numberSystem": 1,
    "port": 9000,
    "separator": ",",
    "terminator": "!"
  },
  "craft": 1,
  "robot": 1,
  "type": 1
}
```

### 服务端

**요청 매개변수**

| 필드 | 유형 | 설명 |
|------|------|------|
| frameHeader | string | 帧头 |
| ip | string | IP 주소 |
| numberSystem | int | 0: 十进制, 1: 十六进制 |
| port | int | 포트 번호 |
| separator | string | 分隔符 |
| terminator | string | 종료符 |
| craft | int | 프로세스号 (1~9) |
| robot | int | 로봇 번호 |
| type | int | 0: 服务器, 1: 客户端 |

**요청 예시**

```json
{
  "craft": 3,
  "robot": 1,
  "server": {
    "frameHeader": "@A",
    "ip": "192.168.1.14",
    "numberSystem": 0,
    "port": 9001,
    "separator": "B",
    "terminator": "C"
  },
  "type": 0
}
```

## 响应网络매개변수조회
