## 6. 동역학 파라미터

> 동역학 식별은 정장 일반 6축, 무부하만 지원하며, 식별 관련 파라미터 설정

### 轨跡 파라미터 설정

| 필드 | 타입 | 설명 |
|------|------|------|
| 0x7601 | TRAJECTORY_PARAM_SET | |

**요청 파라미터:**

| 파라미터명 | 타입 | 필수 | 설명 |
|--------|------|------|------|
| traSize | double | 예 | 轨跡 범위 |
| traVel | double | 예 | 轨跡 속도 |

```json
{
  "traSize": 10.0,
  "traVel": 100.0
}
```
