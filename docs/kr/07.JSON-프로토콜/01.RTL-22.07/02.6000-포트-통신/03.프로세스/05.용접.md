# 용접프로세스

## 용접装置

설정용접装置时，전송：

*0x4001 WELDEQUIPMENT_SET*

```json
{
  "basic_functions": {
    "arc_detection_time": 1.3,
    "arc_depletion_detection_time": 1.3,
    "arc_detection_ensure_time": 1.3,
    "pre_give_gas_time": 1.5,
    "close_time": 1.9,
    "weld_interruption_detection": 1,
    "arc_fly": 1
  },
  "micro_adjustment": {
    "single_adjustment_current": 3,
    "single_adjustment_voltage": 4,
    "single_adjustment_speed": 5,
    "swing_weld_frequency": 6,
    "swing_weld_range": 1.1
  },
  "restart": {
    "restart_times": 1,
    "restart_distance": 2,
    "restart_speed": 3,
    "re_arc_times": 4,
    "restart_enable": false,
    "re_arc_enable": false,
    "restart_mode": 0
  },
  "antiCollision": {
    "enable": false,
    "inputPort": 1,
    "inputValue": 1,
    "stopTime": 80,
    "outputPort": 2,
    "outputValue": 1,
    "shieldFlag": false,
    "shieldTime": 30
  },
  "other": {
    "break_back_time": 1.2,
    "weld_finish_time": 1.3,
    "break_back_enable": false,
    "weld_finish_function": true,
    "simulation_set_zero": false
  },
  "collisionButtonFlag": {
    "error_enable_switch": true
  }
}
```

### basic_functions 매개변수 설명

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| arc_detection_time | double | 起弧检测时间 |
| arc_depletion_detection_time | double | 电弧耗尽检测时间 |
| arc_detection_ensure_time | double | 起弧确认检测时间 |
| pre_give_gas_time | double | 提前送气时间 |
| close_time | double | 延后关气时间 |
| weld_interruption_detection | int | 용접中断检测 |
| arc_fly | int | 飞行起弧 |

### micro_adjustment 매개변수 설명

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| single_adjustment_current | int | 용접电流单次调整量 |
| single_adjustment_voltage | int | 용접电压单次调整量 |
| single_adjustment_speed | int | 용접속도单次调整量 |
| swing_weld_frequency | int | 摆焊频率单次调整量 |
| swing_weld_range | double | 摆焊幅度单次调整量 |

### restart 매개변수 설명

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| restart_times | int | 再启动次数 |
| restart_distance | int | 再启动距离 |
| restart_speed | int | 再启动속도 |
| re_arc_times | int | 再起弧次数 |
| restart_enable | bool | 再启动使能 |
| re_arc_enable | bool | 再起弧使能 |
| restart_mode | int | 再启动모드：0-自动再启动，1-半自动再启动，2-停机 |

### antiCollision 매개변수 설명

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| enable | bool | 防碰撞기능使能 |
| inputPort | int | 防碰撞IO포트 |
| inputValue | int | 防碰撞触发电平 |
| stopTime | int | 防碰撞快速停止时间 |
| outputPort | int | 防碰撞输出提示포트 |
| outputValue | int | 防碰撞输出提示电平 |
| shieldFlag | bool | 屏蔽防碰撞使能 |
| shieldTime | int | 屏蔽时间 |

### other 매개변수 설명

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| break_back_time | double | 断弧回抽时间 |
| weld_finish_time | double | 용접完成回抽时间 |
| break_back_enable | bool | 断弧回抽使能 |
| weld_finish_function | bool | 용접完成回抽기능 |
| simulation_set_zero | bool | 灭弧模拟量置零기능 |

### collisionButtonFlag 매개변수 설명

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| error_enable_switch | bool | 拖拽点动모드切换开关 |

조회용접装置时，전송：

*0x4002 WELDEQUIPMENT_INQUIRE*

data：无

收到조회용접装置时，반환：

*0x4003 WELDEQUIPMENT_RESPOND*

同0x4001

## 설정电流电压匹配

설정电流电压匹配时，전송：

*0x4004 CURVOLMATCH_SET*

```json
{
  "current_control": {
    "set_voltage": [1, 2, 3, 4, 5, 6, 7, 8],
    "actual_current": [1, 2, 3, 4, 5, 6, 7, 8]
  },
  "voltage_control": {
    "set_voltage": [1, 2, 3, 4, 5, 6, 7, 8],
    "actual_voltage": [1, 2, 3, 4, 5, 6, 7, 8]
  },
  "robot": 1
}
```

### 매개변수 설명

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| current_control.set_voltage | array | 설정电流 |
| current_control.actual_current | array | 实际电流 |
| voltage_control.set_voltage | array | 설정电压 |
| voltage_control.actual_voltage | array | 实际电压 |
| robot | int | 로봇 번호 |

조회电流电压匹配时，전송：

*0x4005 CURVOLMATCH_INQUIRE*

data：无

收到조회电流电压匹配时，반환：

*0x4006 CURVOLMATCH_RESPOND*

```json
{
  "current_control": {
    "set_voltage": [1, 2, 3, 4, 5, 6, 7, 8],
    "actual_current": [1, 2, 3, 4, 5, 6, 7, 8]
  },
  "voltage_control": {
    "set_voltage": [1, 2, 3, 4, 5, 6, 7, 8],
    "actual_voltage": [1, 2, 3, 4, 5, 6, 7, 8]
  },
  "IO": {
    "AO_setcurrent": 0,
    "AO_setvoltage": 0
  },
  "robot": 1
}
```

### 반환매개변수 설명

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| current_control.set_voltage | array | 설정电流 |
| current_control.actual_current | array | 实际电流 |
| voltage_control.set_voltage | array | 설정电压 |
| voltage_control.actual_voltage | array | 实际电压 |
| IO.AO_setcurrent | int | 电流输出포트 |
| IO.AO_setvoltage | int | 电压输出포트 |
| robot | int | 로봇 번호 |

## 용접매개변수

설정용접매개변수时，전송：

*0x4007 WELDPARAMETER_SET*

```json
{
  "num": 1,
  "notes": "예시",
  "arcoff_parameters": {
    "arcoff_voltage": 1.1,
    "arcoff_current": 1.1,
    "arcoff_time": 1.1,
    "ramp_time": 1.1,
    "ramp_distance": 1.1,
    "ramp_enable": true,
    "ramp_mode": 1
  },
  "weld_parameters": {
    "weld_current": 1.1,
    "weld_voltage": 1.1
  },
  "arcon_parameters": {
    "arcon_current": 1.1,
    "arcon_voltage": 1.1,
    "arcon_time": 1.1,
    "arcon_ramp_time": 1.1,
    "arcon_ramp_distance": 1.1,
    "arcon_ramp_enable": true,
    "arcon_ramp_mode": 1,
    "arcon_use_param": true,
    "arcoff_use_param": true
  }
}
```

### 매개변수 설명

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| num | int | 매개변수序号，从1开始 |
| notes | string | 注释 |

### arcoff_parameters（收弧매개변수）

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| arcoff_voltage | double | 收弧电压 |
| arcoff_current | double | 收弧电流 |
| arcoff_time | double | 收弧时间 |
| ramp_time | double | 渐变时间 |
| ramp_distance | double | 渐变距离 |
| ramp_enable | bool | 渐变使能 |
| ramp_mode | int | 收弧渐变方式：0-时间渐变，1-距离渐变 |

### weld_parameters（용접매개변수）

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| weld_current | double | 용접电流 |
| weld_voltage | double | 용접电压 |

### arcon_parameters（起弧매개변수）

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| arcon_current | double | 起弧电流 |
| arcon_voltage | double | 起弧电压 |
| arcon_time | double | 起弧时间 |
| arcon_ramp_time | double | 渐变时间 |
| arcon_ramp_distance | double | 渐变距离 |
| arcon_ramp_enable | bool | 起弧渐变使能 |
| arcon_ramp_mode | int | 起弧渐变方式：0-时间渐变，1-距离渐变 |
| arcon_use_param | bool | 起弧매개변수使能 |
| arcoff_use_param | bool | 收弧매개변수使能 |

조회용접매개변수时，전송：

*0x4008 WELDPARAMETER_INQUIRE*

```json
{
  "num": 1,
  "robot": 1
}
```

收到조회용접매개변수时，반환：

*0x4009 WELDPARAMETER_RESPOND*

同0x4007

## 摆焊매개변수

설정摆焊매개변수时，전송：

*0x400B WEAVPARAMETER_SET*

```json
{
  "num": 1,
  "parameter": {
    "horizontal_angle": 5.0,
    "init_direction": 1,
    "isMove": 1,
    "left_standing_time": 4.0,
    "note": "AA",
    "right_standing_time": 3.0,
    "vertical_angle": 6.0,
    "weaving_ampli": 2.0,
    "weaving_freq": 1.0,
    "weaving_type": 1
  }
}
```

### 매개변수 설명

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| num | int | 매개변수序号，从1开始 |

### parameter

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| horizontal_angle | double | 水平偏角：-180 ~ +180 |
| init_direction | int | 起始方向：0-正方向(+1)，1-负方向(-1) |
| isMove | int | 移动모드：0-移动，1-停留 |
| left_standing_time | double | 左停留时间 |
| note | string | 注释 |
| right_standing_time | double | 右停留时间 |
| vertical_angle | double | 竖直偏角：-180 ~ +180 |
| weaving_ampli | double | 摆动幅度：0-50 |
| weaving_freq | double | 摆动频率 |
| weaving_type | int | 摆动유형：0-正弦，1-Z字形，2-圆形摆，3-외부 축定点摆 |

조회摆焊매개변수时，전송：

*0x400C WEAVPARAMETER_INQUIRE*

```json
{
  "num": 1
}
```

收到조회摆焊매개변수时，반환：

*0x400D WEAVPARAMETER_RESPOND*

同0x400B

## 电流电压匹配实际值

설정电流电压匹配매개변수实际值，修改值就会전송，无需保存：

*0x400E WELDFACTCURVOL_SET*

```json
{
  "type": 0,
  "value": 4.13
}
```

### 매개변수 설명

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| type | int | 0-电流제어匹配，2-电压제어匹配 |
| value | double | 值 |

컨트롤러 응답：

*0x400F WELDFACTCURVOL_RESPOND*

```json
{
  "result": 1,
  "robot": 1
}
```

### 반환매개변수 설명

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| result | int | 1-설정成功，0-설정失败 |
| robot | int | 로봇 번호 |

开始进行模拟量匹配：建立电流电压匹配界面会전송true，离开false

*0x4010 WELD_CURVOLMATCH_START*

```json
{
  "start": true
}
```

### 매개변수 설명

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| start | bool | true-开始，false-结束 |

## 용접IO

설정용접IO인터페이스时，전송：

*0x4011 WELDIOPORT_SET*

```json
{
  "Ain": {
    "Voltage": 2,
    "electric": 1
  },
  "Aout": {
    "given_electric_current": 1,
    "given_setvoltage": 2
  },
  "Din": {
    "arc_success": 1,
    "seek_success": 2
  },
  "Dout": {
    "arc_signal": 1,
    "gas_detection": 4,
    "positioning_mode": 5,
    "wire_feed": 2,
    "wire_feed_back": 3
  }
}
```

### Ain（输入模拟量）

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| Voltage | int | 용접电压 |
| electric | int | 용접电流 |

### Aout（输出模拟量）

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| given_electric_current | int | 给定电流信号 |
| given_setvoltage | int | 给定电压信号 |

### Din（数字输入）

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| arc_success | int | 起弧成功 |
| seek_success | int | 寻位成功 |

### Dout（数字输出）

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| arc_signal | int | 起弧成功信号 |
| gas_detection | int | 气体检测信号 |
| positioning_mode | int | 寻位모드 |
| wire_feed | int | 点动送丝信号 |
| wire_feed_back | int | 反向送丝信号 |

조회용접IO인터페이스时，전송：

*0x4012 WELDIOPORT_INQUIRE*

```json
{}
```

컨트롤러收到조회용접IO인터페이스时，반환：

*0x4013 WELDIOPORT_RESPOND*

data：同0x4011

조회용접상태时，전송：

*0x4015 WELDMONITOR_INQUIRE*

data：无

컨트롤러收到조회용접상태时，반환：

| 매개변수名 | 유형 | 설명 |
|--------|------|------|
| antiCollision.shieldFlag | bool | 屏蔽防碰撞 |
| weldEnabled | bool | 용접使能 |

  "commMode": "ModBus RTU",
  "slaveIP": "192.168.1.15",

| 매개변수名 | 유형 | 설명 |
