# 그라인딩 프로세스

---

## 1. 그라인딩 프로세스 매개변수 설정

**0x4601 POLISH_PARAM_SET**

### 요청 매개변수

| 매개변수 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| robot | number | 예 | 로봇 번호 |
| craftID | number | 예 | 프로세스 번호 |
| polish | object | 예 | 그라인딩 매개변수 객체 |

### polish 객체 매개변수

| 매개변수 | 유형 | 필수 | 기본값 | 설명 |
|--------|------|------|--------|------|
| toolWear | number | 아니오 | 0.0 | 도구 마모 보정 |
| startToolOffset | object | 아니오 | - | 시작점 도구 보정 |
| endToolOffset | object | 아니오 | - | 끝점 도구 보정 |
| autoFeed | object | 아니오 | - | 자동 보정 |

### startToolOffset / endToolOffset 객체 매개변수

| 매개변수 | 유형 | 필수 | 기본값 | 설명 |
|--------|------|------|--------|------|
| X | number | 아니오 | 0.0 | X 방향 보정 |
| Y | number | 아니오 | 0.0 | Y 방향 보정 |
| Z | number | 아니오 | 0.0 | Z 방향 보정 |

### autoFeed 객체 매개변수

| 매개변수 | 유형 | 필수 | 기본값 | 설명 |
|--------|------|------|--------|------|
| value | number | 아니오 | 0.0 | 자동 보정 값 |
| cycle | number | 아니오 | 0 | 자동 보정 주기 (회) |

### 요청 예시

```json
{
    "robot": 1,
    "craftID": 1,
    "polish": {
        "toolWear": 0.0,
        "startToolOffset": {
            "X": 0.0,
            "Y": 0.0,
            "Z": 0.0
        },
        "endToolOffset": {
            "X": 0.0,
            "Y": 0.0,
            "Z": 0.0
        },
        "autoFeed": {
            "value": 0.0,
            "cycle": 0
        }
    }
}
```

## 2. 그라인딩 프로세스 매개변수 조회

**0x4602 POLISH_PARAM_INQUIRE**

### 요청 매개변수

| 매개변수 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| robot | number | 예 | 로봇 번호 |
| craftID | number | 예 | 프로세스 번호 (범위: 1-9) |

### 요청 예시

```json
{
    "robot": 1,
    "craftID": 1
}
```

---

## 3. 그라인딩 프로세스 매개변수 반환

**0x4603 POLISH_PARAM_RESPOND**

### 응답 매개변수

| 매개변수 | 유형 | 설명 |
|--------|------|------|
| robot | number | 로봇 번호 |
| craftID | number | 프로세스 번호 |
| polish | object | 그라인딩 매개변수 객체 (구조는 0x4601과 동일) |

### polish 객체 매개변수

| 매개변수 | 유형 | 설명 |
|--------|------|------|
| toolWear | number | 도구 마모 보정 |
| startToolOffset | object | 시작점 도구 보정 (X, Y, Z) |
| endToolOffset | object | 끝점 도구 보정 (X, Y, Z) |
| autoFeed | object | 자동 보정 (value, cycle) |

### 응답 예시

```json
{
    "robot": 1,
    "craftID": 1,
    "polish": {
        "toolWear": 0.0,
        "startToolOffset": {
            "X": 0.0,
            "Y": 0.0,
            "Z": 0.0
        },
        "endToolOffset": {
            "X": 0.0,
            "Y": 0.0,
            "Z": 0.0
        },
        "autoFeed": {
            "value": 0.0,
            "cycle": 0
        }
    }
}
```
