# 컨베이어추적

## 컨베이어매개변수

### 설정컨베이어매개변수

**0x4801 TRACK_CONVEYOR_CONVEYORPARAM_SET**

요청 매개변수：

| 필드 | 유형 | 설명 |
|------|------|------|
| conveyorID | int | 컨베이어프로세스号 |
| robot | int | 로봇 번호 |
| compensation.encoderVal | double | 추적补偿엔코더值 |
| compensation.time | double | 추적补偿时间 |
| conveyor.encoderDirection | int | 엔코더方向：1-正向；-1-反向 |
| conveyor.encoderResolution | double | 엔코더分辨率 |
| conveyor.maxEncoderVal | double | 엔코더计数最大值 |
| conveyor.minEncoderVal | double | 엔코더计数最小值 |
| conveyor.posRecordMode | int | 컨베이어위치모드：0-엔코더；1-恒速설정 |
| conveyor.speed | double | 当컨베이어위치모드为恒速설정时的속도 |
| conveyor.userCoord | int | 사용자좌표계1~9 |
| track.height | int | 追踪目标高도：0-传感器感知；1-추적명령어示教 |
| track.track_on_run_mode_with_target_overrun | int | 目标超限실행方式：0-대기下一번째目标；1-跳줄到추적종료실행 |

```json
{
  "compensation": {
    "encoderVal": 66,
    "time": 10.0
  },
  "conveyor": {
    "encoderDirection": 1,
    "encoderResolution": 44.330,
    "maxEncoderVal": 2147483647.0,
    "minEncoderVal": -2147483648.0,
    "posRecordMode": 0,
    "speed": 9.0,
    "userCoord": 4
  },
  "conveyorID": 1,
  "robot": 1,
  "track": {
    "height": 1,
    "track_on_run_mode_with_target_overrun": 1
  }
}
```

## 매개변수识别

### 설정매개변수识别

**0x4804 TRACK_CONVEYOR_POSCHECKPARAM_SET**

요청 매개변수：

| 필드 | 유형 | 설명 |
|------|------|------|
| conveyorID | int | 컨베이어프로세스号 |
| robot | int | 로봇 번호 |
| detectSrc.type | int | 공작물检测信号源：0-비전；1-IO；2-全局변수 |
| detectSrc.globalVar | string | 当공작물检测信号源为2-全局변수时生效 |
| detectSrc.DI_capturePos | int | 当공작물检测信号源为1-IO时生效 |
| detectSrc.visionID | int | 비전프로세스号 |
| detectSrc.vision_io_filter_type | int | 비전IO信号过滤：0-信号触发过滤；1-信号消失过滤 |
| detectSrc.vision_latch_encoder_value_type | int | 비전锁存엔코더值：0-触发锁存；1-收到데이터锁存 |
| identification.communication | int | 비전통신方式：0-以太网；1-Modbus |
| identification.sensorTrg | int | 传感器触发方式：0-低电平触发；1-高电平触发 |
| identification.type | int | 공작물识别方式：0-비전；1-传感器 |

```json
{
  "conveyorID": 1,
  "detectSrc": {
    "type": 0,
    "globalVar": "GB018",
    "DI_capturePos": 9,
    "visionID": 3,
    "vision_io_filter_type": 0,
    "vision_latch_encoder_value_type": 1
  },
  "identification": {
    "communication": 1,
    "sensorTrg": 1,
    "type": 0
  },
  "robot": 1
}
```
