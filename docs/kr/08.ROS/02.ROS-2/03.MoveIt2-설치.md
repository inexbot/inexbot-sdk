# MoveIt 2 설치

ROS 1과 달리, ROS 2 Humble에서는 MoveIt 2를 수동으로 설치해야 합니다.

## 설치 단계

### 1. MoveIt 2 메인 패키지 설치

```bash
sudo apt install ros-humble-moveit
```

설치 후 오류 없이 실행되면 성공입니다. 다른 ROS 2 버전에서는 명령어가 다르므로, 본 튜토리얼은 Humble을 기준으로 합니다.

### 2. MoveIt Setup Assistant 설치

```bash
sudo apt install ros-humble-moveit-setup-assistant
```

### 3. 모든 MoveIt 관련 패키지 설치

```bash
sudo apt install ros-humble-moveit-*
```

### 4. Setup Assistant 실행

```bash
ros2 run moveit_setup_assistant moveit_setup_assistant
```

다음 화면이 나타나면 설치가 성공한 것입니다. MoveIt Setup Assistant의 사용 방법은 ROS 2와 ROS 1에서 기본적으로 동일하므로 여기서는 다루지 않습니다.
