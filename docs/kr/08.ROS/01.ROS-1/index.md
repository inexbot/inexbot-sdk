# ROS 1

## 소개

ROS(Robot Operating System)는 로봇 소프트웨어 프로그램을 작성하기 위한 고도의 유연성을 갖춘 소프트웨어 프레임워크입니다. 다양한 도구, 라이브러리 코드, 규약 프로토콜을 포함하며, 로봇 플랫폼 전반에 걸쳐 복잡하고 견고한 로봇 동작을 생성하는 과정의 난이도와 복잡성을 줄이는 것을 목표로 합니다.

ROS 설계자들은 ROS를 **"ROS = Plumbing + Tools + Capabilities + Ecosystem"** 으로 표현합니다. 즉, ROS는 통신 메커니즘, 도구 패키지, 로봇 고급 기능 및 로봇 생태계의 집합체입니다.
