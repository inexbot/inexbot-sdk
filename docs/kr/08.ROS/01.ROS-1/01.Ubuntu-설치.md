# Ubuntu 18.04 설치

본 문서는 Ubuntu 18.04의 다운로드, VMware Workstation을 통한 가상 머신 생성, 설치 및 구성, VMware Tools 설치의 전 과정을 자세히 설명합니다.

## 1. Ubuntu 18.04 다운로드

Ubuntu 공식 사이트 또는 국내 미러 사이트에서 Ubuntu 18.04 LTS의 iso 파일을 다운로드합니다.

![Ubuntu 18.04 LTS 공식 다운로드 페이지](assets/ros-31.png)

설치 후 다음과 같습니다:

![다운로드 완료된 Ubuntu 18.04 ISO 이미지 파일](assets/ros-32.png)

## 2. 가상 머신 생성

### VMware 구성 단계

1. VMware Workstation을 열고 **새 가상 머신 만들기**를 클릭합니다.

![VMware Workstation 메인 화면, 새 가상 머신 만들기 클릭](assets/ros-33.png)

2. **사용자 지정(고급)**을 선택한 다음 **다음**을 클릭합니다.

![사용자 지정(고급) 가상 머신 구성 유형 선택](assets/ros-34.png)

3. 설치된 VMware의 버전에 따라 하드웨어 호환성을 선택하고 **다음**을 클릭합니다.

![VMware 하드웨어 호환성 버전 선택](assets/ros-35.png)

4. **나중에 운영 체제를 설치합니다**를 선택하고 **다음**을 클릭합니다.

![나중에 운영 체제를 설치합니다 선택](assets/ros-36.png)

5. **Linux**를 선택하고, 버전을 **Ubuntu 64비트**로 선택한 뒤 **다음**을 클릭합니다.

![Linux Ubuntu 64비트 운영 체제 선택](assets/ros-37.png)

6. 직접 이름을 지정하고, 설치 위치를 선택합니다 (C 드라이브에는 설치하지 않는 것이 좋습니다).

![가상 머신 이름 지정 및 설치 위치 선택](assets/ros-38.png)

7. 프로세서 수는 **1**을 선택하고, 코어 수는 **4**를 선택한 후 **다음**을 클릭합니다.

![프로세서 수를 1, 코어 수를 4로 설정](assets/ros-39.png)

8. 메모리는 **8GB**를 선택하고 **다음**을 클릭합니다.

![가상 머신 메모리를 8GB로 설정](assets/ros-40.png)

9. **네트워크 주소 변환 사용**을 선택하고 **다음**을 클릭합니다.

![네트워크 주소 변환(NAT) 네트워크 유형 선택](assets/ros-41.png)

10. 기본 추천 구성대로 **다음**을 클릭합니다.

![I/O 컨트롤러 유형 선택 (기본 추천 구성)](assets/ros-42.png)

![가상 디스크 유형 선택 (기본 추천 구성)](assets/ros-43.png)

11. **새 가상 디스크 만들기**를 선택하고 **다음**을 클릭합니다.

![새 가상 디스크 만들기 선택](assets/ros-44.png)

12. 디스크를 할당합니다. **80GB**를 추천하며, 실제 필요에 따라 조정 가능합니다. **가상 디스크를 여러 파일로 분할**을 선택하고 **다음**을 클릭합니다.

![디스크 용량을 80GB로 설정하고 가상 디스크를 여러 파일로 분할 선택](assets/ros-45.png)

13. 기본값으로 **다음**을 클릭합니다.

![디스크 파일 저장 위치 지정 (기본값)](assets/ros-46.png)

14. **하드웨어 사용자 지정**을 클릭합니다.

![가상 머신이 준비되었으며, 추가로 구성하기 위해 하드웨어 사용자 지정 클릭](assets/ros-47.png)

15. 하드웨어 사용자 지정으로 들어가서 **ISO 이미지 파일 사용**을 선택하고, 처음에 다운로드한 iso 파일을 선택합니다. **사운드 카드**와 **프린터**를 제거한 다음 **닫기**를 클릭합니다. 가상 머신의 사운드 카드와 프린터는 사용하지 않으므로, 리소스 절약을 위해 비활성화합니다.

![하드웨어 사용자 지정 창, ISO 이미지 파일 선택 및 사운드 카드와 프린터 제거](assets/ros-48.png)

16. **완료**를 클릭하여 설치를 시작합니다.

![가상 머신 생성 완료, 설치를 시작하려면 완료 클릭](assets/ros-49.png)

## 3. Ubuntu 18.04 설치

1. Ubuntu 18.04에 진입하여 가상 머신을 켭니다. **中文 (간체)**(중국어 간체)를 선택하고 **Ubuntu 설치**를 클릭합니다.

![Ubuntu 설치 시작 화면, 중국어 간체를 선택하고 Ubuntu 설치 클릭](assets/ros-50.png)

2. **중국어**를 선택합니다.

![키보드 레이아웃을 중국어로 선택](assets/ros-51.png)

3. **일반 설치**, **Ubuntu 설치 시 업데이트 다운로드를 선택하고**, **계속**을 클릭합니다.

![일반 설치 선택 및 Ubuntu 설치 시 업데이트 다운로드 선택](assets/ros-52.png)

4. **디스크 전체를 삭제하고 Ubuntu 설치**를 선택합니다. 알림이 나타나면 **계속**을 선택하고, 마지막으로 **지금 설치**를 클릭합니다.

![디스크 전체를 삭제하고 Ubuntu 설치 선택](assets/ros-53.png)

![변경 사항을 디스크에 쓰기 확인 알림, 계속 클릭](assets/ros-54.png)

5. 사용자 이름과 비밀번호를 직접 설정하고, 지역은 임의로 입력해도 됩니다. Ubuntu가 스스로 설치하는 것을 기다립니다.

![Ubuntu 사용자 이름과 비밀번호 설정](assets/ros-55.png)

![Ubuntu 설치 진행률이 표시되는 화면](assets/ros-56.png)

## 4. VMware Tools 설치

Ubuntu 설치 후 지금 재부팅을 클릭하여 Ubuntu에 진입한 다음, VMware Tools를 설치합니다.

### VMware Tools 마운트

1. 상단의 **가상 머신**을 클릭하고, **VMware Tools 설치**를 클릭합니다.

![VMware 가상 머신 메뉴에서 VMware Tools 설치 선택](assets/ros-57.png)

다운로드 완료 후 다음과 같습니다:

![VMware Tools 다운로드 완료 알림](assets/ros-58.png)

2. 파일 - VMware Tools에서 이 폴더를 찾아 메인 디렉터리로 이동합니다.

![파일 관리자에서 VMware Tools 설치 패키지 폴더 찾기](assets/ros-59.png)

3. 메인 디렉터리에서 마우스 우클릭하여 **터미널에서 열기**를 클릭하고, 다음 명령을 실행합니다:

```bash
tar -zxvf VMwareTools-xxx.tar.gz
```

실행 후 폴더가 압축 해제된 것을 확인할 수 있습니다:

![VMware Tools 압축 패키지 압축 해제 후의 폴더](assets/ros-60.png)

4. 다음 두 명령을 실행합니다:

```bash
cd vmware-tools-distrib
sudo ./vmware-install.pl
```

첫 번째 정지 시 `yes`를 입력하고, 나머지 정지 시에는 Enter를 누르기만 하면 됩니다.

그 후 재부팅합니다.至此, Ubuntu 18.04 설치가 완료되었습니다.
