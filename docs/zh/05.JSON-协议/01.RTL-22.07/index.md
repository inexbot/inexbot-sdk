# RTL-22.07

RTL-22.07 是纳博特控制器的早期稳定版本。本文档持续维护，但仅进行重大 bug 修复，不再合入新功能。

> 如果你是新项目开发或需要更全面的功能，请查看 [RTL-24.03](../02.RTL-24.03/index.md)（当前主流出货版本）。

## 适用范围

- 所有支持 RTL-22.07 的控制器型号
- 适用场景：现有项目维护、简单集成

- [5000 端口通讯](./01.5000-端口通讯%282207%29) — 文件传输协议

## 阅读顺序

1. 先浏览 [6000 端口通讯](./02.6000-端口通讯%282207%29/01.运动控制/index.md) 的运动控制章节，了解基本的 JSON 命令格式
2. 再查阅你需要使用的具体功能模块（状态控制、工艺、作业文件等）
3. 遇到命令字含义不明时查阅 [命令字列表](./04.附录-命令字列表%282207%29)
