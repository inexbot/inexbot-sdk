# RTL-24.03 通讯协议

RTL-24.03 是纳博特控制器的**当前主流出货版本**，在 RTL-22.07 基础上新增系统配置、安全参数、协作机器人、程序管理和远程操作等能力。**新项目推荐使用此版本。**

> RTL-24.03 进行 bug 修复和关键功能合入。常规功能和实验功能将在更靠后的版本（如 RTL-25.01）中提供，后续版本稳定后开放 SDK 文档。

---

## 适用范围

- 所有支持 RTL-24.03 的控制器型号
- 适用场景：新项目开发、需要系统级功能的高级集成

## 与 RTL-22.07 的主要区别

| 特性     | RTL-22.07            | RTL-24.03                             |
| -------- | -------------------- | ------------------------------------- |
| 通信格式 | JSON 文本            | JSON 文本                             |
| 系统功能 | 运动控制、状态、工艺 | 同左 + 系统配置、安全参数、协作机器人 |
| 程序管理 | 作业文件 CRUD        | 指令级增删改查 + 远程程序启动         |

> - 两版本使用相同的端口体系（5000/6000/6001/7000），详见[版本与兼容性](../../版本与兼容性.md)。
> - 但 RTL-24.03 重新对协议命令字进行了整理，排序更加合理和易查找。

## 命令字速查

### 系统类 (1xx ~ 13xx)

- 系统 (1xx)：控制器配置、升级、备份、重启、关机
- 报错 (11xx)：错误码查询与清除
- 操作参数 (12xx)：操作模式设置
- 示教器通讯连接 (13xx)：示教器与上位机的连接管理

### 机器人类 (2xxx ~ 4xxx)

- **参数设置 (2xxx)**：DH 参数、关节参数、笛卡尔参数、点动参数、运动参数、机器人类型、从动轴、伺服、协作机器人、外部轴
- **运动控制 (30xx~34xx)**：模式设置、伺服使能、速度设置、点动、回零、切换坐标系、上电、清错、急停
- **状态 (30xx~34xx)**：机器人状态查询
- **工具/坐标系 (35xx)**：工具手、用户坐标系、大地坐标系
- **零点/复位 (36xx)**：零点设置、复位点
- **标定 (37xx)**：4 点标定、20 点标定、外部轴标定
- **安全 (38xx)**：干涉区、跟随误差、电机过载保护
- **动力学 (4xxx)**：动力学参数、力学功能、拖动示教、自适应加减速、负载使能、遥操作

### 程序类 (5xxx)

- 程序增删改查、导入导出
- 指令增删改查、操作
- 程序运行控制：启动、暂停、停止、运行次数、倒序、单步、试运行
- 程序状态、远程程序、程序自启动

### 变量类 (6xxx)

- 位置变量、数值变量

### 通讯/外设 (7xxx)

- 总线 (70xx)：EtherCAT、CANopen
- IO (71xx)：配置、状态、功能
- 其他外设 (72xx)
- Modbus (730x)
- TCP 通讯 (732x)
- 数据上传 (733x)
- FinsTCP (734x)
- OPC UA (735x)

### 自定义/其他 (1fxx ~ )

- 用户自定义功能扩展
