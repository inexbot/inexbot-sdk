## 版本提示

- **RTL-22.07** — 早期稳定版本，市场上仍有大量出货设备使用。仅进行重大 bug 修复，不再合入新功能。
- **RTL-24.03** — 当前主流出货版本，功能更全面。进行 bug 修复和关键功能合入。
- 后续版本（如 2501）尚未稳定，暂未开放 SDK 文档。

> SDK 版本需与控制器固件版本匹配，具体参见[版本与兼容性](./版本与兼容性.md)和[下载页](./12.相关下载.md)。
