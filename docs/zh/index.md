# 纳博特科技知识库

纳博特科技成立于 2015 年，是中国高端运动控制领域的技术先锋。我们以"软件定义控制、算法驱动智能"为核心理念，提供从底层硬件、中间件平台到上层应用的全栈式技术体系。

## 知识库目录结构

本仓库按技术领域和应用场景组织，方便开发者与工程师快速检索：

- **控制器** — 控制器二次开发 API、示例与自定义指令
- **示教器** — 示教器二次开发 API 与快速开始
- **上位机** — C++/C#/Python 渐入式教程与接口示例
- **主站库** — 主站通信库说明
- **JSON 协议** — RTL-22.07 JSON 通讯协议参考
- **ROS** — ROS-1 / ROS-2 集成指南
