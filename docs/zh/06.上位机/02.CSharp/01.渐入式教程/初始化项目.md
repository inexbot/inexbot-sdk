# 初始化项目

# C#

首先，在资源下载找到 C# 版本 SDK 的下载栏，选择 SDK_CSharp_x64 下载。

## 1 创建项目，导入SDK

选择 C#  “Windows窗体应用”创建。

![在Visual Studio创建新项目向导中选择"Windows窗体应用"项目模板](assets/csharp-01.png)

创建好项目后，将 SDK 拷贝到新创建的项目文件夹中。

![Windows资源管理器中显示项目目录下的Csharp_api SDK文件夹与项目文件](assets/csharp-02.png)

再将动态库放到生成路径下

![将nrc_host.dll动态库文件放到项目的bin/Release/net8.0-windows输出目录下](assets/csharp-03.png)

在 Visual Studio 2022 顶栏中点击>项目>显示所有文件，来将我们刚刚拷贝的文件显示出来。

![Visual Studio 2022项目菜单中点击"显示所有文件"选项以显示拷贝进来的SDK文件夹](assets/csharp-04.png)

完成上述步骤后，在左侧“解决方案资源管理器”中，会显示出我们刚刚拷贝进来的 “Csharp_api” 文件夹。

右键点击 "Csharp_api" 选择>包括在项目中选项将其包含。

完成后，再一次点击 Visual Studio 2022 顶栏上>项目>显示所有文件，来取消显示所有文件。

点击本地 Windows 调试器，如果使用的 Release 版本的库，需要将 Debug 切换为 Release。

![Visual Studio工具栏将解决方案配置切换为Release并通过本地Windows调试器运行程序](assets/csharp-05.png)

如果没有报错，表示我们成功的引入了 SDK。

更多示例可以浏览接口示例 | 纳博特科技

- 使用的 IDE 为： Visual Studio 2022
- 编译生成工具为：.NET 8.0
