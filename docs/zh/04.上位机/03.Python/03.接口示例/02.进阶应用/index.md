Python SDK 进阶接口示例，演示伺服跟踪运动（servo_move）等高级功能。

| 示例 | 说明 |
|------|------|

## 关键知识点

- **7000 端口：** servo_move 需要单独连接控制器 7000 端口，与 6001 端口分开
- **运行模式：** servo_move 需在运行模式下配合含外部点指令的作业文件使用
- **ServoMovePara 参数：** `clearBuffer`（清空缓冲）、`targetMode`（0=独立点/1=连续轨迹）、`coord`（0=关节/1=直角）
- **安全：** servo_move 会导致机器人实际运动，请先在仿真或低速下测试
