# 渐入式教程

根据你的开发环境选择对应的初始化流程：

| 环境 | 工具链 | 教程 |
|------|--------|------|
| Windows | MinGW + Qt Creator | [MinGW 环境初始化](./01.MinGW环境.md) |
| Windows | MSVC + Visual Studio | [MSVC 环境初始化](./02.MSVC环境.md) |
| Linux | GCC | [Linux 环境初始化](./03.Linux环境.md) |

> **重要：** SDK 与编译器的 ABI 必须匹配。不同编译器、架构、配置生成的库**不能混用**。详见[版本与兼容性](../../../版本与兼容性.md)。
