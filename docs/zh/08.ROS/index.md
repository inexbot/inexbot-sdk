# ROS

将纳博特控制器集成到 ROS（Robot Operating System）生态中。

## 适用范围

- ROS 1（Noetic 等）和 ROS 2（Humble 等）
- Linux 平台
- 适用场景：科研、多机器人协同、需要 ROS 工具链（MoveIt、RViz 等）
