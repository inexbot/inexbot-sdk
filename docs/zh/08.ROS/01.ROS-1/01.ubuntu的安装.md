# Ubuntu 18.04 安装

本文详细说明 Ubuntu 18.04 的下载、通过 VMware Workstation 创建虚拟机、安装配置以及 VMware Tools 安装的全过程。

## 1. 下载 Ubuntu 18.04

从 Ubuntu 官方或国内镜像站下载 Ubuntu 18.04 LTS 的 iso 文件。

![Ubuntu 18.04 LTS 官方下载页面](assets/ros-31.png)

安装好之后如下：

![下载完成后的 Ubuntu 18.04 ISO 镜像文件](assets/ros-32.png)

## 2. 创建虚拟机

### VMware 配置步骤

1. 打开 VMware Workstation，点击 **创建新的虚拟机**。

![VMware Workstation 主界面，点击创建新的虚拟机](assets/ros-33.png)

2. 选择 **自定义（高级）**，然后点击 **下一步**。

![选择自定义（高级）虚拟机配置类型](assets/ros-34.png)

3. 根据安装的 VMware 的版本号，选择硬件兼容性，点击 **下一步**。

![选择 VMware 硬件兼容性版本](assets/ros-35.png)

4. 选择 **稍后安装操作系统**，点击 **下一步**。

![选择稍后安装操作系统](assets/ros-36.png)

5. 选择 **Linux**，版本为 **Ubuntu 64位**，点击 **下一步**。

![选择 Linux Ubuntu 64位 操作系统](assets/ros-37.png)

6. 自己命名，选择安装位置（最好不要安装在 C 盘）。

![为虚拟机命名并选择安装位置](assets/ros-38.png)

7. 处理器数量选择 **1**，内核数量选择 **4**，点击 **下一步**。

![设置处理器数量为 1，内核数量为 4](assets/ros-39.png)

8. 内存选择 **8GB**，点击 **下一步**。

![设置虚拟机内存为 8GB](assets/ros-40.png)

9. 选择 **使用网络地址转换**，点击 **下一步**。

![选择使用网络地址转换(NAT)网络类型](assets/ros-41.png)

10. 按默认推荐配置，点击 **下一步**。

![选择 I/O 控制器类型（默认推荐配置）](assets/ros-42.png)

![选择虚拟磁盘类型（默认推荐配置）](assets/ros-43.png)

11. 选择 **创建新虚拟磁盘**，点击 **下一步**。

![选择创建新虚拟磁盘](assets/ros-44.png)

12. 分配磁盘，推荐 **80GB**，可根据实际需求调整，选择 **将虚拟磁盘拆分为多个文件**，点击 **下一步**。

![设置磁盘容量为 80GB 并选择将虚拟磁盘拆分为多个文件](assets/ros-45.png)

13. 默认，点击 **下一步**。

![指定磁盘文件存储位置（默认）](assets/ros-46.png)

14. 点击 **自定义硬件**。

![虚拟机已准备好，点击自定义硬件进行进一步配置](assets/ros-47.png)

15. 进入自定义硬件，使用 **ISO 映像文件**，选择一开始下载的 iso 文件，移除 **声卡** 和 **打印机**，然后点 **关闭**。因为用不上虚拟机的声卡和打印机，关闭以节省资源。

![自定义硬件窗口，选择 ISO 映像文件并移除声卡和打印机](assets/ros-48.png)

16. 点击 **完成**，开始安装。

![虚拟机创建完成，点击完成开始安装](assets/ros-49.png)

## 3. 安装 Ubuntu 18.04

1. 进入 Ubuntu 18.04，开启此虚拟机。选择 **中文（简体）**，点击 **安装 Ubuntu**。

![Ubuntu 安装欢迎界面，选择中文（简体）并点击安装 Ubuntu](assets/ros-50.png)

2. 选择 **汉语**。

![选择键盘布局为汉语](assets/ros-51.png)

3. 选择 **正常安装**，**安装 Ubuntu 时下载更新**，点击 **继续**。

![选择正常安装并勾选安装 Ubuntu 时下载更新](assets/ros-52.png)

4. 选择 **清除整个磁盘并安装 Ubuntu**，出现提醒选择 **继续**，最后点击 **现在安装**。

![选择清除整个磁盘并安装 Ubuntu](assets/ros-53.png)

![确认将更改写入磁盘的提示，点击继续](assets/ros-54.png)

5. 自己设定好用户名以及密码，地区可以随意填，等待 Ubuntu 自己安装即可。

![设置 Ubuntu 用户名和密码](assets/ros-55.png)

![Ubuntu 正在安装过程中的进度界面](assets/ros-56.png)

## 4. 安装 VMware Tools

安装好 Ubuntu 后点击现在重启，进入 Ubuntu，安装 VMware Tools。

### 挂载 VMware Tools

1. 点击上方 **虚拟机**，点击 **安装 VMware Tools**。

![VMware 虚拟机菜单中选择安装 VMware Tools](assets/ros-57.png)

下载完成如下：

![VMware Tools 下载完成提示](assets/ros-58.png)

2. 在文件—VMware Tools 中找到这个文件夹，把文件夹移入主目录。

![文件管理器中找到 VMware Tools 安装包文件夹](assets/ros-59.png)

3. 在主目录中点击右键，点击 **在终端打开**，运行以下命令：

```bash
tar -zxvf VMwareTools-xxx.tar.gz
```

运行后即可发现文件夹已解压：

![VMware Tools 压缩包解压后的文件夹](assets/ros-60.png)

4. 运行以下两条命令：

```bash
cd vmware-tools-distrib
sudo ./vmware-install.pl
```

除了第一次停顿时输入 `yes`，其他停顿都按回车即可。

之后重启；至此，Ubuntu 18.04 安装完成。
