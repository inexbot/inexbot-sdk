# Python

Use the Python language to call SDK interfaces to connect to the controller.

## Contents

- **Step-by-step Tutorial** — Build a Python project from scratch, hands-on initialization of the development environment
- **API Examples** — Complete code examples for connection, motion control, and servo_move tracking

## Environment Requirements

- Python 3.x
- SDK version must match the Python version, otherwise a "nrc_host.so not found" error will occur
